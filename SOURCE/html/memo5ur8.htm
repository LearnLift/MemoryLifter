Export

Exporting the dictionary to a file

Use this option to export the cards in the dictionary for processing in some other way (e.g. editing them in a spreadsheet or word processor).

 

  1. On the File menu, select Export.

  2. Specify the chapters to export in the chapter selector.

  3. Select the fields to be exported in the field selector.

  4. Choose the file format – whether you want words to be separated by semicolon, comma or tab.

  5. Click Export.

  6. Select the destination folder and name the exported file.

  7. Click Save.
