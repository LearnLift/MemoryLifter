What is New

Welcome to MemoryLifter version 2.0.  MemoryLifter 2.0 has many new features including:

 

  1. Support for Unicode to enable learning of eastern Asian languages (e.g., Japanes, Arabic, Chinese, etc...) which uses non-western characters.

  2. Improved control of the appearance of questions and answers.

  3. Improved media support.

  4. Dictionary format is in XML.

  5. Enhanced learning algorithm to allow time-based learning.

  6. Fixes of various bugs.
