Working with commentary sounds

MemoryLifter uses commentary sounds to help encourage the user during the learning session. You can disable or change the commentary sounds if you like.

 

Changing the commentary sounds:
 

To change the commentary sounds, following the instruction in Dictionary Options.

 

Disabling the commentary sounds:
 

  1. From the Tools menu, select Learning Options.

  2. Click the General tab.

  3. Select Play commentary sound.
