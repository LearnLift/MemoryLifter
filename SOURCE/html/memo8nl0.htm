Adding data (cards) to the dictionary

You can add cards to the dictionary for learning at any time.

  1. On the Cards menu, select Add… or click on the toolbar.
     

  2. Enter information for the fields below as needed:

    • Question: Enter a word/phrase for the question in the original language. For example: Banane

    • Example: Enter an example sentence using the word/phrase specified in the question above. For example, 'Eine Banane ist eine tropische Frucht'

    • Answer: Enter a word/phrase for the answer of the translation. For example, banana

    • Example: Enter an example sentence using the word/phrase specified in the answer above. For example, 'A banana is a tropical fruit'

      Hints:
      MemoryLifter supports synonyms in the Question and Answer fields. Separate synonyms by pressing ENTER.
      If you need to enter special characters, press the character map button.
       

  3. Use the Chapter: combo box to select the chapter you want the card to belong to.
     

  4. Attach an image, sound, and/or video file if desired.  
     

  5. Click Add Card
     

  6. Repeat steps 2 to 5 to add more cards.
     

  7. Click Close when you have finished adding all the cards.

 

If you want to change a card later on, use the Maintain feature.
