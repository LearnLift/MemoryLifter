Dictionary options

You can use the Dictionary options to specify a few options such as the description, author, etc... regarding the dictionary.

 

 To customize the options, select Tools, Dictionary options... to bring up the dialog box with its associate tabs.

 

General tab

 

You can enter the language title of the question and answer.

Example: If you're working on a "Deutsch - English" dictionary, enter "Deutsch" in the Question: field, "English" in the Answer: field.
 

Commentary sounds tab

 

This allows you to specify the sound files to be played as commentary while learning. There are two groups, with six sounds each. The first group, prefixed with an "A", is played when the learning mode's direction is question and answer (default). The second group, prefixed with an "Q", is played when the learning mode's direction is flipping question and answer.

 

Each of the group is divided into two subgroups:

 

The first three wave files are used when there is no audio associated with the card's answer. The commentary could for instance just say "That was correct".

 

The second subgroup is used when the card has an audio. This second commentary could then go like "Sorry, the right answer is" followed by the audio of the correct answer (which is supplied by the MemoryLifter program).

 

Associating sounds with the commentary

 

  1. In the left side of the box, select the appropriate commentary.

  2. To load an existing sound file for the selected commentary, click the wide button under "WAVE file (click to load):". Note: when there is already a sound associated with the item, the button will display the filename.

    OR

    To record the sound for the selected commentary, click the red recording button. The standard Sound recorder pops up.

 

Advanced options tab

  1. Add Author and description information as needed.

  2. Assign a category for the dictionary.

  3. Define a media directory where recorded audios should be put.  The default is "Media\"
