Snooze

Sends the program to the system tray, where it'll snooze until being double clicked or a certain amount of time passes by.

 

  1. From the Tools menu, select Learning Options... or click .

  2. Click the Snooze/Quit planner tab.

  3. Select appropriate option.  See the Learning Options for a description of the snooze options.
