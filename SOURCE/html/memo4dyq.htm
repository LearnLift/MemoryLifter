Sound recorder

The built-in sound recorder is a quick and useful tool for recording sounds to use with the dictionary. It is accessible wherever you see a record button (for example, in the Add new card, Maintain cards, or dictionary options dialog box).

 

Record a single sound (standard)

  1. Click Record to start recording.

  2. Speak the word.

  3. Click Stop to stop recording.

  4. If desired, click Play to test sound.

  5. Click Done when satisfied.

 

Notes:

  • Try keeping delays between the start of the recording and the speaking, as well as the end of the recording as short as possible.
  • Depending on what you are recording, some options on the sound recorder screen will be enabled and disabled as appropriate.
  • By default, MemoryLifter will automatically assign a file name for your recorded audio.  if you want to provide a more descriptive file name after recording, click the   (...) button on the recorder screen.
