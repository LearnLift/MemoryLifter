Características

MemoryLifter ofrece muchas funciones avanzadas fáciles de utilizar que mejoran la experiencia de aprendizaje o la hacen más divertida.

 

  • Aprendizaje automatizado con el sistema de ficheros
    Este método de aprendizaje le ayuda a centrarse en el material más difícil al presentar elementos que no se conocen con una frecuencia comparativamente más alta que la de aquellos elementos que sí se conocen (respondidos correctamente al menos una vez). Al mismo tiempo, los elementos que se conocían se vuelven a probar una y otra vez con un tiempo que aumenta entre repeticiones. Con este método, todo el material de estudio se guarda finalmente en la memoria a largo plazo. Además, la distribución de las fichas en los ficheros (las fichas se guardan en ficheros dependiendo de lo bien que se conozca el material) le proporciona una idea rápida del proceso de aprendizaje.
     

  • Aprendizaje basado en multimedia con imágenes, audio y vídeo
    Para mejorar la experiencia de aprendizaje se pueden añadir imágenes, audio y vídeo a cada ficha para facilitar la pronunciación de palabras o para proporcionar más información sobre el tema de la ficha.
    Incluso puede añadir audio usted mismo mediante el grabador de audio incorporado y fácil de utilizar.

 

  • Módulos de aprendizaje individuales

Cree sus propios módulos de aprendizaje para temas específicos con la ayuda de las funciones de importación, edición y organización de MemoryLifter.

    • Añadir fichas es sencillo con MemoryLifter.
      Escriba el contenido de los campos de texto a mano
      o emplee la función Importar desde el archivo para importar datos que ya se han escrito en formatos comunes basados en texto (delimitado por comas, delimitado por tabuladores, etc.)
      o use la función Importar del portapapeles para importar datos del portapapeles copiados de una páginas web o una hoja de cálculo.

    • Sinónimos  
      Algunas preguntas tienen varias respuestas correctas. MemoryLifter ofrece varias opciones para añadir sinónimos de una palabra de forma fácil e intuitiva.
      Añádalas a la ficha para ampliar sus conocimientos mientras aprende con MemoryLifter.

    • Frases de ejemplo
      Se puede especificar una frase de ejemplo para cada palabra, que le mostrará un posible uso de la misma en contexto o le proporcionará información adicional.

    • Capítulos
      Los capítulos       le ayudan a organizar las fichas en un módulo de aprendizaje para que pueda organizar el material de forma lógica en unidades manejables.
       

 Diseñe las fichas individuales definiendo sus propios estilos.

    • Cambie el aspecto de la ventana de aprendizaje utilizando o modificando los estilos prediseñados.

    • Cree estilos de fichas.

    • Cree estilos de capítulos.

 

  • Modo de aprendizaje individual
    Elija su modo de aprendizaje favorito.

    • Modos de aprendizaje
      Existen los siguientes modos de aprendizaje:

      • Entrada de texto estándar

      • Corregir sobre la marcha

      • Selección múltiple con elementos de distracción aleatorios o predefinidos

      • Comprensión de idioma hablado

      • Autoevaluación

      • Presentación aleatoria

      • Presentación en diapositivas

      • Simulación de una prueba con el temporizador de aprendizaje

    • Exportación, impresión.
      Si tiene que imprimir un conjunto de fichas didácticas, utilice el asistente de impresión.
      Basta con especificar las fichas que desea imprimir, ya sean todas las fichas o sólo las desconocidas o bien capítulos, fichas o ficheros determinados.
      Puede imprimir las fichas con todo el material multimedia o sólo la pregunta y la respuesta.

    • La función ocultar puede poner en pausa su aprendizaje después de la cantidad de tiempo que haya elegido.
      Reduce MemoryLifter a un icono en la barra de tareas, en la que se oculta durante un período de tiempo aleatorio o especificado antes de volver a plantearle más preguntas.

 

  • Determinación del éxito
    Las estadísticas     le proporcionan la información que necesita para comprender cómo está aprendiendo y cómo     progresa su aprendizaje a lo largo del tiempo.
