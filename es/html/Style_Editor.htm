Editor de estilos

Cambio del estilo, tamaño o color de la fuente y otros elementos de la pregunta y respuesta

Los estilos controlan el aspecto de las preguntas y respuestas en la ventana de aprendizaje de MemoryLifter.  

Los estilos contienen información sobre el tipo, color o tamaño de la fuente, el color de fondo, etc.

Para quienes estén familiarizados con la tecnología CSS, se trata en esencia de una hoja de estilos CSS.  

Las preguntas y respuestas poseen cada una su propio estilo.

 

MemoryLifter proporciona algunos estilos prediseñados: predeterminado, BumbleBee, iLifter, Kids y Tropical Beach.  

Puede modificar estos estilos prediseñados mediante el Editor de estilos.

 

Asimismo, puede especificar estilos para una única ficha o un único capítulo mediante el Editor de estilos de fichas.

 

 

Para modificar el estilo que se está usando:

 

  1. En el menú Herramientas, haga clic en Editor de estilos...

  2. Seleccione si desea cambiar la Configuración global (todas las propiedades del estilo) o la Configuración de estilo (solo las propiedades más importantes del estilo).

Nota:

Los siguientes procedimientos describen cómo cambiar la configuración de estilo.

Se pueden modificar muchos otros elementos si decide cambiar la configuración global, aunque el procedimiento sigue siendo el mismo.

 

  1. Seleccione su vista preferida al hacer clic en la ficha Tabla - Vista o en la ficha Texto - Vista.

 

Para modificar el estilo usando Tabla - Vista:

  1. Puede cambiar el aspecto de las categorías siguientes:

    • Pregunta (pregunta)

    • Ejemplo de pregunta (ejemploPregunta)

    • Respuesta  (respuesta)

    • Ejemplo de respuesta (ejemploRespuesta)

    • Respuesta correcta (respuestaCorrecta)

    • Respuesta incorrecta (respuestaIncorrecta)

  2. En la ficha Tabla - Vista, seleccione la fila del elemento o modo cuyo estilo desea modificar.

  3. Seleccione el cuadro Texto - Color para elegir un color.

  4. Seleccione el cuadro Color de la parte posterior para elegir un color de fondo.

  5. Haga clic en Fuente... para especificar la fuente, el estilo de la fuente y el tamaño.

  6. Seleccione el cuadro Alineación horizontal para seleccionar o escribir un valor de alineación (ninguna, izquierda, centro, derecha).

  7. Seleccione el cuadro Alineación vertical para seleccionar o escribir un valor de alineación (ninguna, arriba, medio, abajo).

  8. Repita este procedimiento para cada elemento que desee modificar.

  9. Haga clic en Cerrar para aplicar el estilo.

Notas:

  • En la parte derecha se muestra una vista previa.
    Puede elegir si quiere ver la vista previa de Pregunta, Respuesta correcta o Respuesta incorrecta.

  • Puede hacer clic en Restaurar en cualquier momento pararestablecer el estilo.

 

Para modificar el estilo usando Texto - Vista:

  1. Puede cambiar el aspecto de las categorías siguientes:

    • Pregunta

    • Ejemplo de pregunta

    • Respuesta

    • Ejemplo de respuesta

    • Color de fondo de las respuestas correctas

    • Color de fondo de las respuestas incorrectas

Advertencia:
Al cambiar valores en la ficha Texto - Vista, NO borre el punto y coma (;) al final de cada valor.

 

  1. Busque la sección pertinente en la ficha Texto - Vista y cambie los valores como desee.

Pregunta

    • Busque la sección ".pregunta".

Ejemplo de pregunta

    • Busque la sección ".ejemploPregunta".

Respuesta

    • Busque la sección ".respuesta".

Ejemplo de respuesta

    • Busque la sección ".ejemploRespuesta".

 

    • Cambie el valor del color como desee.

    • Cambie el valor de la familia de fuente como desee.
      Utilice el nombre de fuente exacto tal y como aparece en Windows.

    • Cambie el valor del tamaño de fuente como desee.

    • Cambie el valor del estilo de fuente como desee.

    • Cambie el valor del peso de fuente como desee.

 

Color de fondo de las respuestas correctas

    • Busque la sección ".respuestaCorrecta".

Color de fondo de las respuestas incorrectas

    • Busque la sección ".respuestaIncorrecta".

 

    • Cambie el valor del color de fondo como desee.

 

  1. Haga clic en Cerrar para aplicar el estilo.

 

Notas:

  • En la parte derecha se muestra una vista previa.
    Puede elegir si quiere ver la vista previa de Pregunta, Respuesta correcta o Respuesta incorrecta.

  • Puede hacer clic en Restaurar en cualquier momento para deshacer todos los cambios.
