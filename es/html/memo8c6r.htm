Crear copia de seguridad

Para crear una copia de seguridad de un módulo de aprendizaje:

 

  1. En el menú Archivo, haga clic en Crear copia de seguridad....

  2. Elija una ubicación para el archivo del módulo de aprendizaje y escriba un nombre.

 

Nota:

Debe disponer de permiso para copiar el módulo de aprendizaje actual para crear una copia de seguridad.
