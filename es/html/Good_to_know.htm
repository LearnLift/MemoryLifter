Estilos del módulo de aprendizaje

Además, hemos ampliado las posibilidades de modificación de estilos.

Ahora puede modificar el estilo:
 

  • De todos los estilos predefinidos
    (para todos los módulos de aprendizaje abiertos en el futuro)
    Para modificar todos los estilos predefinidos, seleccione la configuración global en el Editor de estilos.

  • Únicamente del estilo activo
    (para todos los módulos de aprendizaje abiertos en el futuro)
    Para modificar sólo el estilo predefinido activo, seleccione la configuración del estilo en el Editor de estilos.

  • De capítulos
    (sólo para el módulo de aprendizaje activo, cuando Utilizar estilos del módulo de aprendizaje se haya seleccionado en el menú Herramientas)
    Para modificar el estilo de un capítulo, abra el Editor de estilos de fichas en el cuadro de diálogo Administrar capítulos.

  • De fichas
    (sólo para el módulo de aprendizaje activo, cuando Utilizar estilos del módulo de aprendizaje se haya seleccionado en el menú Herramientas)
    Para modificar el estilo de una ficha, abra el Editor de estilos de fichas en el cuadro de diálogo Mantener fichas.

Tenga en cuenta que las propiedades se heredan.

 

Exportar

Puedes exportar tu módulo de aprendizaje para guardarlo en otros formatos.

 

Si desea modificar las fichas con otra herramienta de software, si desea cambiar el orden de los campos de pregunta, respuesta y ejemplo o si desea añadir información de capítulos a las fichas de forma rápida y sencilla, exporte primero el módulo de aprendizaje o campos especificados del mismo a archivos de texto.

 

Importar

Le recomendamos que importe archivos de texto si está familiarizado con el proceso de guardar tablas en un formato delimitado.

Por otro lado, se recomienda la importación desde el portapapeles si desea importar datos mientras navega por Internet o si desea importar datos directamente de otros documentos.
