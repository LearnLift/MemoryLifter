Cerrar

Para cerrar el módulo de aprendizaje actual

 

  1. En el menú Archivo, haga clic en Cerrar.
  2. De esta manera se cierra el módulo de aprendizaje actual y se queda una ventana gris en blanco.
  3. Ahora puede abrir otro módulo mediante la página de inicio o en el menú Módulos de aprendizaje recientes.
 
 

Temas relacionados:

  • Abrir un módulo de aprendizaje existente

  • Familiarización con la página de inicio

  • Inicio de una nueva sesión de aprendizaje

  • Módulos de aprendizaje recientes
