Hinzufügen eines Bildes oder einer Audio- oder Videodatei

MemoryLifter bietet die Möglichkeit, in den Lernkarten ein Bild oder eine Audio- oder Videodatei hinzuzufügen, um die Effektivität der Karten zu steigern.

 

 

Hinzufügen einer Audiodatei

  1. Klicken Sie im Dialog Karten verwalten auf die Audio-Schaltfläche .

  2. Der Audiorecorder wird geöffnet.

  3. Klicken Sie zum Auswählen einer vorhandenen Datei auf Suchen...
    Oder
    Klicken Sie auf Aufzeichnen, wenn Sie eine eigene Audiodatei aufzeichnen möchten.

 

Hinweis:

Es werden folgende Audio-Dateiformate unterstützt: mp3, wav, mid

 

 

Hinzufügen einer Videodatei

  1. Klicken Sie im Dialog Karten verwalten auf die Video-Schaltfläche .

  2. Klicken Sie auf Suchen.

  3. Wählen Sie die gewünschte Datei aus und klicken Sie auf OK.

 

Hinweis:

Es werden folgende Video-Dateiformate unterstützt: avi, wmv

 

 

Hinzufügen einer Bilddatei

Bilder können auf zwei verschiedene Weisen hinzugefügt werden:

  1. Wählen Sie im Menü Bearbeiten den Eintrag Karten verwalten, drücken Sie F10 oder klicken Sie in der Symbolleiste auf .

  2. Führen Sie eine der folgenden Aktionen durch:

  • Klicken Sie im Dialogfenster Karten verwalten auf die Bilder-Schaltflläche und anschließend auf Suchen. Wählen Sie die gewünschte Datei aus und klicken Sie auf OK.

    Oder

  • Ziehen Sie ein Bild per Drag&Drop-Methode in den Bild-Bereich des Dialogfensters.
     

Das Bild wird im Bild-Bereich des Dialogfensters angezeigt.

Beim ersten Bild, das Sie in den Bild-Bereich des Dialogfensters ziehen, erscheint der Dialog Bildgröße ändern.

 

 

Sie können hier wählen, in welcher Größe das Bild im Lernmodul gespeichert werden soll: klein, mittel, groß oder ohne Änderung der Größe.

Treffen Sie Ihre Auswahl, indem Sie auf eines der Symbole im Dialogfenster klicken.

Um sowohl für die Frage als auch für die Antwort dieselbe Bildgröße zu verwenden, markieren Sie das Kontrollkästchen Für beide Bereiche (Frage/Antwort) anwenden.

 

Hinweise:

Während dem Bearbeiten von Karten können Sie die Auswahl der Bildgröße jederzeit wieder durch Klicken auf ändern.

Um für die Antwort dasselbe Bild wie für die Frage einzufügen, markieren Sie das Kontrollkästchen Bild aus Frage verwenden.

Es werden folgende Bild-Dateiformate unterstützt: bmp, gif, jpg, png, ico, emf, wmf
