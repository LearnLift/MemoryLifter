Lernen im Standardmodus

Das Beantworten der Fragen kann je nach ausgewähltem Lernmodus entweder im Standardmodus durch normale Texteingabe erfolgen oder durch Auswahl der Antwort(en) im Multiple Choice-Modus.

 

Beantworten von Fragen im Standardmodus
 

  1. Stellen Sie sicher, dass der Antwortbereich im Eingabemodus ist (Antwortbereich mit Textfeld in der Mitte).

  2. Geben Sie Ihre Antwort im Textfeld ein.
    Drücken Sie die Eingabetaste, um zur nächsten Frage zu gehen.
    Hat eine Frage mehrere richtige Antworten (Synonyme), wird der Cursor durch Drücken der Eingabetaste in die nächste Zeile gesetzt. Sie können nun eine weitere Antwort eingeben.
    Anderenfalls (oder nach Beantwortung des letzten Synonyms), wird Ihre Antwort mit der Eingabetaste abgegeben.

  3. Nach Abgabe der Antwort wird die Bewertung angezeigt. Außerdem wird der rechte Pfeil aktiviert, mit dem Sie die nächste Karte öffnen können.

 

Hinweise:

  • Wenn Sie die Antwort nicht wissen, drücken Sie einfach die Eingabetaste. MemoryLifter zeigt dann die korrekte Antwort an.
  • Für die Eingabe von Sonderzeichen verwenden Sie die Zeichentabelle.

 

 

Bearbeiten von Lernkarten

Um während dem Lernen eine Karte zu ändern, klicken Sie im Menü Bearbeiten auf Lernkarten verwalten oder in der Symbolleiste auf . Alternativ können Sie hierzu auch die Funktionstaste F10 drücken.
Es erscheint das Dialogfenster Karten verwalten, in dem Sie sämtliche Elemente der Lernkarten ändern können.
Klicken Sie auf Änderungen speichern, um Ihre Änderungen anzuwenden und anschließend auf Schließen, um mit dem Lernen fortzufahren.

 

Beenden einer Lernsitzung

  1. Klicken Sie im Fenster rechts oben auf das X.

  2. Wenn Sie die Lernsitzung beenden möchten, beantworten Sie den anschließend angezeigten Dialog mit Ja. Mit Abbrechen kehren Sie zurück und die Lernsitzung wird fortgeführt.

  3. Es werden die Statistiken für diese Runde angezeigt. Klicken Sie auf OK.
