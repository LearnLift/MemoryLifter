Lernmodul-Übersicht

 

Beschränken der Anzahl angezeigter Lernkarten (Anzeigefilter)

Wählen Sie im Kombinationsfeld Kapitel anzeigen das gewünschte Kapitel aus, dessen Karten angezeigt werden sollen oder "Alle Karten".

 

Sortieren der angezeigten Lernkarten

Die in der Liste angezeigten Lernkarten können in aufsteigender oder absteigender Reihenfolge sortiert werden. Klicken Sie hierzu auf den Spaltentitel der jeweiligen Spalte, die Sie als Sortierkriterium verwenden möchten.  

Um beispielsweise die angezeigten Lernkarten nach ihren momentan zugeordneten Boxen zu sortieren, klicken Sie auf den Titelbereich der Spalte "Box".

 

Auswählen von Lernkarten

  1. Um eine einzelne Lernkarte auszuwählen, klicken Sie im unteren Teil des Dialogfensters auf den Eintrag der gewünschten Karte.

Hinweis: Der Inhalt der Lernkarte wird im oberen Teil des Dialogfensters angezeigt.

 

  1. Um mehrere Lernkarten auszuwählen, halten Sie wie in Windows gewohnt die Umschalttaste oder die Strg-Taste gedrückt, während Sie im unteren Teil des Dialogfensters auf die Einträge der gewünschten Karten klicken.
     

Löschen von Lernkarten

  1. Wählen Sie eine oder mehrere Karten aus.

  2. Klicken Sie auf Karte(n) löschen oder klicken Sie mit der rechten Maustaste auf den Eintrag und anschließend auf Löschen.

Achtung:
Dieser Schritt kann nicht rückgängig gemacht werden!

 

Verschieben von Lernkarten in ein bestimmtes Kapitel

  1. Wählen Sie eine oder mehrere Karten aus.

  2. Klicken Sie mit der rechten Maustaste auf den Eintrag und anschließend auf Verschieben. Klicken Sie dann auf Kapitel und anschließend auf das Kapitel, in das die Karte verschoben werden soll.

 

Verschieben von Lernkarten in eine bestimmte Box

  1. Wählen Sie eine oder mehrere Karten aus.

  2. Klicken Sie mit der rechten Maustaste auf den Eintrag und anschließend auf Verschieben. Klicken Sie dann auf Box und anschließend auf die Box, in welche die Karte(n) verschoben werden soll(en).

 

Suchen von Lernkarten

  1. Klicken Sie in das Suchfeld Suchtext eingeben....

  2. Geben Sie ein Suchwort ein.
    Hinweis:         Es werden alle Lernkarten angezeigt, die dieses Suchwort enthalten.
