Sicherungskopie erstellen

Erstellen einer Sicherungskopie eines Lernmoduls

 

  1. Klicken Sie im Menü Datei auf den Eintrag Sicherungskopie erstellen.

  2. Wählen Sie einen Zielordner aus, in dem das Lernmodul gespeichert werden soll, und geben Sie einen Namen ein.

 

Hinweis:

Sie müssen die Berechtigung zum Kopieren des aktuellen Lernmoduls haben, um eine Sicherungskopie zu erstellen.
