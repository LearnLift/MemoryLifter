Neustart einer Lernsitzung

Durch einen Neustart Ihrer laufenden Lernsitzung werden alle Boxen geleert und das Lernmodul in einen Anfangsstatus zurückgesetzt.

 

Neustart einer Lernsitzung:

  1. Klicken Sie im Menü Lernen auf den Eintrag Lernvorgang neu starten.

  2. Es erscheint eine Abfrage, ob Sie wirklich alle Boxen leeren möchten. Bestätigen Sie diese mit Ja.

  3. Wählen Sie im Dialog Kapitelauswahl die Kapitel, die Sie mit dem Auswahlbereich für die Kapitel lernen möchten.

  4. Klicken Sie auf OK.
