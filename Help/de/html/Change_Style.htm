Oberfläche wechseln

Mit dieser Option können Sie einstellen wie das Lernfenster von MemoryLifter aussehen soll! Die folgenden fünf vordefinierten Stile stehen zur Auswahl:

 

  • Default

  • BumbleBee

  • iLifter

  • Kids

  • TropicalBeach

 

Ändern der Oberfläche

  • Klicken Sie im Menü Werkzeuge auf den Eintrag Oberfläche wechseln und wählen Sie in dem dann geöffneten Untermenü die gewünschte Oberfläche aus.

Hinweis:

Die ausgewählte Oberfläche ist mit einem Häkchen markiert.
