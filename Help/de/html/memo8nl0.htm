Hinzufügen von Lernkarten in einem Lernmodul

Gehen Sie wie folgt vor, um Lernkarten in einem Lernmodul hinzuzufügen:

  1. Wählen Sie im Menü Bearbeiten den Eintrag Karte hinzufügen oder klicken Sie in der Symbolleiste auf das Symbol Neue Karte hinzufügen... .

  2. Geben Sie im Dialogfenster Karten verwalten die benötigten Informationen ein:

    • Geben Sie im Textfeld Frage ein Wort, eine Redewendung oder einen Satz für die Frage in der Ausgangssprache ein.
      Beispiel: Banane

    • Geben Sie im Textfeld Beispiel einen Beispielsatz ein, der das im Textfeld Frage eingetragene Wort bzw. die Redewendung enthält.
      Zum Beispiel: "Eine Banane ist eine tropische Frucht."

    • Geben Sie im Textfeld Antwort ein Wort, eine Redewendung oder einen Satz in der Antwortsprache ein.
      Beispiel: Banana

    • Geben Sie im Textfeld Beispiel einen Beispielsatz ein, der das im Textfeld eingetragene Wort bzw. die Redewendung enthält.
      Zum Beispiel: "A banana is a tropical fruit!

    • Tipp:
      MemoryLifter unterstützt Synonyme in den Feldern für die Frage und die Antwort.
      Um Synonyme voneinander zu trennen, drücken Sie die Eingabetaste.
      Für die Eingabe von Sonderzeichen klicken Sie auf das Symbol für die Zeichentabelle .

  1. Wählen Sie im Feld Kapitel aus, zu welchem Kapitel die Lernkarte gehören soll oder geben Sie das Kapitel in das Feld ein.

  2. Hängen Sie ein Bild, eine Audiodatei und/oder eine Videodatei an, falls gewünscht.  
     

  3. Klicken Sie auf Karte hinzufügen.
     

  4. Um weitere Lernkarten hinzuzufügen, wiederholen Sie die Schritte 2 bis 5.
     

  5. Wenn Sie mit dem Hinzufügen von Lernkarten fertig sind, klicken Sie auf Schließen.

 

Um die Lernkarten später wieder zu bearbeiten, verwenden Sie die Funktion Lernkarten verwalten.
