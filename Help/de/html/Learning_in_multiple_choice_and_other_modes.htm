Lernen im Multiple Choice-Modus

Zusätzlich zur Standard-Texteingabe bietet MemoryLifter noch eine Reihe anderer Lernmodi: Multiple Choice-Auswahl, Bilderkennung, Hörverständnis, eigene Bewertung und zufällige Auswahl des Anzeigemodus.

 

Lernen im Multiple Choice-Modus

  1. Klicken Sie im Menü Lernen auf den Eintrag Lernoptionen oder in der Symbolleiste auf .
     

  2. Klicken Sie auf das Register Lernmodus.

  3. Markieren Sie im Bereich Lernmodi das Kontrollkästchen Multiple Choice.

 

Hinweis:

Wenn mehr als ein Lernmodus markiert ist, wählt MemoryLifter jedesmal, wenn eine neue Lernkarte angezeigt wird, den Modus zufällig aus den markierten Modi aus.

 

 

Beantworten von Fragen im Multiple Choice-Modus

  1. Klicken Sie auf die Antwort(en) oder geben Sie die Anzahl der Antworten ein, die Sie für richtig halten.

    2. +

  2. Klicken Sie auf oder drücken Sie die Eingabetaste.

 

Oder
Wenn Sie die Antwort nicht wissen, drücken Sie die Eingabetaste.
Die richtige Antwort wird angezeigt und die Lernkarte kommt in den Zwischenstapel.
Nach kurzer Zeit wird diese Frage erneut gestellt.

 

 

Für das Lernen im Multiple Choice-Modus stehen verschiedene Multiple Choice-Optionen zur Auswahl.

MemoryLifter zeigt normalerweise eine festgelegte Anzahl von möglichen Antworten aus dem Pool von Antworten an (d.h. es erfolgt eine zufällige Auswahl der so gennanten Distraktoren (zur Auswahl angezeigte falsche Antwortmöglichkeiten).

Dabei kann eingestellt werden, wie viele mögliche Antworten im Multiple Choice-Modus angezeigt werden sollen.

 

Die Auswahl der Distraktoren muss jedoch nicht unbedingt zufällig erfolgen. Durch Definieren solcher festgelegten Falschantworten kann der Schwierigkeitsgrad des Lernmoduls erhöht werden.

 

Zusätzlich können Sie auch mehrere richtige Antworten pro Frage erlauben und festlegen, wie viele richtige Antworten pro Frage im Multiple Choice-Modus maximal angezeigt werden sollen.

 

Hinweis:

Im Multiple Choice-Modus bewertet MemoryLifter die Antwort nur dann als korrekt, wenn das komplette Auswahlmuster für die zur Auswahl stehenden Antworten korrekt ist.

 

 

Einstellen der Multiple Choice-Optionen

 

 

  1. Klicken Sie im Menü Lernen auf den Eintrag Lernoptionen oder in der Symbolleiste auf .
     

  2. Klicken Sie auf das Register Lernmodus.

  3. Markieren Sie im Bereich Lernmodi das Kontrollkästchen Multiple Choice.

  4. Wenn Sie möchten, dass MemoryLifter nicht nur die für die Karte definierten Distraktoren anzeigen soll, sondern auch weitere, zufällig ausgewählte Distraktoren aus dem Antworten-Pool, markieren Sie im Fensterbereich Multiple Choice-Optionen das Kontrollkästchen Zufällige Distraktoren erlauben.

  5. Wenn Sie möchten, dass MemoryLifter pro Frage möglicherweise mehr als eine korrekte Antwort zur Auswahl anbietet, markieren Sie im Fensterbereich Multiple Choice-Optionen das Kontrollkästchen Mehrere richtige Antworten erlauben.

  6. Im Feld Anzahl der Optionen können Sie einstellen oder eingeben, wie viele mögliche Antworten eine Frage im Multiple Choice-Modus haben soll.

  7. Im Feld Max. Anzahl richtiger Antworten können Sie einstellen oder eingeben, wie viele richtige Antworten im Multiple Choice-Modus pro Frage maximal möglich sein sollen.
    Hinweis    : Dies ist die maximale Anzahl an Karten, die innerhalb der Lernsitzung angezeigt werden. Die tatsächliche Anzahl korrekter Antworten wird in der Karte selbst eingestellt.

  8. Klicken Sie auf OK, um das Lernen im Multiple Choice-Modus zu beginnen.

 

 

 

Für die anderen Lernmodi und weitere interessante Optionen, probieren Sie einfach die Lernoptionen aus.
