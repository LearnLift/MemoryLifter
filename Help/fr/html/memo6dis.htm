Importation des cartes d'un fichier

MemoryLifter permet d'importer des données à partir d'un fichier texte ou csv avec les délimiteurs suivants.

 

  • Tabulation : les paires de mot sont séparées par une tabulation et les cartes par des retours chariot

  • Point-virgule : les paires de mot sont séparées par un point virgule et les cartes par des retours chariot

  • Virgule : les paires de mot sont séparées par une virgule et les cartes par des retours chariot

 

Remarques :

  • Actuellement, seuls les formats de fichier texte sont pris en charge.
    Si vous souhaitez importer des données à partir d'un autre format (traitement de texte par exemple), essayez d'enregistrer les données au format texte standard.

  • Dans les fichiers délimités par une virgule, l'importation peut échouer si une virgule est utilisée dans des guillemets, par exemple ""Donne-le moi", dit-il".

  • Si des caractères spéciaux n'apparaissent pas dans le fichier exporté d'Excel, essayez de l'enregistrer au format TXT (*.txt - Unicode Text) ou au format CSV (*.csv - Comma delimited).

  • Si Unicode n'est pas sélectionné, Excel n'exporte généralement pas correctement les caractères étrangers (caractères dans une autre langue que celle du système).

  • Sélectionnez le jeu de caractères correct pour vos données.

 

Pour importer un fichier

  1. Dans le menu Edition, sélectionnez Importation de cartes => A partir du fichier...

  2. Dans la boîte de dialogue Importer, cliquez sur Charger le fichier...

  3. Sélectionnez le fichier à importer.

  4. Cliquez sur Ouvrir.

Remarque :

MemoryLifter va désormais tenter de déterminer le délimiteur utilisé par le fichier, charger le contenu du fichier et vous permettre de visualiser les 19 premières cartes.
Si MemoryLifter échoue, saisissez ou sélectionnez le format de fichier approprié dans le champ Format des fichiers.

 

  1. Spécifiez les champs à importer à l'aide du sélecteur de champ.

  2. Organisez les champs à l'aide du sélecteur de champ.

Remarque :

Reproduisez la séquence des colonnes dans votre fichier source.

 

  1. Prévisualisez les cartes dans le champ Aperçu.

Remarque :

Si la prévisualisation des colonnes n'est pas bonne, essayez de sélectionner un autre format de fichier dans le champ Format de fichiers.

Si la prévisualisation des caractères n'est pas bonne, essayez de sélectionner un autre jeu de caractères dans le champ Jeu de caractères (page de codes).

 

  1. Désélectionnez l'option Contient l'en-tête si le fichier source ne contient pas de ligne d'en-tête.

  2. Sélectionnez l'option Séparer les synonymes si vous avez séparé les synonymes de la question ou de la réponse dans le fichier source par des virgules.

  3. Dans le champ Chapitre par défaut, sélectionnez un chapitre pour les cartes importées.

  4. Cliquez sur Importer pour lancer l'importation.

  5. Cliquez sur Fermer pour commencer l'apprentissage.
