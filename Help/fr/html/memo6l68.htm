Mappe de caractères

La mappe de caractères permet de saisir sur vos cartes certains caractères spéciaux et étrangers non accessibles sur votre clavier.

 

Pour utiliser la mappe de caractères

  1. Dans le menu Outils, cliquez sur Mappe de caractères ou cliquez sur dans la barre d'outils.
    Remarque :     Le bouton s'éteint lorsqu'il est sélectionné.

  2. Cliquez sur le caractère à ajouter dans la boîte de symboles.

 

Pour interrompre l'utilisation de la mappe de caractères

  • Dans le menu Outils, cliquez sur Mappe de caractères ou cliquez sur dans la barre d'outils.
    Remarque :     Le bouton s'allume lorsqu'il n'est pas sélectionné.
