Exporter les cartes

Exportation d'un module d'apprentissage dans un fichier texte

Utilisez cette option pour exporter les cartes d'un module d'apprentissage de manière à les traiter autrement (édition dans une feuille de calcul ou un traitement de texte, par exemple).

 

  1. Dans le menu Edition, cliquez sur Exporter les cartes... ou appuyez sur Ctrl+E.

  2. Dans la boîte de dialogue Exporter :

    • spécifiez les chapitres à exporter avec le sélecteur de chapitre.

    • spécifiez les champs à exporter à l'aide du sélecteur de champ.

    • spécifiez le format du fichier (données séparées par un point virgule, une virgule ou une tabulation).

    • Cliquez sur Exporter.

  3. Sélectionnez le dossier de destination et saisissez le nom du fichier exporté dans le champ Nom du fichier.

  4. Cliquez sur Enregistrer.
