Ajouter la carte

Ajout de cartes au module d'apprentissage

Pour ajouter des cartes au module d'apprentissage :

  1. Dans le menu Edition, cliquez sur Ajouter la carte… ou sur Ajouter une nouvelle carte... dans la barre d'outils.

  2. Renseignez correctement les champs de la boîte de dialogue Maintenir les cartes :

    • Dans la zone de texte Question, saisissez un mot ou une phrase dans la langue d'interrogation.
      Par exemple : Banane

    • Dans la zone de texte Exemple, saisissez une phrase exemple mettant en contexte le mot/la phrase demandé ci-dessus.
      Par exemple : "Une banane est un fruit tropical"

    • Dans la zone de texte Réponse, saisissez un mot ou une phrase dans la langue de réponse.
      Par exemple : banana

    • Dans la zone de texte Exemple, saisissez une phrase exemple mettant en contexte le mot ou la phrase spécifié(e) dans la réponse.
      Par exemple : "A banana is a tropical fruit"

    • Conseil :
      MemoryLifter prend en charge les synonymes dans les champs Question et Réponse.
      Séparez les synonymes en appuyant sur ENTREE.
      Si vous devez saisir des caractères spéciaux, cliquez sur le bouton mappe de caractères.
       

  1. Dans la zone de texte Chapitre, saisissez ou sélectionnez le chapitre auquel vous souhaitez associer la carte.
     

  2. Liez un fichier image, audio et/ou vidéo au besoin.  
     

  3. Cliquez sur Ajouter la carte.
     

  4. Répétez les étapes 2 à 5 pour ajouter d'autres cartes.
     

  5. Cliquez sur Fermer une fois que vous avez terminé d'ajouter les cartes.

 

Pour modifier une carte ultérieurement, utilisez la fonction Maintenir les cartes.
