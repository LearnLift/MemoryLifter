Redémarrage d'une session d'apprentissage

Vous pouvez redémarrer votre session d'apprentissage en cours et vider toutes les boîtes, pour réinitialiser complètement votre apprentissage.

 

Pour redémarrer une session d'apprentissage :

  1. Dans le menu Apprendre, cliquez sur Redémarrer l'apprentissage.

  2. MemoryLifter vous demande de confirmer le vidage de toutes les boîtes, cliquez sur Oui.

  3. Dans la boîte de dialogue Sélection des chapitres, sélectionnez les chapitres à apprendre avec le sélecteur de chapitre.

  4. Cliquez sur OK.
