Changer l'habillage

Vous pouvez modifier l'apparence de la fenêtre d'apprentissage de MemoryLifter. Cinq styles prédéfinis vous sont proposés :

 

  • Default

  • BumbleBee

  • iLifter

  • Kids

  • TropicalBeach

 

Pour modifier l'habillage

  • Dans le menu Outils, cliquez sur Changer l'habillage, puis sélectionnez le nom de l'habillage souhaité.

Remarque :

L'habillage sélectionné est indiqué par une coche.
