Nouveautés

Bienvenue dans MemoryLifter™

 

Avantages

  • Nouveau format de fichier (y compris icône du fichier du nouveau module d'apprentissage) :

    • toutes les données stockées dans un fichier (plus besoin de fichier média ou dossier supplémentaire)

    • possibilité d'enregistrer dans chaque fichier la progression d'apprentissage de plusieurs apprenants

    • sécurité accrue des données

  • Ouvrir / apprendre les modules d'apprentissage à partir d'une base de données centrale sur un serveur MemoryLifter (obtenir le logiciel gratuit sur www.memorylifter.com)

  • Gestion facile des modules d'apprentissage grâce à la page de démarrage ("Mes modules d'apprentissage"), qui vous permet d'accéder facilement à tous vos modules d'apprentissage (en local ou sur le serveur)

 

Améliorations :

  • Nouveaux habillages d'IU

  • Nouvelle présentation de l'option choix multiple

  • Simplification des menus avec représentation par des icônes correspondant à celles présentes dans la barre d'outils

  • Simplification de la barre d'outils

  • Nouvel algorithme des résultats (basé sur le pourcentage des cartes connues dans le module d'apprentissage)

  • Améliorations générales de l'interface utilisateur

  • Le nouveau feedback des réponses

    • ne se trouve plus avec le rendu des réponses (qui peut être fermé également)

    • mise en surbrillance des synonymes vrai/faux

  • Améliorations des performances

  • Multitude d'améliorations mineures

 

Résolution de bogues :

  • L'impression ne génère plus de pages vides

  • Certains problèmes de rendu ont été corrigés avec les polices Asie

  • Certaines polices ont été corrigées ce qui a provoqué une exception lors du rendu d'une carte

  • Un grand nombre de bogues minimes résolu

 

Remarques :

  • MemoryLifter convertit automatiquement les modules d'apprentissage antérieurs (odx) au nouveau format mlm.  
    Par contre, vous ne pouvez pas utiliser les nouveaux fichiers mlm dans d'anciennes versions de MemoryLifter (versions 2.1.et antérieures).

  • Vous pouvez exécuter MemoryLifter 2.2/2.3 et MemoryLifter 2.1 simultanément et prendre ainsi votre temps pour convertir vos modules d'apprentissage du format odx au nouveau format mlm.
