Statistiques

Affichage des statistiques d'apprentissage

Les statistiques vous donnent un aperçu de la progression de votre apprentissage.

Plus la période couverte est longue, plus elles sont particulièrement intéressantes.

 

Remarque :

Lorsque vous redémarrez l'apprentissage, les statistiques sont effacées.

 

Pour afficher les statistiques

  • Dans le menu Apprendre, cliquez sur Afficher les statistiques… ou cliquez sur dans la barre d'outils.

Remarque :

Pour afficher les statistiques, vous devez avoir utilisé et fermé le module d'apprentissage au moins une fois.

Dans le cas contraire, le message "Aucune statistique n'est encore disponible." s'affiche.

 

  • Cliquez sur les onglets pour découvrir les diverses statistiques :

    • Connaissances (cartes auxquelles vous avez correctement répondu pendant la période)

    • Répartition de la mémoire (Nombre de cartes dans chaque boîte pendant la période)

    • Autre :

      • Temps passé sur le module d'apprentissage

      • Nombre de cartes demandées

      • Nombre de bonnes réponses

      • Nombre de mauvaises réponses

      • Pourcentage de bonnes réponses

  • Dans la zone combinée Temps d'affichage :, vous pouvez sélectionner cinq périodes prédéfinies :

    • Tout

    • Aujourd'hui

    • 7 derniers jours

    • 31 derniers jours

    • 365 derniers jours

  • Cliquez sur OK pour fermer la boîte de dialogue Statistiques.
