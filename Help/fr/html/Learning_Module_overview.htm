Présentation du module d'apprentissage

 

Pour limiter le nombre de cartes affichées

Sélectionnez "Toutes les cartes" ou un chapitre spécifique dans la zone combinée Afficher le chapitre.

 

Pour trier les cartes affichées

Vous pouvez trier les cartes affichées dans l'ordre croissant ou décroissant en cliquant sur les en-têtes de colonne appropriés.  

Par exemple, pour afficher les cartes en fonction des boîtes dans lesquelles elles se trouvent, cliquez sur l'en-tête de colonne "Boîte".

 

Pour sélectionner les cartes

  1. Pour sélectionner une seule carte, cliquez sur la carte souhaitée dans la partie inférieure de la fenêtre.

Remarque : le contenu apparaît dans la partie supérieure de la fenêtre.

 

  1. Pour sélectionner plusieurs cartes, utilisez la méthode de sélection par clic MAJ ou clic CTRL de Windows.
     

Pour supprimer des cartes

  1. Sélectionnez une ou plusieurs cartes.

  2. Cliquez sur Supprimer la/les carte(s) ou faites un clic droit et sélectionnez Supprimer.

Attention :
Vous ne pouvez pas annuler cette action !

 

Pour déplacer des cartes dans un chapitre spécial

  1. Sélectionnez une ou plusieurs cartes.

  2. Effectuez un clic droit puis cliquez sur Déplacer vers, sélectionnez ensuite Chapitre, puis cliquez sur le chapitre vers lequel vous souhaitez déplacer la carte.

 

Pour déplacer des cartes dans une boîte spécifique

  1. Sélectionnez une ou plusieurs cartes.

  2. Effectuez un clic droit puis cliquez sur Déplacer vers, sélectionnez ensuite Boîte, puis cliquez sur la boîte vers laquelle vous souhaitez déplacer la(les) carte(s).

 

Pour rechercher des cartes

  1. Cliquez dans le champ Cherchez...

  2. Tapez un mot clé.
    Remarque :         Les cartes correspondant à ce mot clé s'affichent.
