Visão geral do Módulo de Aprendizagem

 

Para restringir os cartões mostrados

Selecione "Todos os cartões" ou um capítulo específico da caixa de combinação Mostrar capítulo.

 

Para classificar os cartões mostrados

Você pode classificar os cartões exibidos em ordem crescente ou decrescente clicando nos títulos de colunas apropriados.  

Por exemplo, para exibir os cartões com base nas caixas em que estão, clique no título da coluna "Caixa".

 

Para selecionar cartões

  1. Para selecionar um único cartão, clique no cartão desejado na parte inferior da janela.

Nota: O conteúdo é exibido na parte superior da janela.

 

  1. Para selecionar vários cartões, use o método Shift-clique ou Ctrl-clique padrão do Windows.
     

Para excluir cartões

  1. Selecione um ou mais cartões.

  2. Clique em Excluir cartão(ões) ou clique com o botão direito do mouse e selecione Excluir.

Atenção:
Esta ação não pode ser desfeita!

 

Para mover os cartões para um determinado capítulo

  1. Selecione um ou mais cartões.

  2. Clique com o botão direito e clique em Mover para, então clique em Capítulo, então clique no capítulo para o qual deseja mover o cartão.

 

Para mover os cartões para uma determinada caixa

  1. Selecione um ou mais cartões.

  2. Clique com o botão direito do mouse e clique em Mover para, então clique em Caixa, então clique na caixa para a qual deseja mover o cartão.

 

Para pesquisar cartões

  1. Clique no campo Procura...

  2. Digite uma palavra-chave.
    Nota:         Os cartões correspondentes à essa palavra-chave são exibidos.
