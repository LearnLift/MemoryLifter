Ativação do Módulo de Aprendizagem

Se você adquirir Módulos de Aprendizagem pré-feitos, desenvolvidos por especialistas em nosso website www.memorylifter.com, você precisará ativar seu Módulo de Aprendizagem para poder utilizá-lo.

Os Módulos de Aprendizagem marcados com um ícone de chave prata na página inicial precisam ser ativados antes de serem estudados.

 

Há duas maneiras de ativar um Módulo de Aprendizagem, dependendo se seu computador acessa a Internet ou não

ativação on-line e off-line

 

 

Para ativar um Módulo de Aprendizagem on-line

Se você abrir um Módulo de Aprendizagem com um ícone de chave prata, o assistente de ativação é iniciado.

 

 

É solicitado que você insira a licença que você recebeu por e-mail depois de adquirir o Módulo de Aprendizagem.

 

 

Após inserir a licença de 16 dígitos, a ativação é concluída.

 

 

Para ativar um Módulo de Aprendizagem off-line

 

Se o computador que você utiliza para aprender com o MemoryLifter não tiver acesso à Internet, você precisa realizar a "ativação off-line".

O Assistente de Ativação explicará cada etapa e o guiará através do procedimento.

 

 

 

 

Agora, você precisa de um computador com acesso à Internet.

Abra a seguinte página da web: http://www.memorylifter.com/web-activation.html

Insira a licença e id da máquina no formulário de Ativação do Módulo de Aprendizagem via Web.

 

 

Volte para o computador em que você está aprendendo com o MemoryLifter.

Você terá que executar o assistente de ativação novamente. Então insira o Código de Desbloqueio e o Código de Ativação.

 

 

 

 

Nota:

Sempre verifique se você está utilizando os códigos corretos correspondentes ao Módulo de Aprendizagem que você deseja ativar. Se você misturar os códigos de diversos módulos adquiridos de uma só vez, a ativação não funcionará.
