First steps

Welcome to the MemoryLifter Help.

This section will lead you through your first few steps.

 

To see a description of a procedure in detail, just click on the link.

 

To use MemoryLifter you can either

  • open an existing Learning Module or

  • create your own Learning Module

and then start learning.

 

To create your own Learning Module, follow these steps

  • create a new Learning Module

  • add cards

    • by typing text and adding images, audio or video files

    • by importing data from various sources via the clipboard

    • by importing data from text-based file formats

 

To learn with MemoryLifter

You should first get familiar with the learning window.

You can specify, which cards are asked:

  • all cards

  • cards from specified chapters

  • cards from specified boxes

 

You can modify the learning mode, i.e. the way the questions are asked, the way you can answer, and when a card is demoted or promoted.

 

Depending on which media is included in your Learning Module, cards can be shown and asked

  • with all media

  • with sound and image only (for listening comprehension or image recognition)

  • without image

MemoryLifter can also show the cards in slideshow mode.

 

You can answer

  • by typing the answer

  • by typing the answer while you are corrected on the fly

  • by typing all the synonyms

  • by selecting an answer or answers in multiple choice mode

 

You can let MemoryLifter decide, whether the question has been answered correctly, i.e. whether the card is promoted or demoted.

To specify the criteria for when MemoryLifter promotes or demotes a card:

  • set how many synonyms should be typed correctly

  • set how many typing mistakes are allowed

  • set whether upper and lower case is checked

  • set which characters are ignored during correction

 

Alternatively, you can decide by yourself, whether the question is promoted or demoted.

  • learn in self assessment mode

  • set confirmation of each demotion

  • set confirmation of each demotion for synonym or spelling mistakes

 

Notes:

  • Press F1 at any time to get help.

  • You can access this window later by clicking First steps under the Help menu.
