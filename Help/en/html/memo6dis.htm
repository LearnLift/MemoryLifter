Import Cards From File

Importing cards from a file

MemoryLifter allows you to import data from a text or csv file with one of the following delimiters.

 

  • Tab separator: word pairs are separated by tabs, cards by carriage returns

  • Semicolon separator: word pairs are separated by semicolon, cards by carriage returns

  • Comma separator: word pairs are separated by a comma, cards by carriage returns

 

Notes:

  • Currently, only text-based file formats are supported.
    If you want to import from another format (e.g. a word processor), try saving your data as a standard text file.

  • In comma-separated files, the import might fail in the case of a comma appearing within quotes, e.g. ""Give it to me", he said".

  • If special characters are not shown in your file exported from Excel, try to save it as "Unicode Text (*.txt)" rather than "CSV (Comma delimited) (*.csv)".

  • If you do not choose Unicode, Excel often does not export foreign characters (characters from a different language than the system language) correctly.

  • Select the correct character set for your data.

 

To import a file

  1. Within the Edit menu, select Import Cards => From File...

  2. In the Import dialog box, click Load File...

  3. Select the file to be imported.

  4. Click Open.

Note:

MemoryLifter will now attempt to determine the separator used in the file, load the content of the file, and let you preview the first 19 Cards.
If MemoryLifter does not succeed, type or select the appropriate file format in the File format box.

 

  1. Specify the fields to be imported using the field selector.

  2. Arrange the fields using the field selector.

Note:

Reproduce the sequence of the columns in your source file.

 

  1. Check the card preview in the Preview box.

Note:

If the preview of the columns is not correct, try to select another file format in the File format box.

If the preview of the characters is not correct, try to select another charset in the Charset (Codepage) text box.

 

  1. Clear the Contains header check box, if your source file does not have a header row.

  2. Check the Split synonyms check box, if you separated synonyms for the question or answer in the source file with commas.

  3. In the Default chapter box, type in a chapter for the imported cards.

  4. Click Import to start the import process.

  5. Click Close to start learning.
