Adding Cards to the Learning Module

To add cards to the Learning Module:

  1. In the Edit menu, click Add Card… or click Add new Card... on the toolbar.

  2. In the boxes of the Maintain Cards dialog box, type information as needed:

    • In the Question text field, type a word, phrase, or sentence for the question in the question language.
      For example: Banane

    • In the Example text field, type an example sentence using the word or phrase specified in the question.
      For example: 'Eine Banane ist eine tropische Frucht.'

    • In the Answer text field, type a word, phrase, or sentence answer in the answer language.
      For example: banana

    • In the Example text field, type an example sentence using the word or phrase specified in the answer.
      For example: 'A banana is a tropical fruit'

    • Tip:
      MemoryLifter supports synonyms in the Question and Answer fields.
      Separate synonyms by pressing ENTER.
      If you need to type special characters, click the character map button .
