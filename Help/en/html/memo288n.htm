Getting familiar with the main learning window

 

The learning window (main screen) consists of these main parts

 

  1. Menu bar

  2. Tool bar

  3. Scoring and learning progress information bar

  4. Question box

  5. Answer box

  6. Stack of the last few words not known

 

 

Menu bar

You can access the dialog boxes by clicking the commands in the five menus: File, Edit, Learn, Tools and Help.

 

With the commands in the File menu you can create new Learning Modules, open existing ones, print them, and create backups. You can also specify the Learning Module Properties.

 

With the commands in the Edit menu you can add, maintain, import, export cards, and edit the chapters.

 

With the commands in the Learn menu you can specify Learning Options, box sizes, and chapters and boxes to learn. You can also switch to slideshow mode, restart the learning session, and see statistics.

 

With the commands in the Tools menu you can apply changes to the learning window, e.g. switching the skin or the language. You can also activate the character map and switch to snooze mode.

 

With the commands in the Help menu you can get assistance with learning and creating Learning Modules.

 

 

Tool bar

With the help of the buttons and the menu on the toolbar you can access often used dialog boxes easily.

You can even change the learning modes by using the Select Learning Mode(s) menu on the toolbar.

Depending on the selected learning mode(s) the icon changes its color. For more information see Learning Options.  

 

 

Scoring and progress information bar

The scoring/progress bar gives you information on the current learning status:

 

  • the number of the box the current card is in, ranging from 1 (lowest) to 10 (highest).
    In the beginning of the learning session all cards are in the pool, so "Pool" is shown.

  • the number of cards in the box, and the maximum number of cards it can contain.
    The higher the box number, the larger the maximum size.

  • counts the number of right answers.

  • counts the number of wrong answers.

  • increases with each right answer and shows how well you know the cards of the Learning Module. For more information see Learning Progress Bar.

  • is only displayed when the timer has been activated in the learning options.
    It starts counting with a new question. The longer the question, the higher the time it starts from

  • shows the chapter the current card is in

  • shows the user who is learning at the moment

 

Question and Answer box

The questions and the answers are displayed in the two main boxes in the learning window.

 

The box on the left shows the question. Depending on card content there might be an example sentence, an image, video, and/or audio (indicated with a play button).

 

The box on the right is for the answer.

The answer box has two states:

the first one is the input mode, the second one shows the correct answer, if possible, again with sentences, audio, and image.

 

 

The stack

The stack shows you the last four cards you didn't know or only partially knew. If you do not need this information, just click on Hide Card Stack.
