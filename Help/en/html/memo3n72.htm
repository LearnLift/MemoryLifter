Opening an existing Learning Module

A Learning Module is a file containing a specific number of cards, organized in chapters.

 

Learning Modules are recognized by the
 

  • .mlm extension in MemoryLifter 2.2 and higher

  • .odx extension in previous versions of MemoryLifter.

 

 

To open an existing Learning Module (mlm or odx)

  1. Within the File menu, select Open…, press Ctrl+O, or click on the toolbar.

  2. On the start page, open the desired Learning Module from the My Learning Modules box
     

    1. by clicking on it and pressing Enter

    2. by clicking on it and then on OK

    3. by double-clicking on it

    4. by right-clicking on it and clicking Open within the menu
       

 

Notes:

  • You can add Learning Modules from different locations to the My Learning Modules box by dragging & dropping on the Start Page or directly onto a connection within the Learning Modules box.

  • MemoryLifter will automatically convert old Learning Modules (odx) to the new mlm format.  
    However, you will not be able to use the new mlm files with older MemoryLifter versions (version 2.1 and below).

  • You can visit our website at www.memorylifter.com to download pre-made, expert-developed Learning Modules.

  • The most recently used Learning Modules are listed within the File menu, in the submenu named Recent Learning Modules.
    Just select from the list to open.

  • Learning Modules are automatically installed in the folder you selected during the first startup of MemoryLifter. By default this will be the "Learning Modules" subdirectory of your "My Documents" folder.
    To change this preference, use Change Learning Module Folder within the Tools menu.
