Learning with the correct-on-the-fly option

When entering text for your answer, MemoryLifter offers the option to have your input corrected on-the-fly.

You can only enter correct characters. Otherwise, the input will stay at the same place.

 

To activate the correct-on-the-fly option

  1. Within the Learn menu, click Learning Options... or click on the toolbar.

  2. Click the Typing tab.

  3. Check Correct on the fly.

 

When using the correct-on-the-fly option, you can additionally specify how MemoryLifter should promote/demote cards depending on your typing mistakes.

 

To type your answers using correct-on-the-fly 

  1. Make sure that the answer flashcard is in input mode (answer pane contains only a centered text box).

  2. Enter the answer.
    If the character you enter is incorrect, it will become red.
    The cursor only advances when you enter the correct character.

  3. As soon as the answer is complete, the answer flash card shows the complete answer.
    If there's more than one meaning to a word, the cursor will jump to the next line after each known word before showing the complete answer.
