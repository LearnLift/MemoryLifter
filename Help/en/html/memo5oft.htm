Basic terms in MemoryLifter

Learning Module

A Learning Module is a file which contains a specific number of virtual flashcards, organized in chapters.

Learning Modules are identified by the .mlm extension in the Windows file Explorer.

 

Chapter
A chapter is used to organize the cards in a Learning Module into a logical group.

 

Card
A card is a unit of the material to learn, i.e. vocabulary, definitions, terms, dates, formulas, or any other subject matter.

It is designed like a traditional educational flashcard made of paper or cardboard, with the question on the front and the answer on the back.

In MemoryLifter you can add example sentences, sounds, videos, and images to each card.

 

Synonym
Some words have synonyms, i.e. words with the same meaning. Some questions have several correct answers. You can add these to your card so that you broaden your vocabulary and knowledge while learning with MemoryLifter.

 

"Standard" Learning Mode

When you start the learning process in standard mode, MemoryLifter will present you with a learning environment where the question displayed in a window on the left and a prompt for the answer appears on the right.

If you do not know the answer, press ENTER and the program will show you the correct answer.

Click the arrow to continue to the next card.
