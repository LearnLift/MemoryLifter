Features

MemoryLifter offers many advanced, easy-to-use features that improve your learning experience, or just make it more fun.

 

  • Automated learning with the box system.
    This learning method helps you to focus on more difficult material by presenting items that are not known at a comparatively higher frequency than those items that were known (answered correctly at least once). At the same time, items that were known, are retested again and again with increasing time between repetitions. With this method, the entire study material is eventually stored in long-term memory. Additionally, the distribution of the cards in the boxes (cards are sorted into boxes depending on how well the material is known) gives you an idea of your learning progress at a glance.
     

  • Multimedia-based learning with images, audio, and video.
    To enhance your learning experience, images, audio, and video can be added to each card, giving you a better idea how to pronounce words, or giving you more information about the card's topic.
    You can even add audio yourself using the easy-to-use, built-in audio recorder.

 

  • Individual Learning Modules

Create your own Learning Modules for specific topics with the help of MemoryLifter's import, editing, and organizing features.

    • Adding cards is easy with MemoryLifter.
      Either type your content in the text fields by hand,
      or use the Import from File feature to import data that has already been written in common, text-based formats (csv delimited, tab delimited, etc.)
      or use the Import from Clipboard feature to import data on the clipboard copied from a web page or spreadsheet.

    • Synonyms.  
      Some questions have several correct answers. MemoryLifter offers various options to add synonyms for a word in an easy, intuitive way.
      Add these to your card so that you broaden your knowledge while learning with MemoryLifter.

    • Example Sentences.
      An example sentence can be provided for each word, showing you a possible use of the word in context, or giving additional information.

    • Chapters.
      Chapters       help you to organize the cards in a Learning Module, you can organize your material logically into manageable units.
       

 Design individual cards by defining your own styles

    • change the way the learning window looks by using or modifying the pre-designed styles

    • create card styles

    • create chapter styles

 

  • Individual Learning Mode
    Choose your favorite Learning Mode.

    • Learning modes.
      There are several learning modes to choose from, including

      • standard text input

      • correct-on-the-fly

      • multiple choice with random or predefined distractors

      • listening comprehension

      • self assessment

      • random presentation

      • slideshow

      • simulating a test with the learning timer

    • Exporting, printing.
      If you need to print a set of flashcards, use the print wizard.
      Just specify the cards you want to print, whether all cards, only the unknown cards, or certain chapters, cards, or boxes.
      You can print the cards with all media, or only with the question and answer.

    • The snooze function can pause your learning after a chosen amount of time.
      It reduces MemoryLifter to an icon on the task bar, where it sleeps for a random or specified amount of time before asking you questions again.

 

  • Measure Your Success
    Statistics     provide the feedback you need to understand how you learn and how your     learning progresses over time.
