Learning cards from a certain box only

By default, cards in all boxes are asked by MemoryLifter.

However, you can choose to learn cards only from a certain box.

 

To learn cards from a certain box only

 

  • Within the Learn menu, click Select Box and then select the box you want MemoryLifter to ask the cards from.
    Note:     The selected box is marked with a checkmark.
