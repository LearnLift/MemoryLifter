Editor de estilos de fichas

Con el Editor de estilos de fichas puede modificar el estilo de las fichas, es decir, la forma en la que se muestran todos los elementos en los cuadros Pregunta y Respuesta de la ventana de aprendizaje.

 

Puede modificar el estilo de los siguientes elementos y modos:

  • Pregunta

  • Ejemplo de pregunta

  • Respuesta

  • Ejemplo de respuesta

  • Respuesta correcta

  • Respuesta incorrecta

 

Puede acceder al Editor de estilos de fichas desde los cuadros de diálogo Mantener fichas y Editar capítulos.

Los estilos definidos en el Editor de estilos de fichas pueden ajustarse para una única ficha activa en el cuadro de diálogo Mantener fichas o para un capítulo activo en el cuadro de diálogo Editar capítulos.

 

Para modificar el estilo:

  1. Abra el Editor de estilos de fichas; para ello:

    • Haga clic en Añadir/editar estilo de ficha en el cuadro de diálogo Mantener fichas.

    • Como alternativa, haga clic en Añadir/editar estilo en el cuadro de diálogo Editar capítulos.

  2. En el cuadro de diálogo Editor de estilos de fichas, haga clic en la ficha del elemento o modo cuyo estilo desea modificar.

  3. Marque la casilla de verificación Color de primer plano para elegir un color.

  4. Marque la casilla de verificación Color de fondo para elegir un color de fondo.

  5. Marque la casilla de verificación Familia de fuente para especificar una fuente.

  6. En el cuadro Estilo de fuente, desmarque la casilla de verificación Ninguno para seleccionar un estilo de fuente (negrita, tachado, normal, cursiva o subrayado).

  7. Marque la casilla de verificación Tamaño de fuente para seleccionar o escribir un tamaño de fuente.

  8. Marque la casilla de verificación Alineación horizontal para seleccionar o escribir el valor de alineación.

  9. Marque la casilla de verificación Alineación vertical para seleccionar o escribir el valor de alineación.

  10. En el cuadro Otros elementos añada otros elementos CSS para personalizar aún más el estilo.

  11. Haga clic en Aceptar para aplicar el estilo.

Nota:
En la parte derecha se muestra una vista previa.
En cualquier momento puede hacer clic en Borrar el estilo para restablecer el estilo al predeterminado.
