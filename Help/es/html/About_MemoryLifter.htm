Acerca de MemoryLifter

MemoryLifter™ es un software educativo freeware.

Con MemoryLifter, podrá:

 

  • ampliar su vocabulario;

  • estudiar su asignatura favorita; o

  • simplemente recordar datos.

 

MemoryLifter utiliza el sistema de ficheros de Leitner, una técnica científica de aprendizaje que ha demostrado que mejora el proceso de memorización para conseguir una mejor retención en la memoria a corto y largo plazo.  El sistema Leitner combina las repeticiones espaciadas con un mecanismo que centra el aprendizaje en el material que aún no se domina, lo que le permite recordar datos mucho más rápidamente y retenerlos durante más tiempo.

 

MemoryLifter está diseñado para todos los usuarios, incluidos estudiantes, padres, profesores y profesionales.  Con MemoryLifter, puede crear fácilmente un módulo de aprendizaje propio de fichas didácticas multimedia virtuales. El programa facilita la gestión de las fichas didácticas y realiza el seguimiento de su evolución del aprendizaje.  Puede crear sus propias fichas didácticas con texto, sonido, imágenes y vídeos, o aprender con módulos de aprendizaje desarrollados por expertos listo para utilizar disponibles en la sección de descarga de módulos de aprendizaje del sitio web MemoryLifter.

 

Las características principales de MemoryLifter son:

 

  1. Aprendizaje automatizado
    Automatice el aprendizaje con el sistema de Sebastian Leitner basado en fichas didácticas de ficheros.
    El planificador de aprendizaje le recordará cuándo tiene que repetir las fichas.

  2. Aprendizaje basado en multimedia
    Incluya elementos multimedia como imágenes, audio y vídeo en los módulos de aprendizaje para mejorar la experiencia de aprendizaje.

  3. Aprendizaje individual
    Cree sus propios módulos de aprendizaje para temas específicos con la ayuda de las funciones de importación y edición de MemoryLifter.
    Diseñe las fichas individuales definiendo sus propios estilos.
    Elija su modo de aprendizaje favorito: introducción de texto estándar, selección múltiple con elementos de distracción predefinidos o aleatorios, comprensión de idioma hablado, autoevaluación o el modo de presentación en diapositivas.

  4. Determinación del éxito
    Reciba datos estadísticos que le ayudarán a comprender cómo aprende y a mejorar la eficiencia en el aprendizaje.
