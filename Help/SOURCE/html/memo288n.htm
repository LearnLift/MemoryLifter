Getting familiar withe the main learning window

 

The learning window (main screen) is the integral part of MemoryLifter. It is comprised of three parts:

 

  • The question and the answer flashcards
  • Scoring/progress information
  • Stack (of the last few words not known)

 

Below are more information on the three parts and instruction on how to answer the question and how to end a learning session.

 

Question and answer flashcards

The question and the answer flashcards are represented by two boxes in the Learning Window. The card on the left shows the question. Depending on their availability (can vary for each card) there might be a sample sentence (smaller, below the word), an image or audio (indicated with a Play button).

 

The card on the right is for the answer. The answer flashcard has two states: the first one is the input mode, the second one shows the answer in its full form, if possible, again with sentences, audio, and image.

 

Scoring/Progress information

The scoring/progress part gives you some information on the current learning status:

  • Box number is the number of the box the current card is in, ranging from 1 (lowest) to 10 (highest).
  • Box size shows you how many cards are actually in the box, and the maximum cards it can contain. The higher the box number, the higher the maximum size.
  • Right / wrong answers counts the number of right / wrong answers. Note that when a word has multiple meanings, each right / wrong answer will yield you a point.
  • The Score increases with a right word, depending on speed, length of word, as well as the number of right words in a row.
  • The Timer is only displayed when it has been selected in the learning options. It starts counting with a new question. The longer the question, the higher the time it starts from

 

The stack

The stack shows you the last few cards you didn't know or only partially knew.

 

Answering a question

The answer to the question can be in the form of a standard input or multiple choice depending on the learning mode. Below are the instructions on answering the two types.

 

Standard input

There are two possibilities for the standard input: it can behave like a standard edit control as known from Windows, or correct your input on the fly.

 

Correct on the fly

  1. Make sure that the answer flashcard is in input mode (flashcard is empty apart from a cursor).

  2. Enter the answer. If the character you enter is incorrect, it will become red. The cursor only advances when you enter the correct character.

  3. As soon as the answer is complete, the answer flash card shows the complete answer. If there's more than one meaning to a word, the cursor will jump to the next line after each known word before showing the complete answer.

 

Standard input

  1. Make sure that the answer flashcard is in input mode (flashcard is empty apart from a cursor).

  2. Enter the answer like in a standard edit control. Use Ctrl-ENTER to advance. If there are meaning to a word, the cursor jumps to a new line, prompting you for another meaning. Press Enter to skip this.

  3. By pressing Ctrl-ENTER with no meaning left, or just ENTER, the answer flash card will show the complete answer.

 

Hints

  • If you do not know the answer, press Enter and MemoryLifter will show the answer(s).
  • Click the OK button for the next question, or press ENTER.
  • If you need special characters, use the character map.

 

Multiple choice

  1. Click the answer you think is the right.

  2. Click the OK button.

 

If you don't know a word

  1. Press ENTER.

  2. The correct answer is shown, the card is added to the stack. It will be asked again after a short time.

 

To edit a card

In case you want to modify a card while learning, press F10. The Maintain cards window is shown, and you can change the given word. Click Change to apply changes, then Close to continue learning.

 

Ending a learning session

  1. Click the X in the upper right corner

  2. If you want to end your learning session, answer the new dialog box with Yes, Cancel brings you back to learning.

  3. Statistics for this round are shown, click OK
