Importing data from a file

MemoryLifter allows you to import data from a text or csv file with the following delimiter.

 

  • Tab delimited: word pairs are seperated by tabs, cards by carriage returns
  • Semicolon-delimited: word pairs are seperated by semicolon, cards by carriage returns
  • Comma delimited: word pairs are seperated by a comma, cards by carriage returns
 

Notes:

  • Currently, only text-based file formats are supported. If you want to import from another format (e.g. a word processor), try saving its data as standard text file.
  • In comma-delimited files, the import might fail in case the string ", appears inside quotes, e.g. ""Give it to me", he said".
 

To import a file

  1. On the File menu, select Import.

  2. In the Import dialog box, click Load...

  3. Select the file to be imported.

  4. Click Open.  MemoryLifter will now attempt to determine the delimiter used in the file, load the content of the file and let you preview the first five cards.

  5. Select the appropriate file format if MemoryLifter could not determine it properly.

  6. Select the fields that are in the table, as well as their order using the field selector.

  7. Check the card preview on the right. If the first few cards seem to be correct, continue with step 8.

  8. Select a chapter where you want to put the cards from the Chapter: combo box.
    Note:     if your file already contains chapter information recognized by MemoryLifter, this combo box is disabled.

  9. Click Import to start the import process.
