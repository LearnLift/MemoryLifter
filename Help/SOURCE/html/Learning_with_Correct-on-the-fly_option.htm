Learning with "Correct on the fly" option

When entering text for your answer during learning, MemoryLifter offers the option to have your input to be corrected on the fly -- that is, you can only enter correct characters, otherwise, the input will stay at the same place.  

 

  1. From the Tools menu, select Learning Options.

  2. Click the Typing tab.

  3. Select Correct on the fly.

 

When using "Correct on the fly" option, you can additionally specify how MemoryLifter should promote/demote cards depending on your typing mistakes.
