Opening an existing dictionary or a packed dictionary

A dictionary is a file containing a specific number of cards, ordered logically by chapters. Dictionaries are recognized by the .ODX extension in the new MemoryLifter version 2.0 and .ODF extension in previous version of MemoryLifter.

 

DZP (Dictionary Zipped) file is a compressed dictionary which allows easy distribution of the dictionary to friends, colleagues, and others.

 

Both dictionary and packed dictionary can be opened through this option.

 

To open an existing dictionary (odx or odf)

  1. On the File menu, select Open… or click on the toolbar.

  2. Select the desired dictionary.  Note: MemoryLifter will automatically convert the previous dictionary (odf) to the new odx format.  However, you will not able to use the new .odx files with older MemoryLifter versions (version 1.x and below).

  3. Click Open to open the dictionary.

 

Note: You can visit our website at www.memorylifter.com to download some pre-made dictionaries.

 

To open a packed dictionary (dzp)

  1. From the File menu, select Open… or click on the toolbar.

  2. Select the desired dzp file.

  3. Click Open.

  4. Select the destination folder to extract the dictionary.

  5. Click OK to unpack the dzp file and open the extracted dictionary.

 

Notes

  • The last 4 opened dictionaries are listed under the File menu. Just select from the list to open.
  • Dictionaries are automatically installed in the "MLDictionaries" subdirectory of your "My Documents" folder. To change this preference, use Specify Dictionary folder from the Tools menu.
