Restart

 

You can restart your current learning session and empty all boxes, thus resetting the dictionary to a clean state where no learning has taken place.  As a precautionary measure, MemoryLifter makes a backup file of the current state of the dictionary.

 

To restart a learning session:

  1. From the Learn menu, select Restart.

  2. MemoryLifter will ask you if you really want to wipe all boxes – click Yes.

  3. MemoryLifter will inform you that it has saved a backup file – click OK.

  4. At the Chapter Selection dialogue box, select the chapters to learn with the chapter selector.

  5. Click OK.
