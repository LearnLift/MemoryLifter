Change Style

You can change the appearance of MemoryLifter's learning window. You can choose between 5 pre-defined styles:

 

  • Active

  • Covered

  • Default

  • Kids

  • Simple

 

To change the style:

 

  1. From the Tools menu, select Change Style.

  2. Select the style you like. The selected style is marked with a checkmark.
