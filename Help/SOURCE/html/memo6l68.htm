Show character map

With the character map, you can choose and enter some special or foreign characters that you need for your cards but are not on your keyboard

 

To use the character map

  1. From the Tools menu, select Show Character Map or click on the toolbar. Note: The button will be down when selected.

  2. Click the character in symbol box you wish to add.

 

To stop using the character map

1.From the Tools menu, select Show Character Map or click on the toolbar. Note: The button will be up when unselected.
