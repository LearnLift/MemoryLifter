Learning cards from a certain box only

By default, cards in all boxes are asked by MemoryLifter. However, you can choose to learn cards from a certain box only.

 

To learn cards from a certain box only:

 

From the Learn menu, select Learn Box and choose the box you want MemoryLifter to ask the cards from. The selected box is marked with a checkmark.
