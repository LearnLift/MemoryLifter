Alterar idioma

O Memory Lifter está disponível nos idiomas a seguir:

 

  • Inglês

  • Francês

  • Alemão

  • Português

  • Espanhol

 

 

Para alterar o idioma

  • No menu Ferramentas, clique em Alterar idioma, em seguida clique no idioma desejado.

Nota:

O idioma selecionado é marcado com um sinal de verificação.
