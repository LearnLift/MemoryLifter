Novidades

Bem-vindo ao MemoryLifter™

 

Benefícios

  • Novo formado de arquivo (inclusive novo ícone de arquivo do Módulo de Aprendizagem):

    • todos os dados armazenados em um arquivo (sem a necessidade de arquivos adicionais de mídia ou pastas)

    • o progresso de aprendizagem de mais de um aprendiz pode ser armazenado por arquivo

    • segurança de dados aumentada

  • Abrir/aprender Módulos de Aprendizagem a partir de banco de dados central no Servidor MemoryLifter (obtenha o software de servidor gratuito em www.memorylifter.com)

  • Fácil gerenciamento dos Módulos de Aprendizagem com a página inicial ("Meus Módulos de Aprendizagem"), em que você pode acessar facilmente todos os seus módulos de aprendizagem (locais e baseados no servidor)

 

Aprimoramentos:

  • Novas skins para interface de usuário

  • Entrada de múltiplas escolhas tem um novo visual

  • Os menus foram simplificados e os ícones são iguais aos ícones na barra de ferramentas

  • A barra de ferramentas foi simplificada

  • Novo algoritmo de pontuação (baseado no percentual de cartões conhecidos no Módulo de Aprendizagem)

  • Aprimoramentos gerais na interface do usuário

  • Novo feedback de resposta

    • não é mais misturado com a atribuição do lado da resposta (também pode ser fechado)

    • sinônimos corretos/errados realçados

  • Aperfeiçoamentos de desempenho

  • Muitos aperfeiçoamentos secundários

 

Correções de problemas:

  • A impressão não gera mais páginas vazias

  • Alguns problemas de renderização com fontes Asiáticas corrigidos

  • Algumas fontes corrigidas que exibiam exceção quando um cartão era distribuído

  • Muitas correções de pequenos erros

 

Notas:

  • O MemoryLifter converterá automaticamente os Módulos de Aprendizagem (odx) anteriores para o novo formato mlm.  
    No entanto, você não poderá usar os novos arquivos mlm com as versões anteriores do MemoryLifter (versão 2.1.e anteriores).

  • Você pode executar o MemoryLifter 2.2/2.3 e o MemoryLifter 2.1 simultaneamente, portanto é possível mover seus Módulos de Aprendizagem do formato odx para o novo formato mlm.
