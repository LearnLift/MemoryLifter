(CSV é uma implementação do formato de arquivo de valores separados por delimitador, que pode usar qualquer caractere como delimitador. Muitos aplicativos como o Microsoft Excel™ permitem que arquivos com a extensão csv usem qualquer caractere delimitador.)

(Os capítulos são projetados para ajudá-lo a organizar os cartões no dicionário em tópicos lógicos -- por exemplo, Restaurantes, Cumprimentos, Compras.)

Importar Cartões do Arquivo

Importar cartões de um arquivo

O MemoryLifter permite importar dados de um texto ou arquivo csv com um dos seguintes delimitadores.

 

  • Separador de guia: pares de palavras são separados por tabulações, cartões por retornos de carro

  • Delimitado por ponto-e-vírgula: pares de palavra são separados por ponto-e-vírgula, cartões por retorno de carro

  • Delimitado por vírgulas: pares de palavras são separados por vírgula, cartões por retorno de carro

 

Notas:

  • Atualmente, são suportados apenas os formatos de arquivo baseados em texto.
    Se quiser importar a partir de outro formato (por exemplo, um processador de texto), tente salvar seus dados como arquivo de texto padrão.

  • Em arquivos delimitados por vírgula, a importação pode falhar caso uma vírgula apareça entre aspas, por exemplo, ""Passe para mim", ele disse".

  • Se caracteres especiais não forem mostrados no seu arquivo exportado do Excel, tente salvá-lo como Texto Unicode (*.txt)", em vez de "CSV (delimitado por vírgula) (*.csv)".

  • Se você não escolher o Unicode, o Excel não exporta corretamente os caracteres estrangeiros (caracteres de um idioma diferente do idioma do sistema).

  • Selecione o conjunto correto de caracteres para seus dados.

 

Para importar um arquivo

  1. No menu Editar, selecione Importar Cartões=> a partir do Arquivo...

  2. Na caixa de diálogo Importar, clique em Carregar arquivo...

  3. Selecione o arquivo a ser importado.

  4. Clique em Abrir.

Nota:

Agora, o MemoryLifter tentará determinar o separador usado no arquivo, carregará o conteúdo do arquivo e permitirá que você veja os primeiros 19 cartões.
Se o MemoryLifter não conseguir, digite ou selecione o formato de arquivo apropriado na caixa Formato de arquivo.

 

  1. Especifique os campos a serem importados usando o seletor de campos.

  2. Organize os campos usando o seletor de campos.

Nota:

Reproduza a seqüência das colunas no arquivo de origem.

 

  1. Selecione a visualização do cartão na caixa Visualizar.

Nota:

Se a visualização das colunas não estiver correta, tente selecionar outro formato de arquivo na caixa Formato de arquivo.

Se a visualização dos caracteres não estiver correta, tente selecionar outro conjunto de caracteres na caixa de texto Conj. caracteres (página de códigos).

 

  1. Desmarque a caixa de seleção Contém cabeçalho, se o arquivo de origem não possuir uma linha de cabeçalho.

  2. Marque a caixa de seleção Separação de sinônimos se você separar os sinônimos da pergunta ou resposta no arquivo de origem por vírgulas.

  3. Na caixa Capítulo padrão, digite um capítulo para os cartões importados.

  4. Clique em Importar para iniciar o processo de importação.

  5. Clique em Fechar para iniciar a aprendizagem.
