Estilos de Módulo de Aprendizagem

Além disso, expandimos a capacidade de modificar estilos.

Agora, você pode modificar o estilo

  • de todos os estilos predefinidos
    (para todos os Módulos de Aprendizagem abertos no futuro)
    Para modificar todos os estilos predefinidos, selecione as configurações globais no Editor de Estilos.

  • do estilo ativo apenas
    (para todos os Módulos de Aprendizagem abertos no futuro)
    Para modificar o estilo ativo predefinido, selecione as configurações de estilo no Editor de Estilo

  • dos capítulos
    (para o Módulo de Aprendizagem ativo apenas quando Use os estilos do Módulo de Aprendizagem estiver selecionado no menu Ferramentas)
    Para modificar o estilo de um capítulo, abra o Editor de estilo do cartão na caixa de diálogo Gerenciar capítulos.

  • dos cartões
    (para o Módulo de Aprendizagem ativo apenas quandoUse os estilos do Módulo de Aprendizagem estiver selecionado no menu Ferramentas)
    Para modificar o estilo de um cartão, abra o Editor de estilo do cartão na caixa de diálogo Manter cartões.

Observe que as propriedades são herdadas.

 

Exportar

Para salvar seu Módulo de Aprendizagem em outros formatos, você pode exportá-lo.

 

Se quiser modificar seus cartões com outra ferramenta de software ou quiser alterar a ordem dos campos de pergunta, resposta e exemplos ou mesmo adicionar informações de capítulo a seus cartões de uma maneira rápida e fácil, exporte seu Módulo de Aprendizagem ou seus campos especificados primeiramente para um arquivo de texto.

 

Importar

Recomendamos importar arquivos de texto, se você estiver familiarizado a salvar tabelas em formato delimitado.

Embora seja recomendável importar a partir da área de transferência, se quiser, você pode importar dados enquanto navega pela Internet ou se deseja importar dados diretamente de outros documentos.
