Field selector  

In the MemoryLifter dialog boxes Import from File and Export, you can arrange the various parts ("fields") of a card.

 

In the Export dialog box, you can specify which fields you want to export, as well as the order in which the program should arrange them in the exported file.

 

In the Import from File dialog box, you can specify which fields you want to import, as well as the order in which the program should import them.

 

The list on the left contains the available fields. This is either just the two words of a card and their sample sentences, or the attached media as well.

 

The list on the right contains the selected fields.

 

Note: By default, the question and answer fields are selected.

 

To add a field

  1. Select the field you want to add from the left side by clicking it.
