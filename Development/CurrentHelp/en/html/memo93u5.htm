The box system

MemoryLifter uses virtual flashcards and the Leitner box system to augment your learning ability.

 

The box system

The box system (also called "Leitner system") was originally conceived by German learning-psychologist Sebastian Leitner in the 1970s.

Its purpose is to provide a structure to help people learn and retain information in short-term and long-term memory.

 

In the box system flashcards are assigned to a set of x boxes (10 boxes in the case of MemoryLifter).

Box sizes increase with the box's number.

By the time a card has been promoted to the highest box, the material should be fixed in long-term memory.

 

The quantity of flashcards inside each box level provides a quick feedback of the current state of the user's knowledge and memory retention (short-term and/or long-term).

 

MemoryLifter's box system algorithm

MemoryLifter uses a slightly modified version of the original box system algorithm to present and track the cards for learning.

 

  1. Initially, all cards are pulled from the pool, which contains all available cards.

  2. MemoryLifter checks whether any of the boxes are full.
    If so, it asks a card from that box.
    If none of the boxes are full, MemoryLifter places a new card from the pool in the first box and asks it.

  3. If a card is answered correctly, MemoryLifter promotes it to the next box.
    Otherwise, it is demoted to box 1.

  4. When placing a card in a box, MemoryLifter puts it at the end of the box.
    When asking cards from the box, they are asked in the order they were placed in the box (first in - first out principle).

  5. MemoryLifter continues with step 2 to 4 until the pool is empty.

  6. If the pool is empty, the oldest card from the Learning Module will be asked.
    A message box will be opened to inform you of this.
