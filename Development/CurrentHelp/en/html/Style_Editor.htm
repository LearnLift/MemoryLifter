Style Editor

Changing the font style, font size, color, and more of the question and answer

The way the questions and answers look in the MemoryLifter learning window is controlled by styles.  

Styles contain information like font type, font color, font size, background color, etc...

It is basically a CSS stylesheet for those who are familiar with CSS technology.  

The question and answer each have their own style.

 

MemoryLifter provides a few pre-designed styles: Default, BumbleBee, iLifter, Kids, and Tropical Beach.  

You can modify these pre-designed styles using the Style Editor.

 

Additionally you can specify styles for single cards or chapters using the Cardstyle Editor.

 

 

To modify the style currently in use

 

  1. Within the Tools menu, click Style Editor…

  2. Choose whether you want to change Global settings (all properties of the style) or Style settings (only the most important properties of the style).

Note:

The following procedures describe changing style settings.

If you choose to change global settings, there are many more elements which you can modify, but the procedure remains the same.

 

  1. Choose your preferred View by clicking either the Table - View tab or the Text - View tab.

 

To modify the style using the Table - View

  1. You can change the appearance for the following categories:

    • Question (question)

    • Question Example (questionExample)

    • Answer  (answer)

    • Answer Example (answerExample)

    • Correct Answer (answerCorrect)

    • Wrong Answer (answerWrong)

  2. In the Table - View tab choose the row of the element or mode, for which you want to modify the style.

  3. Select the Text-Color box to choose a color.

  4. Select the Back-Color box to choose a color for the background.

  5. Click Font... to specify a font, font style, and size.

  6. Select the H-Align box to select or type an alignment value (none, left, center, right).

  7. Select the V-Align box to select or type an alignment value (none, top, middle, bottom).

  8. Repeat for each element you want to modify.

  9. Click Close to apply the style.

Notes:

  • A preview is displayed on the right hand side.
    You can choose, whether you want to see the Question, Answer correct or Answer wrong preview.

  • You can click Restore at any time to reset.

 

To modify the style using the Text - View

  1. You can change the appearance for the following categories:

    • Question

    • Question Example

    • Answer

    • Answer Example

    • Correct Answer's background color

    • Wrong Answer's background color

Caution:
When changing values in the Text - View tab, DO NOT delete the semi-colon (;) at the end of the value.

 

  1. Find the appropriate section in the Text - View tab and change the values as desired.

Question

    • Find the ".question" section.

Question Example

    • Find the ".questionExample" section.

Answer

    • Find the ".answer" section.

Answer Example

    • Find the ".answerExample" section.

 

    • Change the value of the "color" as desired.

    • Change the value of the "font-family" as desired.
      Use the exact font name as you would see in Windows.

    • Change the value of the "font-size" as desired.

    • Change the value of the "font-style" as desired.

    • Change the value of the "font-weight" as desired.

 

Correct Answer's background color

    • Find the ".answerCorrect" section.

Wrong Answer's background color

    • Find the ".answerWrong" section.

 

    • Change the value of the "background-color" as desired.

 

  1. Click Close to apply the style.

 

Notes:

  • A preview is displayed on the right hand side.
    You can choose, whether you want to see the Question, Answer correct or Answer wrong preview.

  • You can click Restore at any time to undo all your changes.
