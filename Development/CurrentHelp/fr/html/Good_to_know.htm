Bon à savoir

Si vous avez déjà utilisé MemoryLifter, nous souhaiterions vous communiquer quelques informations sur les nouvelles fonctionnalités et astuces intégrées du logiciel.

 

Styles des modules d'apprentissage

Nous avons développé les possibilités de modification des styles.

Vous pouvez désormais modifier le style
 

  • de tous les styles prédéfinis
    (pour tous les modules d'apprentissage que vous ouvrirez ultérieurement)
    Pour modifier tous les styles prédéfinis, sélectionnez les paramètres généraux dans l'éditeur de style.

  • du style actif seulement
    (pour tous les modules d'apprentissage que vous ouvrirez ultérieurement)
    Pour ne modifier que le style prédéfini actif, sélectionnez les paramètres du style dans l'éditeur de style.

  • des chapitres
    (du module d'apprentissage actif uniquement, lorsque Utiliser les styles du module d'apprentissage est sélectionné dans le menu Outils)
    Pour modifier le style d'un chapitre, ouvrez l'éditeur de style de carte dans la boîte de dialogue Gestion des chapitres.

  • des cartes
    (du module d'apprentissage actif uniquement, lorsque Utiliser les styles du module d'apprentissage est sélectionné dans le menu Outils)
    Pour modifier le style d'une carte, ouvrez l'éditeur de style de carte dans la boîte de dialogue Maintenir les cartes.

Notez que les propriétés sont héritées.

 

Exporter

Pour enregistrer votre module d'apprentissage dans un format différent, vous pouvez l'exporter.

 

Si vous souhaitez modifier vos cartes à l'aide d'un autre logiciel, modifier l'ordre des champs (question, réponse, exemple) ou compléter vos cartes par des informations sur le chapitre rapidement et facilement, exportez le module d'apprentissage ou les champs spécifiés dans des fichiers texte.

 

Importer

Nous vous recommandons d'importer des fichiers texte si vous maîtrisez correctement l'enregistrement des tables dans un format délimité.

Nous vous recommandons plutôt la l'importation depuis le presse-papiers si vous souhaitez importer des données tout en surfant sur Internet ou directement depuis d'autres documents.
