Apprentissage en mode d'auto-évaluation

MemoryLifter permet un apprentissage en mode d'auto-évaluation, ce qui vous permet de juger par vous-même si votre réponse est exacte ou non.  

 

Pour apprendre en mode révision

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

  2. Cliquez sur l'onglet Saisie.

  3. Sélectionnez Auto-évaluation.

  4. Cliquez sur OK.

  5. Saisissez votre réponse.

  6. Cliquez sur le bouton d'entrée (en fonction de l'habillage, et le bouton "OK" ou le bouton matérialisé par une ampoule).

  7. La réponse correcte s'affiche. Vous pouvez alors cliquer sur le bouton Je sais ou J'ignore (habituellement les boutons "pouces vers le haut/bas" ou "sourire/grimace" en regard des touches fléchées gauche/droite) selon que vous estimez avoir bien ou mal répondu à la carte.
    Vous pouvez également cliquer sur Ne plus redemander, si vous pensez maîtriser suffisamment la carte ou si vous ne souhaitez pas l'apprendre.
    Remarque :
    Si vous cliquez sur Ne plus redemander, la carte bascule dans le champ "Aucune".
    Cette action revient à désélectionner l'option La carte est active dans la boîte de dialogue Maintenir les cartes.
