Importation de cartes depuis le presse-papiers

MemoryLifter vous permet d'importer des cartes depuis des pages web et des feuilles de calcul Excel copiées dans le presse-papiers.

 

Importation depuis le presse-papiers

  1. Dans le menu Edition, cliquez sur Importation de cartes puis choisissez Depuis presse-papiers....

  2. Dans la boîte de dialogue Importer depuis presse-papiers, sélectionnez les champs à importer.

  3. Effectuez un tri,

    • en sélectionnant le champ à déplacer vers le haut ou le bas dans la liste et

    • en cliquant sur les flèches ou en déplaçant l'élément vers le haut ou le bas.

      Remarque :

Il n'est pas nécessaire que tous les champs entre les éléments sélectionnés soient cochés car MemoryLifter importe uniquement les éléments sélectionnés dans l'ordre dans lequel ils apparaissent dans la liste.
 

  1. Cliquez sur Terminer pour ouvrir MemoryLifter CardCollector.

  1. Ouvrez votre fichier source, sélectionnez les données à collecter dans le fichier source et copiez-les dans le presse-papiers.

Remarque :

Si vous avez sélectionné "Avertissement de ligne complète" dans CardCollector, vous entendrez un bip.

 

  1. Revenez dans MemoryLifter pour afficher les données collectées dans le tableau ci-dessous.

  2. Cliquez sur Arrêter.

Attention :
A ce stade, vos données n'ont pas encore été sauvegardées !
Ne vous contentez pas de fermer CardCollector en cliquant sur le bouton Fermer !
 

  1. Sélectionnez une carte dans le tableau ci-dessous de manière à faire apparaître les données collectées dans les champs de texte connexes.

  2. Ajoutez la carte à votre module d'apprentissage en cliquant sur Ajouter la carte.

  3. Répétez la même opération pour toutes les cartes à ajouter.

  4. Fermez la fenêtre pour commencer l'apprentissage.
