Propriétés

Définition des options du module d'apprentissage

Dans le menu Fichier, cliquez sur Propriétés... pour afficher la boîte de dialogue Options du module d'apprentissage et ses onglets :

 

  • Propriétés

  • Informations

  • Titres / Langues

  • Options d'apprentissage autorisées

  • Avancé

 

 

Propriétés, onglet

Le nom du module d'apprentissage apparaît dans la zone de texte Nom.

  1. Dans la zone combinée Catégorie, saisissez ou sélectionnez la catégorie à laquelle le module d'apprentissage appartient.

  2. Vous pouvez indiquer le nom de l'auteur du module d'apprentissage dans la zone de texte Auteur.

  3. Vous pouvez saisir une brève description de votre module d'apprentissage dans la zone de texte Description.

 

Informations, onglet

L'emplacement du module d'apprentissage affiché apparaît dans la zone de texte Emplacement du fichier.

Vous pouvez également obtenir des informations sur le nombre de cartes que contient votre module d'apprentissage, l'espace disque utilisé, le nombre de fichiers média, votre score et votre meilleur score.

 

Titres / Langues, onglet

  • Dans les zones de texte Titre, tapez les titres des volets de question et de réponse.

    Exemple :

Si vous travaillez un module d'apprentissage "Français - Anglais", saisissez "Français" dans la zone de texte Titre correspondant à la question et "Anglais" dans celle correspondant à la réponse.

Si vous travaillez un module d'apprentissage sur la "Terminologie médicale", saisissez "Définition" dans la zone de texte Titre correspondant à la question et "Terme" dans celle correspondant à la réponse.

 

  • Dans les zones de texte Langue, saisissez ou sélectionnez la langue des questions et des réponses du module d'apprentissage.

 

Options d'apprentissage autorisées, onglet

Pour spécifier dans quels modes d'apprentissage vous pouvez apprendre les cartes du module d'apprentissage, sélectionnez diverses options dans les champs Modes d'apprentissage et Direction.

Le contenu détermine les modes d'apprentissage appropriés.

Par exemple, vous pouvez apprendre du vocabulaire et les définitions correspondantes dans les deux sens.

 

  • Pour déterminer le sens que vous préconisez pour l'apprentissage, sélectionnez les options appropriées.

 

Remarque :

Le nom des cases est adapté aux titres saisis dans les zones de texte Titre de l'onglet Titres / Langues.

 

  • Pour déterminer les modes d'apprentissage que vous préconisez pour l'apprentissage, sélectionnez les options appropriées.

 

Remarque :

Les options choisies ont une incidence sur la boîte de dialogue Options d'apprentissage dans laquelle vous spécifiez le mode d'apprentissage de la session d'apprentissage en cours.

Les sens et les modes d'apprentissage que vous désactivez sont grisés et vous ne pouvez plus les sélectionner pour configurer vos sessions d'apprentissage.

 
 

Avancé, onglet

Les options avancées vous permettent :

 

  • de spécifier les fichiers audio à écouter pendant l'apprentissage ; et

  • d'ajouter ou de modifier le style par défaut du module d'apprentissage

 

Deux groupes de commentaires audio comportant six sons chacun existent :

Le premier groupe, visible dans le volet gauche, est audible lorsque le mode d'apprentissage fonctionne dans le sens "Question => Réponse" (par défaut).

Le second groupe, visible dans le volet droit, est audible lorsque le mode d'apprentissage fonctionne dans le sens "Réponse => Question".

 

Chaque groupe est divisé en deux sous-groupes :

 

Les trois premiers fichiers audio sont "autonomes". Ils sont utilisés lorsque aucun son n'est associé à la réponse de la carte. Le commentaire peut se contenter d'apporter une réponse simple, "Bonne réponse", par exemple.

 

Le second sous-groupe est utilisé lorsque des données audio sont associées à la carte. Ce second commentaire, par exemple, "Vous vous êtes trompé. La bonne réponse est", est suivi de l'élément audio de la bonne réponse (qui peut être défini sur chaque carte individuelle).

 

 

Pour modifier les commentaires audio

  1. Cliquez sur le commentaire approprié dans le champ Question ou Réponse.

  2. Pour charger un fichier audio existant ou enregistrer un fichier audio pour le commentaire sélectionné, cliquez sur Modifier le son .

  3. L'enregistreur de sons standard s'ouvre.

 

Pour ajouter ou modifier le style par défaut du module d'apprentissage

  • Cliquez sur Ajouter/Editer un style.

  • Utilisez l'éditeur de style de carte.
