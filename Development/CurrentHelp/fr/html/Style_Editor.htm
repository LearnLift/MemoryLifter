Editeur de style

Changement de style de police, de taille de caractères, de couleur et autres informations sur les questions/réponses

L'apparence des questions et des réponses dans la fenêtre d'apprentissage de MemoryLifter est contrôlée par les styles.  

Les styles contiennent des informations comme le type, la couleur et la taille des caractères, la couleur de fond, etc.

Il s'agit fondamentalement d'une feuille de style CSS pour ceux qui maîtrisent cette technologie.  

Question et réponse ont chacune leur propre style.

 

MemoryLifter propose plusieurs styles prédéfinis : Default, BumbleBee, iLifter, Kids et Tropical Beach.  

Vous pouvez modifier ces styles prédéfinis à l'aide de l'éditeur de style.

 

Vous pouvez également spécifier les styles par carte ou chapitre dans l'éditeur de style de carte.

 

 

Pour modifier le style en cours d'utilisation

 

  1. Dans le menu Outils, cliquez sur Editeur de style…

  2. Choisissez si vous souhaitez modifier les Paramètres globaux (toutes les propriétés du style) ou les Paramètres de style (uniquement les propriétés les plus importantes du style).

Remarque :

Les procédures suivantes décrivent comment modifier les paramètres de style.

Si vous choisissez de modifier les paramètres globaux, la procédure reste la même même si le nombre d'éléments que vous pouvez modifier est bien plus important.

 

  1. Sélectionnez votre vue favorite en cliquant sur l'onglet Table - Vue ou sur l'onglet Texte- Vue.

 

Pour modifier le style à l'aide de Table - Vue

  1. Vous pouvez modifier l'apparence des catégories suivantes :

    • Question (question)

    • Exemple de question (exempleQuestion)

    • Réponse  (réponse)

    • Exemple de réponse (exempleRéponse)

    • Réponse correcte (réponseCorrecte)

    • Réponse erronée (réponseErronée)

  2. Dans l'onglet Table - Vue, sélectionnez la ligne de l'élément ou mode dont vous souhaitez modifier le style.

  3. Cochez la case Texte - Couleur pour choisir une couleur.

  4. Cochez la case Couleur fond pour sélectionner une couleur d'arrière-plan.

  5. Cliquez sur Police... pour spécifier une police, un style de police et la taille.

  6. Cochez la case Alignement H pour sélectionner ou saisir une valeur d'alignement (aucune, gauche, centrale, droite).

  7. Cochez la case Alignement V pour sélectionner ou saisir une valeur d'alignement (aucune, haut, centre, bas).

  8. Répétez cette opération pour chaque élément à modifier.

  9. Cliquez sur Fermer pour appliquer le style.

Remarques :

  • Un aperçu s'affiche dans le volet droit.
    Vous pouvez choisir de prévisualiser la question, la réponse correcte ou la réponse erronée.

  • Vous pouvez cliquer sur Restaurer à tout moment pour réinitialiser.

 

Pour modifier le style à l'aide de Texte - Vue

  1. Vous pouvez modifier l'apparence des catégories suivantes :

    • Question

    • Exemple de question

    • Réponse

    • Exemple de réponse

    • Couleur de fond de la bonne réponse

    • Couleur de fond de la mauvaise réponse

Attention :
Lorsque vous modifiez les valeurs dans l'onglet Texte - Vue, NE supprimez PAS le point virgule (;) à la fin de la valeur.

 

  1. Recherchez la section appropriée dans l'onglet Texte - Vue et modifiez la valeur comme il vous convient.

Question

    • Recherchez la section ".question".

Exemple de question

    • Recherchez la section ".exempleQuestion".

Réponse

    • Recherchez la section ".réponse".

Exemple de réponse

    • Recherchez la section ".exempleRéponse".

 

    • Modifiez la valeur de "couleur" comme il vous convient.

    • Modifiez la valeur de "famille de police" comme il vous convient.
      Utilisez les noms tels qu'ils doivent apparaître dans Windows.

    • Modifiez la valeur de "taille des caractères" comme il vous convient.

    • Modifiez la valeur de "style de police" comme il vous convient.

    • Modifiez la valeur de "épaisseur de police" comme il vous convient.

 

Couleur de fond de la bonne réponse

    • Recherchez la section ".bonneRéponse".

Couleur de fond de la mauvaise réponse

    • Recherchez la section ".mauvaiseRéponse".

 

    • Modifiez la valeur de "couleur fond" comme il vous convient.

 

  1. Cliquez sur Fermer pour appliquer le style.

 

Remarques :

  • Un aperçu s'affiche dans le volet droit.
    Vous pouvez choisir de prévisualiser la question, la réponse correcte ou la réponse erronée.

  • Vous pouvez cliquer sur Restaurer à tout moment pour annuler toutes vos modifications.
