Apprentissage en mode choix multiple

Outre la saisie de texte classique, MemoryLifter propose de nombreux modes d'apprentissage, y compris le choix multiple, la reconnaissances des images, la compréhension auditive, l'auto-évaluation et le mode aléatoire.

 

Pour apprendre en mode choix multiple

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.
     

  2. Cliquez sur l'onglet Mode d'apprentissage.

  3. Dans le champ Modes d'apprentissage, sélectionnez Choix multiple.

 

Remarque :

Si vous sélectionnez plusieurs modes, MemoryLifter en choisit un aléatoirement dans la liste des modes sélectionnés chaque fois qu'une nouvelle carte s'affiche.

 

 

Pour répondre à une question en mode choix multiple

  1. Cliquez sur la(les) réponse(s) ou saisissez le(s) numéro(s) correspondant aux réponses qui selon vous sont exactes.

  2. Cliquez sur ou appuyez sur ENTREE

 

ou
Si vous ignorez la réponse, appuyez sur ENTREE.
La bonne réponse s'affiche et la carte est ajoutée à la pile.
Elle vous sera rapidement reproposée.

 

 

Lors d'une session d'apprentissage en mode choix multiple, plusieurs options de choix multiple vous sont proposées.

MemoryLifter présente généralement un nombre spécifié de réponses possibles extraites du pool de réponses, il s'agit de distracteurs aléatoires.

Vous pouvez choisir d'afficher autant de réponses possibles que vous le souhaitez en mode choix multiple.

 

Si les cartes que vous avez créées intègrent des distracteurs, il n'est pas nécessaire d'accepter les distracteurs aléatoires.  Définir de mauvaises réponses spécifiques augmente la difficulté du module d'apprentissage.

 

Vous pouvez également accepter plusieurs bonnes réponses et définir le nombre maximal de bonnes réponses que vous souhaitez afficher en mode de choix multiple.

 

Remarque :

En mode choix multiple, MemoryLifter n'indique qu'une réponse est correcte que si toutes les options sélectionnées sont correctes.

 

 

Pour définir des options de choix multiple

 

 

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.
     

  2. Cliquez sur l'onglet Mode d'apprentissage.

  3. Dans le champ Modes d'apprentissage, sélectionnez Choix multiple.

  4. Dans le champ Options de choix multiple, sélectionnez Autoriser les distracteurs aléatoires pour permettre à MemoryLifter d'ajouter d'autres distracteurs aux distracteurs définis pour la carte, à partir du pool de réponses.
     

  5. Dans le champ Options de choix multiple, sélectionnez Autoriser plusieurs réponses correctes pour permettre à MemoryLifter d'avoir plusieurs réponses correctes à une question.
     

  6. Dans le champ Nombre de choix, saisissez ou sélectionnez le nombre de réponses possibles que vous souhaitez afficher en mode choix multiple.
     

  7. Dans le champ Nombre max. de réponses correctes, saisissez ou sélectionnez le nombre maximum de réponses correctes possibles que vous souhaitez afficher en mode choix multiple.
    Remarque     : Le nombre réel d'éléments corrects est défini dans la carte. Il s'agit d'un nombre maximum de cartes affichées au cours de cette session.

  8. Cliquez sur OK pour commencer l'apprentissage en mode choix multiple.
