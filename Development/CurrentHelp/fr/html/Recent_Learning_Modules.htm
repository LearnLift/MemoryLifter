Modules d'apprentissage récents

Pour ouvrir un module d'apprentissage récemment ouvert

 

  1. Dans le menu Fichier, cliquez sur Modules d'apprentissage récents.
  2. Un sous-menu contenant les modules d'apprentissage dernièrement ouverts s'ouvre.
  3. Pour ouvrir un module dans ce sous-menu, cliquez dessus.
 

Remarque : Tout module ouvert sera alors fermé.  Pour les éléments du sous-menu, l'emplacement du module d'apprentissage figure entre parenthèses "( )" en regard du nom.
