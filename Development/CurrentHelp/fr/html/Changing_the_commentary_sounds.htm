Apprendre avec des commentaires audio

MemoryLifter utilise des commentaires audio pour encourager l'utilisateur dans son apprentissage.
Vous pouvez désactiver ou modifier les commentaires audio si vous le souhaitez.

 

Pour modifier les commentaires audio :

Observez les instructions sous Propriétés.

 

Pour désactiver les commentaires audio :

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

  2. Cliquez sur l'onglet Général.

  3. Décochez la case Ecouter le commentaire audio.
