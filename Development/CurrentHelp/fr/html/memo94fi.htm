Lancement d'une nouvelle session d'apprentissage

  1. Démarrez une nouvelle session d'apprentissage en ouvrant un module d'apprentissage sur la page de démarrage.

  2. Sélectionnez le ou les chapitres à apprendre au besoin.

  1. Cliquez sur OK pour commencer l'apprentissage.

 

Remarque :
Après avoir cliqué sur OK pour démarrer le processus d'apprentissage, MemoryLifter affiche les questions dans le volet gauche. Les réponses doivent être entrées dans le volet droit.

 

Si vous ignorez la réponse, appuyez sur ENTREE. MemoryLifter affiche la bonne réponse.

Cliquez sur la flèche pour passer à la question suivante.
