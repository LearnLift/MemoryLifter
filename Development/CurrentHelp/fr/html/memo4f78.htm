Imprimer

Impression de cartes

Vous pouvez imprimer sur papier les cartes d'un module d'apprentissage.

Dans la boîte de dialogue Sélectionner les paramètres d'impression, vous pouvez sélectionner les cartes à imprimer.

 

Vous pouvez choisir les options suivantes :

  • Toutes les cartes

  • Cartes inconnues

  • Sélectionner les cartes

  • Sélectionner les chapitres

  • Sélectionner les boîtes

 

Vous pouvez également fermer l'assistant et utiliser l'ancienne boîte de dialogue Imprimer.

 

 

Pour imprimer des cartes

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, sélectionnez les cartes à imprimer.

  3. Cliquez sur Suivant.

 

 

  1. Dans la boîte de dialogue Trier les cartes par :, saisissez ou sélectionnez les cartes à trier par :

    • Question

    • Réponse

    • Boîte

    • Chapitre

    • Aucun (c'est-à-dire l'ordre dans lequel vous avez ajouté les cartes au fichier de votre module d'apprentissage)

  2. Dans la zone de texte Définir le style d'impression :, indiquez si vous souhaitez imprimer les cartes

    • avec toutes les informations (question, réponse, phrases exemples et images) ; ou

    • avec la question et la réponse uniquement.

  3. Si vous souhaitez afficher un aperçu avant impression, cliquez sur Aperçu avant impression.

  4. Si vous souhaitez modifier la mise en page, cliquez sur Mise en page.

  5. Cliquez sur Imprimer pour imprimer les cartes.

 

Pour imprimer toutes les cartes

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, sélectionnez Toutes les cartes.

  3. Passez à l'étape 3 de Pour imprimer les cartes.

 

 

Pour n'imprimer que les cartes inconnues

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, sélectionnez Cartes inconnues.

  3. Passez à l'étape 3 de Pour imprimer les cartes.

 

 

Pour n'imprimer que les cartes spécifiées

 

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, sélectionnez Sélectionner les cartes.

  3. Cliquez sur Suivant.

 

 

  1. Dans la boîte de dialogue Sélectionner les cartes à imprimer, sélectionnez les cartes à imprimer.

  2. Passez à l'étape 3 de Pour imprimer les cartes.

 

 

 

Pour n'imprimer que les cartes des chapitres spécifiés

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, sélectionnez Sélectionner les chapitres.

  3. Cliquez sur Suivant.

 

 

  1. Dans la boîte de dialogue Sélectionner les chapitres à imprimer, sélectionnez les chapitres à imprimer à l'aide du sélecteur de chapitre.

  2. Passez à l'étape 3 de Pour imprimer les cartes.

 

 

Pour n'imprimer que les cartes des boîtes spécifiées

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, activez l'option Sélectionner les boîtes.

  3. Cliquez sur Suivant.

 

 

  1. Dans la boîte de dialogue Sélectionner les boîtes à imprimer, sélectionnez les boîtes à imprimer.

    • Sélectionnez Toutes les boîtes pour imprimer toutes les cartes de votre module d'apprentissage.

    • Sélectionnez Pool uniquement pour n'imprimer que les cartes auxquelles vous devez encore répondre.

    • Dans le champ Individuel, tapez le numéro des cartes à imprimer.

  2. Passez à l'étape 3 de Pour imprimer les cartes.

 

Pour imprimer les cartes sans passer par l'assistant d'impression

  1. Dans le menu Fichier, cliquez sur Imprimer les cartes depuis le module…, appuyez sur Ctrl+P ou cliquez sur dans la barre d'outils.

  2. Dans la boîte de dialogue Sélectionner les paramètres d'impression, cliquez sur Fermer l'assistant.

 

 

  1. Dans la zone de texte Sélectionner les chapitres, sélectionnez les chapitres à imprimer à l'aide du sélecteur de chapitre.
    Par défaut, tous les chapitres sont sélectionnés.

  2. Dans la zone de texte Sélectionner les boîtes, sélectionnez les boîtes à imprimer.

  3. Dans la zone de texte Options d'impression, indiquez ou sélectionnez dans le champ Trier les cartes par si vous souhaitez trier les cartes par :

    • Question

    • Réponse

    • Boîte

    • Chapitre

    • Aucun (c'est-à-dire l'ordre dans lequel vous avez ajouté les cartes au fichier de votre module d'apprentissage)

  4. Dans la zone de texte Options d'impression, indiquez ou sélectionnez dans le champ Utiliser le style d'impression si vous souhaitez :

    • Imprimer tout (pour imprimer les cartes et toutes les informations qu'elles contiennent : question, réponse, phrases exemples et images) ou

    • Imprimer sans media (pour imprimer les cartes avec la question et la réponse seulement)

  5. Cliquez sur Aperçu avant impression pour afficher un aperçu avant impression.

  6. Dans la fenêtre Aperçu avant impression, cliquez sur Imprimer pour lancer l'impression.

Remarque :

Vous pouvez basculer vers l'assistant d'impression à tout moment en cliquant sur Ouvrir l'assistant.
