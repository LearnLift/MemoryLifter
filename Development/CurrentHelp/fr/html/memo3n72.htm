Ouverture d'un module d'apprentissage existant…

Un module d'apprentissage est un fichier qui contient un nombre spécifique de cartes, organisées en chapitres.

 

Les modules d'apprentissage portent les extensions suivantes :
 

  • .mlm dans MemoryLifter 2.2 et les versions ultérieures

  • .odx dans les versions précédentes de MemoryLifter.

 

 

Pour ouvrir un module d'apprentissage existant (mlm ou odx)

  1. Dans le menu Fichier, choisissez Ouvrir…, appuyez sur Ctrl+O ou cliquez sur dans la barre d'outils.

  2. Sur la page de démarrage, ouvrez le module d'apprentissage de votre choix à partir de la zone Mes modules d'apprentissage
     

    1. en cliquant dessus et en appuyant sur Entrée

    2. en cliquant dessus puis sur OK

    3. en double-cliquant dessus

    4. en effectuant un clic droit et en cliquant sur Ouvrir dans le menu
       

 

Remarques :

  • Vous pouvez ajouter des modules d'apprentissage à partir de différents emplacements dans la zone Mes modules d'apprentissage par simple glisser-déposer sur la page de démarrage ou directement dans la zone Modules d'apprentissage.

  • MemoryLifter convertit automatiquement les modules d'apprentissage antérieurs (odx) au nouveau format mlm.  
    Par contre, vous ne pouvez pas utiliser les nouveaux fichiers mlm dans d'anciennes versions de MemoryLifter (versions 2.1.et antérieures).

  • Visitez notre site Internet à l'adresse www.memorylifter.com et téléchargez des modules d'apprentissage pré-créés, développés par des experts.

  • Les modules d'apprentissage les plus récemment utilisés sont répertoriés dans le menu Fichier, dans le sous-menu nommé Modules d'apprentissage récents.
    Il suffit d'en sélectionner un dans la liste pour l'ouvrir.

  • Les modules d'apprentissage sont automatiquement installés dans le dossier que vous avez sélectionné au premier démarrage de MemoryLifter. Il s'agit par défaut du sous-répertoire "Modules d'apprentissage" figurant dans le dossier "Mes documents".
    Vous pouvez modifier cette préférence à l'aide de l'option Modifier le dossier des modules d'apprentissage du menu Outils.
