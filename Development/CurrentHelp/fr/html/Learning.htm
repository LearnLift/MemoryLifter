Apprentissage en mode standard

Vous pouvez répondre à une question par une entrée standard ou en sélectionnant une/des réponse(s) en mode choix multiple.

Tout dépend du mode d'apprentissage que vous avez sélectionné.

 

Pour répondre à une question en mode standard
 

  1. Vérifiez que la carte mémoire de réponse est en mode de saisie (carte mémoire avec champ de texte au centre).
     

  2. Saisissez la réponse dans le champ de texte.
    Appuyez sur ENTREE pour continuer.
    Si plusieurs réponses correctes (synonymes) sont possibles, la touche ENTREE vous permet de descendre d'une ligne pour saisir une autre réponse.
    Dans le cas contraire, (ou après le dernier synonyme), la touche ENTREE permet d'envoyer votre réponse.

  3. Une fois votre réponse envoyée, vous pouvez la revoir. Le bouton de flèche droite est actif et vous permet de passer à la carte suivante.

 

Remarques :

  • Si vous ignorez la réponse, appuyez sur ENTREE pour que MemoryLifter l'affiche.
  • La mappe de caractères permet d'utiliser des caractères spéciaux.

 

 

Pour modifier une carte

Si vous souhaitez modifier une carte pendant une session d'apprentissage, allez dans le menu Edition, cliquez sur Maintenir les cartes..., appuyez sur F10 ou cliquez sur dans la barre d'outils.
La boîte de dialogue Maintenir les cartes s'affiche. Vous pouvez y modifier les éléments de chaque carte.
Cliquez sur Enregistrer les modifications pour appliquer les changements, puis sur Fermer pour poursuivre l'apprentissage.

 

Pour terminer une session d'apprentissage
 

  1. Cliquez sur le bouton X dans le coin supérieur droit de la fenêtre.

  2. Si vous souhaitez terminer votre session d'apprentissage, sélectionnez Oui dans la nouvelle boîte de dialogue, Annuler vous permet de continuer l'apprentissage.

  3. Les statistiques de cette session s'affiche, cliquez sur OK.
