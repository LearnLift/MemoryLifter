Fermer

Pour fermer le module d'apprentissage en cours

 

  1. Dans le menu Fichier, cliquez sur Fermer.
  2. Vous quittez ainsi le module d'apprentissage en cours. Une fenêtre grise vide s'affiche alors à la place.
  3. Vous pouvez alors ouvrir un autre module à l'aide de la Page de démarrage ou du menu Modules d'apprentissage récents.
