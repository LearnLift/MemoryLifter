Style Editor

The appearance of questions and answers in the MemoryLifter learning window is controlled by what we called "style".  The style contains information like font type, font color, font size, background color, etc... It is basically an XSL stylesheet for those who are familiar with the XSL technology.  The question and answer each have its own style.

 

MemoryLifter provide several pre-designed styles: Active, Covered, Default, Kids, and Simple.  You can modify the pre-designed style using the Style Editor.

 

To change the appearance of the pre-designed style currently in use:

 

  1. From the Tools menu, select Style Editor…

  2. On the lower left window, choose the question side stylesheet you would like to change.  By default, the question side stylesheet is selected.

  3. you can change the appearance for the following categories:

Note: when changing the value, DO NOT delete the semi-colon (;) at the end of the value.

Question

    • Find the ".mlquestion" section.

    • Change the value of the "font-family" as desired. Use the exact font name as you would see in Windows.

    • Change the value of the "font-size" as desired.

    • Change the value of the "font-style" as desired.

    • Change the value of the "font-weight" as desired.

Question Example

    • Find the ".mlquestionexample" section.

    • Change the value of the "font-family" as desired. Use the exact font name as you would see in Windows.

    • Change the value of the "font-size" as desired.

    • Change the value of the "font-style" as desired.

    • Change the value of the "font-weight" as desired.

Answer

    • Find the ".mlanswer" section.

    • Change the value of the "font-family" as desired. Use the exact font name as you would see in Windows.

    • Change the value of the "font-size" as desired.

    • Change the value of the "font-style" as desired.

    • Change the value of the "font-weight" as desired.

Answer Example

    • Find the ".mlanswerexample" section.

    • Change the value of the "font-family" as desired. Use the exact font name as you would see in Windows.

    • Change the value of the "font-size" as desired.

    • Change the value of the "font-style" as desired.

    • Change the value of the "font-weight" as desired.

Correct Answer's background color

    • Find the ".mlcorrectanswer" section.

    • Change the value of the "background-color" as desired.

Wrong Answer's background color

    • Find the ".mlwronganswer" section.

    • Change the value of the "background-color" as desired.

  1. Now select the answer side stylesheet on the lower left window and repeat all the changes applied in step 3 above.

Note: You also need to change the answer side stylesheet because MemoryLifter has a learning mode where the question and answer can be flipped.

  1. Click Close when done.

 

Hints:

  • Click ‘Preview’ you can see the results of your changes on the right side of the dialog box.

  • If you are not satisfied with your changes simply click ‘Restore’ and all your changes are undone.
