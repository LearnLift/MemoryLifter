What is New

MemoryLifter™ 2.0 – Unicode enabled

 

Welcome to MemoryLifter™ version 2.0.  We are very excited about this release as we now can support users that need the Unicode ability to learn foreign languages like Chinese and Japanese.  Beside Unicode, we have added some learning features our user community have asked for.

 

Benefits:

 

MemoryLifter™ 2.0 has new benefits including:

 

  1. Support for Unicode to enable learning of eastern Asian languages (e.g., Japanese, Arabic, Chinese, etc...) which uses non-western characters.

  2. Dictionary format is in XML

  3. Improved control of the appearance of questions and answers via XSL stylesheet.

  4. Improved recorder quality to record at 1411 kbps Bit Rate, 44 kHz Sample rate, 16 bit Audio Sample Size, and Stereo

  5. Improved media support (i.e., can now have image for both the question and answer side).

  6. Image recognition learning mode no longer requires the "(image)" keyword in the question text.

  7. You can now edit the current card you are learning by selecting Cards, Maintain or pressing F10.

  8. You can now sort the cards via the appropriate columns (i.e., sort by question, by answer, by box, by chapter, etc...)

  9. New "slideshow" learning mode: Answer is shown with question to allow you to study.

  10. Change the appearance of the questions and answers using pre-designed style or using the built-in style editor.

  11. Enhanced box learning algorithm to allow time-based learning (in the future).

  12. Fixes of various bugs:

    • Box sizes can now be adjusted below the number of cards currently in the box.

    • Question and answer window have scroll bars when window is resized smaller.

    • Character map now works for the answer window.

    • Images are displayed at its normal size.

    • Other fixes...

 

Notes:

 

  1. MemoryLifter will automatically convert the old 1.x dictionary to the new XML format when you open them.  However, the new 2.0 dictionary is not backward compatible.

  2. You can run the MemoryLifter 2.0  and MemoryLifter 1.7 at the same time so you can take your time and move your 1.7 dictionary over to the new version 2.0.

 

If you have any questions, please send an Email to support@memorylifter.com or fill out a feedback form via Help, send feedback from within MemoryLifter.

 

Thanks and Have fun learning!
