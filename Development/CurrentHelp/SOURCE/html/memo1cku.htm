Specify chapters to learn

 

You can add or remove chapters from the list you are currently learning.

 

  1. On the Learn menu, select Specify chapters to learn.

  2. When the Chapter Selection dialog box opens, select the chapters you would like to learn using the chapter selector.

  3. Click ‘OK’ to start the learning session.

 

Note: Chapters which are removed will loose the tracking of its learning progress. Only remove the chapter(s) if you don't want to learn the chapter anymore.
