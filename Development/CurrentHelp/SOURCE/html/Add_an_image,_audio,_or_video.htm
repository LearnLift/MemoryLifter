Adding an image, audio, or video

MemoryLifter offers you the possibility to add an image, sound and/or video to your flashcards to increase your learning success.

 

  1. Press the Add Media button.

  2. Open the directory where the media files are located.

  3. Select the desired file and click OK.

  4. Check one of the following options for the media:
     

    • Show on question side - Associate the media for the question side.  
      Note      : In the future, several media files can be attached and shown for the question.

    • Set as question standard audio - When checked, MemoryLifter will play this media automatically on the question side.  This option only applies to audio files.

    • Set as question example audio - Associate the audio as the example part of the question.  This option only applies to audio files.

    • Show on answer side - Associate the media for the answer side.  
      Note      : In the future, several media files can be attached and shown for the answer.

    • Set as answer standard audio - When checked, MemoryLifter will play this media automatically on the answer side.  This option only applies to audio files.

    • Set as answer example audio - Associate the audio as the example part of the answer.  This option only applies to audio files.

  5. To view an image or playback an audio or video, click View Media.

 

Supported media file types are:

  • Images - BMP, .GIF, JPG, ICO, EMF, WMF

  • Audio- mp3, wav, mid

  • Video - avi, wmv
     

Notes

  • An image can be associated with the question or the answer, or both the question and answer.
