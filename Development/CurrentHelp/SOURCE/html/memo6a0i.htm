Field selector

In various  MemoryLifter dialog boxes, you can choose the various parts ("fields") of a card in a certain order for processing.

 

For instance, in the export dialog, you can specify which fields you want to export, as well as the order the program should put them in.

 

The list on the left contains the available fields. This is either just the two words of a card and their sample sentences, or the attached media as well.

 

The list on the right contains the selected fields.

 

Hint: By default, the question and answer fields are added.

 

To add fields

  1. Select the field you want to add from the left side by clicking it.

  2. Add it either by double clicking or clicking the arrow.

  3. Repeat steps 1 and 2 until all needed fields are added.

 

To select more than one field, hold either Ctrl while picking the fields, or select the first field, then hold Shift while clicking the last field you want to add.

 

To add all fields

  1. Click the select all arrow.

 

To remove fields

  1. Select the field(s) you want to remove from the right list.

  2. Click the arrow.

 

To remove all fields

  1. Click the remove all arrow.

 

To rearrange the order of the fields

  1. Click the field to be moved in the right list.

  2. Drag field to new position in the list.

  3. Release mouse button.
