Starting a new learning session

  1. Start a new learning session by opening a dictionary in the MemoryLifter main window.

  2. Select the chapter(s) to learn if needed.

  1. Click the OK button to start learning.

 

Note: After pressing OK to start the learning process, MemoryLifter will present you with the questions in the left box and you enter the answer in the right box.

 

If you do not know the answer, press the Enter key and the program will show you the correct answer. Press OK to continue to the next vocabulary.
