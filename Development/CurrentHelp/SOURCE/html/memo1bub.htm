Show / Edit Box sizes

 

The Leitner box system is an integral part of MemoryLifter's learning algorithm. As you learn, MemoryLifter promotes and demotes cards into boxes. At any time, you can see the distribution of the cards over the boxes currently in your learning session. You can also adjust the box sizes to change the maximum number of cards to hold in each box.

 

To view the box sizes

From the Learn menu, select Show/edit box sizes… or click on the toolbar.

 

Changing box sizes

  • Click on the up/down arrows on the spin edit box to increase or decrease the maximum number of cards for each box.

OR

  • Enter a maximum size in the spin edit box directly - must be equal, above, or below the number of cards currently in the box.

 

To reset the box sizes, click Default.
