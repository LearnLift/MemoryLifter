Lernen mit Korrektur während der Eingabe

MemoryLifter bietet die Möglichkeit, dass das System während der Eingabe der Antwort eine automatische Überprüfung auf Schreibfehler vornimmt.

Wenn diese Funktion eingeschaltet ist, bleibt der Cursor so lange auf der jeweiligen Position stehen, bis Sie das korrekte Zeichen eingegeben haben und rückt erst dann auf die nächste Position vor.

 

Aktivieren der Korrektur während der Eingabe

  1. Klicken Sie im Menü Lernen auf den Eintrag Lernoptionen oder in der Symbolleiste auf .

  2. Klicken Sie auf das Register Eingabe.

  3. Markieren Sie Korrektur während der Eingabe.

 

Wenn die Funktion "Korrektur während der Eingabe" markiert ist, können Sie außerdem anhand verschiedener Optionen auswählen, wie das Höher- und Tieferstufen der Lernkarten in Abhängigkeit der bei der Eingabe gemachten Schreibfehler vorgenommen werden soll.

 

Eingabe der Antwort mit Korrektur während der Eingabe

  1. Stellen Sie sicher, dass der Antwortbereich im Eingabemodus ist (Antwortbereich enthält nur ein Textfeld in der Mitte).

  2. Geben Sie die Antwort ein.
    Ist ein eingegebenes Zeichen nicht korrekt, wird dieses rot.
    Der Cursor wird erst dann weitergestellt, wenn Sie das richtige Zeichen eingegeben haben.

  3. Sobald die Antwort komplett ist, wird im Antwortbereich die gesamte in der Lernkarte definierte Antwort angezeigt.
    Gibt es für ein Wort mehrere Bedeutungen, wird der Cursor nach jedem gewussten Wort in die nächste Zeile gesetzt, bevor schließlich die gesamte Antwort angezeigt wird.
