Kartenstileditor

Im Kartenstileditor können Sie den Stil der Elemente im Fragebereich und im Antwortbereich des Lernfensters ändern, d.h. einstellen, wie diese Bereiche des Lernfensters aussehen sollen.

 

Die Stile der folgenden Elemente und Zustände können eingestellt werden:

  • Frage

  • Beispiel für Frage

  • Antwort

  • Beispiel für Antwort

  • Richtige Antwort

  • Falsche Antwort

 

Der Kartenstileditor kann aus den Dialogfenstern Karten verwalten und Kapitel bearbeiten heraus geöffnet werden.

Die im Kartenstileditor definierten Stile können für eine einzelne im Dialogfenster Karten verwalten ausgewählte Karte eingestellt werden oder für ein im Dialogfenster Kapitel bearbeiten ausgewähltes Kapitel.

 

Ändern des Stils

  1. Öffnen Sie den Kartenstileditor:

    • Klicken Sie im Dialogfenster Karten verwalten auf Kartenstil hinzufügen/bearbeiten.

    • Oder klicken Sie im Dialogfenster Kapitel bearbeiten auf Stil hinzufügen/bearbeiten.

  2. Öffnen Sie im Dialogfenster Kartenstileditor das Registerblatt des zu ändernden Elementes oder Zustands.

  3. Um die Vordergrundfarbe einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Farbe aus.

  4. Um die Hintergrundfarbe einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Farbe aus.

  5. Um die Schriftart einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Schrift aus.

  6. Um die Schriftauszeichnung einzustellen, entfernen Sie im Bereich Schriftschnitt die Markierung des Kontrollkästchens Keine und markieren stattdessen die gewünschten Kontrollkästchen für die Schriftauszeichnung (Fett, Durchgestrichen, Normal, Kursiv oder Unterstrichen).

  7. Um die Schriftgröße einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Schriftgröße aus.

  8. Um die Ausrichtung horizontal einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Ausrichtung aus.

  9. Um die Ausrichtung vertikal einzustellen, markieren Sie das entsprechende Kontrollkästchen und wählen die gewünschte Ausrichtung aus.

  10. Im Bereich Andere Elemente können Sie weitere CSS-Elemente hinzufügen, um weitergehende Gestaltungen des Stils vorzunehmen.

  11. Klicken Sie auf OK, um den Stil anzuwenden.

Hinweis:
Auf der rechten Seite des Dialogfensters wird eine Vorschau angezeigt.
Durch Klicken auf Stil löschen können Sie den Stil jederzeit auf Standard zurücksetzen.
