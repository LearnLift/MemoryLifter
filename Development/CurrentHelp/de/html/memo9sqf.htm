Erstellen von neuen Lernmodulen

Erstellen eines Lernmoduls

  1. Klicken Sie im Menü Datei auf den Eintrag Neu... oder in der Symbolleiste auf oder drücken Sie Strg+N.

  2. Wählen Sie im Dialogfenster Neues Lernmodul erzeugen zuerst den Speicherort für das neue Lernmodul.

  3. Klicken Sie auf Weiter.

  4. Geben Sie die Detailinformationen zu Ihrem Lernmodul ein:

    • Den Namen des Lernmoduls im Feld Name.

    • Die Kategorie des Lernmoduls im Feld Kategorie.

    • Den Namen des Lernmodul-Autors im Feld Autor.

    • Eine Kurzbeschreibung des Lernmoduls im Feld Beschreibung.

  5. Klicken Sie auf Weiter.

  6. Gehen Sie im Dialogfenster Einstellungen für Frage-/Antwortbereiche wie folgt vor:

    •  Geben Sie in die Felder Titel die Titel für Frage- und Antwortbereich ein.

    •  Wählen Sie in den Feldern Sprache die jeweilige Sprache für die Fragen und die Antworten aus oder geben diese ein.

Hinweis:

Die hier vorgenommenen Einstellungen können Sie später im Eigenschaften-Dialog ändern (zu öffnen aus dem Menü Datei).
 

  1. Klicken Sie auf Beenden.

  2. Führen Sie im Dialogfenster Kapitel verwalten eine der folgenden Aktionen aus:

 

    • Klicken Sie auf OK, wenn Sie den voreingestellten Kapitelnamen (Kapitel 1) verwenden möchten.

Oder

    • Wenn Sie den voreingestellten Kapitelnamen ändern möchten, ändern Sie die Angaben in den Feldern Titel und Beschreibung nach Bedarf ab, klicken Sie auf Übernehmen und anschließend auf OK.
      Oder
    • Lesen Sie im Thema Kapitel verwalten nach, wenn Sie mehrere Kapitel erstellen und diese in einer bestimmten Reihenfolge anordnen möchten.
