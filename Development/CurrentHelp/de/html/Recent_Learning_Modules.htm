Vorige Lernmodule

Öffnen eines zuletzt geöffneten Lernmoduls

 

  1. Klicken Sie im Menü Datei auf den Eintrag Vorige Lernmodule.
  2. Ein Untermenü wird geöffnet, das die zuletzt geöffneten Lernmodule enthält.
  3. Klicken Sie im Untermenü auf einen Modunamen, um dieses Modul zu öffnen.
 

Hinweis: Ist gerade ein Lernmodul geöffnet, wird dieses geschlossen, wenn Sie ein Modul aus der Liste der vorigen Module wählen. Dem Eintrag im Untermenü können Sie auch den Speicherort des Lernmoduls (in Klammern "()" angezeigt) entnehmen.
