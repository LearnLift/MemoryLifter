Schließen des aktuellen Lernmoduls

 

  1. Klicken Sie im Menü Datei auf den Eintrag Schließen.
  2. Dadurch wird das aktuelle Lernmodul geschlossen und Sie sehen ein leeres graues Fenster.
  3. Sie können nun über die Startseite oder über das Menü Vorige Lernmodule ein anderes Lernmodul öffnen.
