Lernen von Karten in einer bestimmten Box

Standardmäßig werden von MemoryLifter die Lernkarten aus allen Boxen abgefragt.

Sie können jedoch auch einstellen, dass nur die Karten in einer bestimmten Box abgefragt werden.

 

Lernen der Karten in einer bestimmten Box

 

  • Klicken Sie im Menü Lernen auf den Eintrag Box auswählen und wählen Sie anschließend, aus welcher Box MemoryLifter die Karten abfragen soll.
    Hinweis:     Die ausgewählte Box ist mit einem Häkchen markiert.
