Stileditor

Ändern der Schriftart, Schriftgröße, Farbe und anderer Eigenschaften für Frage und Antwort

Die Art und Weise, wie die Fragen und Antworten im Lernfenster von MemoryLifter dargestellt werden, wird durch so genannte Stile gesteuert.  

Diese Stile enthalten alle Informationen zu Schriftart, Schriftfarbe, Schriftgröße, Hintergrundfarbe, usw.

Für diejenigen, die mit der CSS-Technologie vertraut sind, sei gesagt, dass es sich hierbei grundsätzlich um CSS-Stylesheets handelt.  

Frage und Antwort verwenden jeweils eigene Stile.

 

MemoryLifter stellt verschiedene vordefinierte Stile zur Verfügung: Default, BumbleBee, iLifter, Kids und Tropical Beach.  

Mit dem Stileditor können diese vordefinierten Stile geändert werden.

 

Mit dem Kartenstileditor lassen sich außerdem eigene Stile für einzelne Lernkarten oder Kapitel definieren.

 

 

Ändern des aktuell verwendeten Stils

 

  1. Klicken Sie im Menü Werkzeuge auf den Eintrag Stileditor....

  2. Wählen Sie aus, welche Einstellungen Sie ändern möchten: Globale Einstellungen (alle Eigenschaften des Stils) oder Stileinstellungen (nur die wichtigsten Eigenschaften des Stils).

Hinweis:

Die nachfolgenden Beschreibungen beziehen sich auf das Ändern von Stileinstellungen.

Für das Ändern der globalen Einstellungen stehen noch weitere Elemente zur Verfügung, die hier nicht beschrieben sind. Die generelle Vorgehensweise ist jedoch identisch.

 

  1. Wählen Sie aus, welche Darstellungsart Sie im Stileditor haben möchten: Tabellenansicht oder Textansicht.

 

Ändern von Stilen in der Tabellenansicht

  1. Für folgende Kategorien kann das Erscheinungsbild geändert werden:

    • Frage (question)

    • Beispiel für Frage (questionExample)

    • Antwort  (answer)

    • Beispiel für Antwort (answerExample)

    • Richtige Antwort (answerCorrect)

    • Falsche Antwort (answerWrong)

  2. Wählen Sie im Registerblatt Tabellenansicht die Zeile des Elementes (Kategorie), dessen Stil Sie ändern möchten.

  3. Stellen Sie im Feld Textfarbe die gewünschte Farbe ein.

  4. Stellen Sie im Feld Hintergrundfarbe die gewünschte Farbe für den Hintergrund ein.

  5. Klicken Sie auf Schrift..., um Schriftart, Schriftstil und Schriftgröße einzustellen.

  6. Stellen Sie im Feld Ausrichtung horizontal die horizontale Ausrichtung der Schrift ein (keine, links, zentriert, rechts).

  7. Stellen Sie im Feld Ausrichtung vertikal die vertikale Ausrichtung ein (keine, oben, mittig, unten).

  8. Wiederholen Sie dies für jedes zu ändernde Element.

  9. Klicken Sie auf Schließen, um den Stil anzuwenden.

Hinweise:

  • Auf der rechten Seite des Dialogfensters wird eine Vorschau angezeigt.
    Sie können auswählen, für welche Kategorie Sie die Vorschau sehen möchten: Frage, Richtige Antwort oder Falsche Antwort.

  • Durch Klicken auf Wiederherstellen können Sie die Einstellungen jederzeit wieder zurücksetzen.

 

Ändern von Stilen in der Textansicht

  1. Für folgende Kategorien kann das Erscheinungsbild geändert werden:

    • Frage

    • Beispiel für Frage

    • Antwort

    • Beispiel für Antwort

    • Hintergrundfarbe für korrekte Antwort

    • Hintergrundfarbe für falsche Antwort

Achtung:
Achten Sie darauf, dass Sie beim Ändern von Werten in der Textansicht NIEMALS den Strichpunkt hinter dem Wert löschen.

 

  1. Suchen Sie im Registerblatt Textansicht den gewünschten Abschnitt und ändern Sie die Werte nach Wunsch ab.

Frage

    • Suchen Sie den Abschnitt ".question".

Beispiel für Frage

    • Suchen Sie den Abschnitt ".questionExample".

Antwort

    • Suchen Sie den Abschnitt ".answer".

Beispiel für Antwort

    • Suchen Sie den Abschnitt ".answerExample".

 

    • Ändern Sie den Wert für "color" (Farbe) nach Wunsch.

    • Ändern Sie den Wert für "font-family" (Schriftart) nach Wunsch.
      Verwenden Sie die exakte Bezeichnung für die Schriftart, so wie sie in Windows verwendet wird.

    • Ändern Sie den Wert für "font-size" (Schriftgröße) nach Wunsch.

    • Ändern Sie den Wert für "font-style" (Schriftstil) nach Wunsch.

    • Ändern Sie den Wert für "font-weight" (Schriftauszeichnung) nach Wunsch.

 

Hintergrundfarbe für korrekte Antwort

    • Suchen Sie den Abschnitt ".answerCorrect".

Hintergrundfarbe für falsche Antwort

    • Suchen Sie den Abschnitt ".answerWrong".

 

    • Ändern Sie den Wert für "background-color" (Hintergrundfarbe) nach Wunsch.

 

  1. Klicken Sie auf Schließen, um den Stil anzuwenden.

 

Hinweise:

  • Auf der rechten Seite des Dialogfensters wird eine Vorschau angezeigt.
    Sie können auswählen, für welche Kategorie Sie die Vorschau sehen möchten: Frage, Richtige Antwort oder Falsche Antwort.

  • Durch Klicken auf Wiederherstellen können Sie alle gemachten Änderungen jederzeit wieder rückgängig machen.
