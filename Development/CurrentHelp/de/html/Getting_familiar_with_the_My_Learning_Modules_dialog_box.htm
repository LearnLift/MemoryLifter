Die Startseite von MemoryLifter

 

 

Jedes Mal, wenn Sie MemoryLifter starten, wird die Startseite (Meine Lernmodule) angezeigt.

 

Hinweis:

Soll die Startseite nicht bei jedem Start von MemoryLifter angezeigt werden, aktivieren Sie das Kontrollkästchen Startseite nicht anzeigen unten links im Dialogfenster.

 

Sie können die Startseite jederzeit wieder öffnen, indem Sie entweder den Menüpunkt Öffnen im Menü Datei wählen, die Tastenkombination Strg+O drücken oder in der Symbolleiste auf klicken.

 

 

Die Startseite besteht aus folgenden Bereichen:

 

  1. Meine Lernmodule

  2. Vorschau

  3. Statistiken

  4. Lernmodule

  5. Vorige Lernmodule

  6. Neuigkeiten

 

Zusätzlich stehen Schaltflächen zur Verfügung, mit denen verschiedene Ansichten gewählt werden oder die Lernmodule neu geordnet werden können.

Mit Hilfe des Suchfeldes wird eine Live-Suche durchgeführt, d.h. das Ergebnis der Suche wird sofort nach Eingabe jedes einzelnen Zeichens angezeigt. Der Inhalt des Suchfeldes kann durch Klicken auf gelöscht werden.

 

 

Meine Lernmodule

Im Bereich Meine Lernmodule finden Sie eine Liste aller Lernmodule, die in Ihrem Standard-Lernmodulordner gespeichert sind.

Für jedes Lernmodul wird neben dem Modul-Namen auch ein Symbol angezeigt:

 

  • steht für ein normales Lernmodul

  • steht für ein gesperrtes Lernmodul. Diese Module können nur mit MemoryLifter 2.3 geöffnet werden.

 

Lernmodule können von anderen Orten per Drag&Drop in den Bereich Meine Lernmodule eingefügt werden.

Wenn Sie auf ein Lernmodul rechtsklicken, erscheint ein Kontextmenü. Sie können hier Lernmodule öffnen, löschen und umbenennen oder die Eigenschaften des Moduls ändern.

 

 

Vorschau

Im Bereich Vorschau wird eine Lernkarte des aktuell ausgewählten Lernmoduls angezeigt.

 

Statistiken

Im Bereich Statistiken werden Informationen zum aktuell ausgewählten Lernmodul angezeigt: Die Dauer der letzten Lernsitzung, die Anzahl abgefragter Karten, die Anzahl richtig beantworteter und falsch beantworteter Karten, der Anteil gewusster Karten im Lernmodul, die durchnittliche Zeit, die zur Beantwortung einer Karte benötigt wurde und die Anzahl der pro Minute gelernten Karten.

 

Lernmodule

Im Bereich Lernmodule wird der Speicherort des aktuell ausgewählten Lernmoduls angezeigt.

 

Vorige Lernmodule

Im Bereich Vorige Lernmodule werden die zuletzt geöffneten Lernmodule angezeigt.

 

Neuigkeiten

Im Bereich Neuigkeiten werden Neuigkeiten zu MemoryLifter und zu den von LearnLift angebotenen Lernmodulen angezeigt.

 

Schaltflächen, Links

Mit der Schaltfläche können Sie die Bereiche Vorschau, Statistiken und Neuigkeiten ausblenden, sodass nur der Bereich Lernmodule angezeigt wird. Entsprechend können die ausgeblendeten Bereiche durch erneutes Anklicken der Schaltfläche wieder eingeblendet werden.

 

Über die Schaltfläche können Sie die Anzeige Ihrer Lernmodule nach Speicherort, Kategorie oder Autor ordnen.

 

Mit der Schaltfläche können die verschiedenen Ansichten "Große Symbole", "Liste", "Details" oder "Kacheln" gewählt werden.

 

 

Durch Klicken auf Neu können Sie ein neues Lernmodul erstellen.

 

Mit einem Klick auf Öffnen von... können Sie nach einem Lernmodul suchen, das an einem anderen Speicherort als dem Standardordner gespeichert ist.
