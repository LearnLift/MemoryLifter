Karten aus Datei importieren

Importieren von Karten aus einer Datei

MemoryLifter erlaubt den Import von Daten aus Text- oder csv-Dateien mit den folgenden Trennzeichen.

 

  • Trennung durch Tabulator: Die Wortpaare sind durch Tabulator getrennt, die Lernkarten durch eine Absatzschaltung.

  • Trennung durch Strichpunkt: Die Wortpaare sind durch Strichpunkt getrennt, die Lernkarten durch eine Absatzschaltung.

  • Trennung durch Komma: Die Wortpaare sind durch Komma getrennt, die Lernkarten durch eine Absatzschaltung.

 

Hinweise:

  • Derzeit werden nur textbasierte Dateiformate unterstützt.
    Falls Sie Daten aus anderen Dateiformaten (z.B. Textverarbeitungsprogrammen) importieren möchten, sollten Sie versuchen, diese Daten in einem Standard-Textformat zu speichern.

  • Bei kommagetrennten Dateien kann der Import fehlschlagen, wenn innerhalb von Anführungszeichen das Zeichen " verwendet wird, z.B. "Er sagte, "Ich möchte das haben".".

  • Falls in einer aus Excel exportierten Datei keine Sonderzeichen angezeigt werden, sollten Sie versuchen, diese Datei statt im Format "CSV (Trennzeichen-getrennt) (*.csv)" als "Unicode-Text (*.txt)" zu speichern.

  • Wenn nicht Unicode eingestellt ist, exportiert Excel oftmals Sonderzeichen (Zeichen aus anderen Sprachen als der Systemsprache) nicht korrekt.

  • Wählen Sie den richtigen Zeichensatz für Ihre Daten.

 

Importieren einer Datei

  1. Wählen Sie im Menü Bearbeiten den Eintrag Lernkarten importieren => Aus Datei.

  2. Klicken Sie im Importieren-Dialog auf Datei laden....

  3. Wählen Sie die zu importierende Datei aus.

  4. Klicken Sie auf Öffnen.

Hinweis:

MemoryLifter versucht nun, das in der Datei verwendete Trennzeichen zu ermitteln, den Inhalt der Datei zu laden und die ersten 19 Lernkarten als "Vorschau" anzuzeigen.
Falls dies nicht möglich ist, müssen Sie das vorliegende Dateiformat im Feld Dateiformat manuell eingeben oder auswählen.

 

  1. Legen Sie im Auswahlbereich für die Felder fest, welche Felder importiert werden sollen.

  2. Ordnen Sie die zu importierenden Felder im Auswahlbereich für die Felder nach Wunsch an.

Hinweis:

Die Anordnung muss der Reihenfolge der Spalten in Ihrer Quelldatei entsprechen.

 

  1. Überprüfen Sie im Vorschau-Bereich, ob die Lernkarte korrekt angezeigt wird.

Hinweis:

Falls in der Vorschau die Reihenfolge der Spalten nicht korrekt ist, wählen Sie im Feld Dateiformat ein anderes Format.

Falls in der Vorschau die Zeichen nicht korrekt dargestellt werden, wählen Sie im Feld Zeichensatz (Codepage) einen anderen Zeichensatz.

 

  1. Wenn Ihre Quelldatei keinen Datei-Header benützt, entfernen Sie die Markierung im Kontrollkästchen Enthält Header.

  2. Wenn in der Quelldatei die Synonyme für Fragen oder Antworten durch Komma getrennt sind, markieren Sie das Kontrollkästchen Synonyme sind getrennt.

  3. Wählen Sie im Feld Standardkapitel das Kapitel für die importierten Lernkarten.

  4. Klicken Sie auf Importieren, um den Import zu starten.

  5. Klicken Sie auf Schließen, um mit dem Lernen zu beginnen.
