Starten einer neuen Lernsitzung

  1. Starten Sie eine neue Lernsitzung, indem Sie im Hauptfenster von MemoryLifter ein neues Lernmodul öffnen.

  2. Wählen Sie ein oder mehrere zu lernende Kapitel aus, falls erforderlich.

  1. Klicken Sie auf OK, um das Lernen zu beginnen.

 

Hinweis:
Nachdem Sie den Lernprozess durch Klicken auf OK gestartet haben, zeigt MemoryLifter im linken Fensterbereich die Fragen an. Die entsprechende Antwort geben Sie im rechten Fensterbereich ein.

 

Wenn Sie die Antwort nicht wissen, drücken Sie einfach die Eingabetaste. MemoryLifter zeigt dann die korrekte Antwort an.

Klicken Sie auf den Pfeil, um mit der nächsten Frage fortzufahren.
