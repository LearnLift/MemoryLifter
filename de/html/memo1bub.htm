Boxengröße anzeigen / bearbeiten

Das Lernen mit MemoryLifter basiert auf dem Karteikartensystem nach Leitner.

Bei diesem System werden während dem Lernen die einzelnen Lernkarten in eine andere Box (10 Boxen stehen zur Verfügung) mit einer höheren oder einer niedrigeren Nummer verschoben (d.h. höher- oder tiefergestuft), je nachdem, ob die Antwort richtig oder falsch war.

Die aktuelle Verteilung der Karten in den einzelnen Boxen kann während der Lernsitzung jederzeit angezeigt werden.

Sie können außerdem die maximale Anzahl an Karten ändern, die eine Box aufnehmen kann.

 

Anzeigen der Boxengrößen

Klicken Sie im Menü Lernen auf den Eintrag Boxengröße anzeigen / bearbeiten, drücken Sie Strg+B oder klicken Sie in der Symbolleiste auf .

 

Ändern der Boxengrößen

Wählen Sie in den entsprechenden Feldern für die Boxgengrößen die zulässige Anzahl von Karten für jede Box aus oder geben Sie diese ein.

 

Hinweis:

Um die Boxengrößen auf die Standardwerte zurückzusetzen, klicken Sie auf Standard.
