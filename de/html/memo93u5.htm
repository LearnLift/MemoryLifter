Das Karteikartensystem

MemoryLifter arbeitet mit virtuellen Lernkarten, basierend auf dem Karteikartensystem nach Leitner. Das Programm soll Ihnen so das Lernen erleichtern.

 

Das Karteikartensystem

Das Karteikartensystem (auch "Leitner-System" genannt) wurde ursprünglich in den 1970er Jahren von dem deutschen Lern-Psychologen Sebastian Leitner vorgestellt.

Zweck war es, eine strukturierte Lernmethode zu finden, die das Lernen und Einprägen von Informationen sowohl im Kurzzeit- als auch im Langzeitgedächtnis erleichtert.

 

Bei dem Karteikartensystem werden die so genannten Lernkarten, welche die zu erlernenden Informationen enthalten, in mehreren Karteikästen (Boxen) abgelegt (bei MemoryLifter werden 10 Boxen verwendet).

Je höher die Nummer der Box, desto größer ist die Box, d.h. desto mehr Karten kann sie aufnehmen.

Wird eine Frage richtig beantwortet, wird die jeweilige Lernkarte in eine höhere Box verschoben. Mit der Zeit wird die Lernkarte so in der Box mit der höchsten Nummer abgelegt. Der Lerninhalt der Karte sollte sich dann im Langzeitgedächtnis eingeprägt haben.

 

Die Verteilung der Lernkarten in den Boxen (d.h. wie viele Karten in den einzelnen Boxen enthalten sind) gibt dabei schnell Auskunft über den Wissensstand des Benutzers und wie gut die zu erlernenden Inhalte sich im Gedächtnis eingeprägt haben (Kurzzeit- oder Langzeitgedächtnis).

 

Funktionsweise des von MemoryLifter verwendeten Systems

MemoryLifter verwendet für die Abfrage der Lernkarten und die Verfolgung des Lernziels eine leicht modifizierte Version der ursprünglichen Funktionsweise des Karteikartensystems.

 

  1. Zuerst werden alle Lernkarten aus einem so genannten Pool gezogen, welcher alle vorhandenen Lernkarten enthält.

  2. MemoryLifter prüft, ob eine der Boxen voll ist.
    Ist dies der Fall, so wird eine Lernkarte aus dieser Box abgefragt.
    Ist keine der Boxen voll, verschiebt MemoryLifter eine neue Karte aus dem Pool in die erste Box und fragt diese ab.

  3. Beantworten Sie eine Karte richtig, wird diese von MemoryLifter in die nächste Box hochgestuft.
    Andernfalls wird sie in Box 1 heruntergestuft.

  4. Beim Verschieben von Karten in eine andere Box platziert MemoryLifter diese automatisch immer am Ende der Box.
    Das Abfragen der Karten einer Box erfolgt in der Reihenfolge, wie sie in der Box abgelegt wurden ("First-In-First-Out-Methode").

  5. MemoryLifter führt die Schritte 2 bis 4 solange durch, bis der Pool leer ist.

  6. Wenn der Pool leer ist, wird die älteste Karte aus dem Lernmodul abgefragt.
    Es erscheint eine Meldung, um Sie darauf hinzuweisen.
