Verwendung von Distraktoren im Multiple Choice-Modus

Distraktoren sind zur Auswahl angebotene falsche Antwortmöglichkeiten beim Multiple Choice-Verfahren.

 

Multiple Choice-Fragen sind ein Bewertungsverfahren, bei dem Sie aus einer Auswahl vorgegebener Antwortmöglichkeiten eine oder mehrere zutreffende Antworten auswählen. Eine Multiple Choice-Frage besteht dabei aus dem so genannten Stamm und einer Anzahl von Optionen, die zur Auswahl stehen.

 

  • Als Stamm wird der erste Teil bezeichnet, der das zu lösende Problem wiedergibt, z.B. in Form einer simplen Frage, einer unvollständigen Aussage, usw.

  • Die Optionen sind die Antwortmöglichkeiten, die dem Prüfling zur Auswahl angeboten werden. Dabei können eine oder mehrere Antworten richtig sein, die falschen Antworten sind die so genannten Distraktoren.

 

Der Lerneffekt kann hierbei durch sorgfältiges individuelles Erstellen von Distraktoren deutlich gesteigert werden gegenüber der Verwendung von zufällig ausgewählten Distraktoren aus dem Antworten-Pool.

 

Erstellen von Lernkarten mit Distraktoren

  1. Klicken Sie im Menü Bearbeiten entweder auf den Eintrag Karte hinzufügen oder auf Lernkarten verwalten.

  2. Geben Sie im Feld Antwort die korrekte Antwort ein und so viele falsche Antworten, wie Sie haben möchten. Gehen Sie hierzu gleich vor wie zum Definieren von Synonymen und trennen Sie die einzelnen Antworten durch Drücken der Eingabetaste oder indem Sie auf klicken und die nächste leere Zeile auswählen.

  3. Markieren Sie die gewünschten Einträge als Distraktoren, indem Sie mit der rechten Maustaste auf den jeweiligen Eintrag klicken und anschließend Distraktor auswählen.

  4. Klicken Sie auf Karte hinzufügen oder Änderungen speichern.
