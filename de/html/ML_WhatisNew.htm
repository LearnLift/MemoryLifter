Was ist neu

Willkommen zu MemoryLifter™

 

Leistungsmerkmale:

  • Neues Dateiformat (und neues Symbol für Lernmodul-Dateien):

    • Alle Daten werden in einer Datei gespeichert (es werden keine zusätzlichen Mediendateien oder Ordner mehr benötigt).

    • Die Lernfortschritte mehrerer lernenden Personen können in Dateien gespeichert werden.

    • Die Datensicherheit wurde erhöht.

  • Lernmodule können jetzt auch von einer zentralen Datenbank auf einem MemoryLifter-Server geöffnet und von dort gelernt werden (die kostenlose Serversoftware können Sie auf www.memorylifter.com downloaden).

  • Einfache Verwaltung der Lernmodule mit Hilfe der Startseite ("Meine Lernmodule"), die einen einfachen Zugriff auf Ihre Lernmodule (sowohl lokale wie auch auf Server-Lernmodule) ermöglicht.

 

Verbesserungen:

  • Neue Oberflächen-Designs.

  • Die Eingabemaske für das Multiple Choice-Verfahren hat ein neues Aussehen erhalten.

  • Die Menüstruktur wurde vereinfacht und die Symbole der einzelnen Menüeinträge stimmen nun mit den Symbolen in der Symbolleiste überein.

  • Die Symbolleiste wurde vereinfacht.

  • Neuer Algorithmus zur Berechnung des Lernfortschritts (basierend auf dem prozentualen Anteil richtig beantworteter Karten im Lernmodul).

  • Allgemeine Verbesserungen an der Benutzeroberfläche.

  • Neue Ergebnisdarstellung für Antworten.

    • Die Ergebnisse werden nicht mehr mit der Anzeige auf der Antwortseite gemischt und können außerdem geschlossen werden.

    • Richtige und falsche Synonyme werden hervorgehoben.

  • Die Geschwindigkeit wurde verbessert.

  • Viele kleinere Verbesserungen wurden durchgeführt.

 

Behobene Fehler:

  • Beim Ausdrucken von Lernkarten werden keine leeren Seiten mehr ausgegeben.

  • Einige Darstellungsprobleme mit asiatischen Schriften wurden behoben.

  • Einige Schriften, die einen Ausnahmefehler bei der Darstellung von Karten zur Folge hatten, wurden geändert.

  • Viele kleinere Fehler wurden behoben.

 

Hinweise:

  • Lernmodule aus älteren MemoryLifter-Versionen (.odx-Format) werden von MemoryLifter automatisch in das neue .mlm-Format konvertiert.  
    MemoryLifter-Dateien im .mlm-Format (Version 2.2 und höher) können nicht von älteren MemoryLifter-Versionen gelesen werden.

  • Die Programmversionen MemoryLifter 2.2/2.3 und MemoryLifter 2.1 können gleichzeitig ausgeführt werden, so dass Sie Ihre Lernmodule im .odx-Format in das neue .mlm-Format übernehmen können.
