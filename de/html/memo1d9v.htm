Lernoptionen

Anpassen der Lernsitzung

MemoryLifter bietet eine ganze Reihe von Optionen, mit denen Sie die Lernsitzung nach Bedarf anpassen können.

 

Anpassen der Lernoptionen im Dialogfenster "Lernoptionen"

Klicken Sie im Menü Lernen auf den Eintrag Lernoptionen oder in der Symbolleiste auf .

 

 

Registerblatt Lehrer

Hier können Sie einstellen, wie strikt die Bewertung Ihrer Antworten sein soll.

Je nachdem, welchen Lehrertyp Sie hier auswählen, werden die Einstellungen in den anderen Registerblättern (siehe nachfolgende Beschreibungen) automatisch entsprechend geändert.

Eine detailierte Auflistung der geänderten Einstellungen finden sie im Thema Lehrer.

Dies bedeutet, dass Sie die Einstellungen in den anderen Registerblättern nicht noch zusätzlich selbst ändern müssen.  

Probieren Sie einfach aus, welcher Lehrertyp Ihren Wünschen am nächsten kommt.

Sie können jederzeit wieder auf die Standardeinstellung zurückwechseln.

 

 

 

Registerblatt Lernmodus

 

Richtung

Hier können Sie festlegen, welche "Sprachrichtung" abgefragt werden soll. Es gibt drei Möglichkeiten:

  • Frage => Antwort. Im Falle eines Moduls Spanisch => Englisch würden spanische Fragen gestellt und englische Antworten erwartet.
  • Antwort => Frage. Im Falle eines Moduls Spanisch => Englisch würden englische Fragen gestellt und spanische Antworten erwartet.
  • Zufällig. In diesem Fall wid die Sprachrichtung für jede Karte zufällig ausgewählt. In einem Modul Spanisch => Englisch gibt es spanische Fragen und englische Antworten sowie englische Fragen und spanische Antworten.

 

 

Lernmodi

MemoryLifter bietet fünf Lernmodi an:

 

Symbol

Lernmodus

Beschreibung

Standard

MemoryLifter zeigt links die Frage an, im rechten Fensterbereich geben Sie die Antwort ein.
Alle für die Frage verfügbaren Informationen werden angezeigt bzw. ausgegeben (d.h. Bild, Audiodateien, Beispielsätze, usw.).

Multiple Choice

MemoryLifter bietet eine von Ihnen angegebene Anzahl möglicher Antworten an, aus denen Sie auswählen können.

Sätze

MemoryLifter fragt anstatt des Wortes die Beispielsätze ab.

Hörverständnis

MemoryLifter gibt als Frage nur die hinterlegte Audiodatei aus (z.B. das gesprochene Wort). Sie müssen die Antwort eingeben.

Bilderkennung

MemoryLifter zeigt als Frage nur das hinterlegte Bild an. Sie müssen die Antwort eingeben.

 

Wenn Sie mehr als einen Modus auswählen, wird für jede Karte der Modus nach dem Zufallsprinzip bestimmt.

 

Sie können den Lernmodus entweder über die Registerkarte Lernmodi ändern oder durch Auswahl eines Modus im Menü Lernmodus(modi) auswählen in der Symbolleiste.

 

Je nach gewähltem Lernmodus bzw. gewählten Lernmodi, werden die Symbole in unterschiedlichen Farben dargestellt.

 

Hinweis:

Wenn eine Lernkarte nicht in dem gewählten Lernmodus angezeigt werden kann (weil kein Bild, keine Audiodatei, kein Beispielsatz oder nicht genügend andere Lernkarten vorhanden sind), wird der Standard-Lernmodus verwendet.

 

Außerdem können die Multiple Choice-Optionen eingestellt werden.

 

 

 

Registerblatt Synonyme

 

Für den Fall, dass die Lernkarten Synonyme enthalten, können Sie hier einstellen, wie sich das Programm verhalten soll, wenn Sie nicht alle Synonyme korrekt eingeben.

 

  • Höherstufen wenn 100% gewusst - Die Lernkarte wird nur in die nächste Box höhergestuft, wenn Sie alle Synonyme wissen. Andernfalls wird die Karte in Box 1 heruntergestuft.

  • Höherstufen wenn 50% gewusst - Die Lernkarte wird in die nächste Box höhergestuft, wenn Sie die Hälfte der Synonyme wissen.

  • Höherstufen wenn erstes Wort gewusst - Die Lernkarte wird in die nächste Box höhergestuft, wenn Sie das erste Synonym wissen. Dies ist besonders nützlich, wenn die gebräuchlichste Bedeutung zuerst angegeben ist.

  • Höherstufen wenn ein Wort gewusst - Die Lernkarte wird in die nächste Box höhergestuft, wenn Sie eines der Synonyme wissen.

  • Bestätigen wenn nicht 100% - Der Benutzer wird benachrichtigt, wenn er nicht alle Antworten gewusst hat.

 

 

Registerblatt Eingabe

 

Korrektur während der Eingabe

Markieren Sie Korrektur während der Eingabe, wenn Sie möchten, dass während Ihrer Eingabe eine automatische Überprüfung auf Schreibfehler vorgenommen werden soll. Der Cursor bleibt dann so lange auf der jeweiligen Position stehen, bis Sie das korrekte Zeichen eingegeben haben und rückt erst danach auf die nächste Position vor.

 

Groß-/Kleinschreibung beachten

Wenn Ihre Eingabe auf Groß- und Kleinschreibung geprüft werden soll, markieren Sie Groß-/Kleinschreibung beachten.

 

Selbst bewerten

Markieren Sie Selbst bewerten, wenn Sie selbst entscheiden möchten, ob eine Lernkarte höher- oder tiefergestuft werden soll.

 

Diese Zeichen für die Korrektur ignorieren

In diesem Feld geben Sie die üblicherweise verwendeten Satzzeichen ein. Diese werden für die Korrektur nicht berücksichtigt.

Sie haben dadurch bei der Eingabe mehr Freiheiten, da beim Eingeben der Antwort keine korrekte Zeichensetzung erforderlich ist.

 

Tippfehler
Wenn im Registerblatt Eingabe die Funktion Korrektur während der Eingabe markiert ist, können Sie mit den Optionen im Bereich Tippfehler auswählen, wie das Höher- und Tieferstufen der Lernkarten in Abhängigkeit der bei der Eingabe gemachten Schreibfehler gehandhabt werden soll. Es stehen vier Optionen zur Auswahl:

 

  • Nur höherstufen wenn keine - Die Lernkarte wird nur in die nächste Box höhergestuft, wenn bei der Eingabe keine Schreibfehler gemacht wurden.

  • Höherstufen wenn > 50% - Die Lernkarte wird in die nächste Box höhergestuft, wenn mehr als die Hälfte des Wortes korrekt geschrieben wurde.

  • Immer höherstufen - Schreibfehler spielen keine Rolle.

  • Bestätigen - Wenn Schreibfehler gemacht wurden, muss der Benutzer eine entsprechende Abfrage bestätigen, damit die Karte höhergestuft wird.

 

 

 

 

Registerblatt Schlummern/Beenden

Sie können hier einstellen, dass das Programm nach einer bestimmten Zeitdauer oder einer bestimmten Anzahl von abgefragten oder korrekt beantworteten Lernkarten automatisch in den Schlummermodus wechseln oder beendet werden soll.

Durch Markieren einer oder mehrerer Optionen wird die Sitzung automatisch beendet und das Programm zu einem Symbol im Infobereich der Taskleiste minimiert.

Nach der angegebenen Zeit wird das Programm automatisch wieder geöffnet.

Alternativ kann das Programm nach Ende der Sitzung auch komplett beendet werden.

 

 

 

Registerblatt Allgemein

Es können folgende allgemeine Optionen ausgewählt werden:
 

  • Countdown-Timer:
    Beantwortung gegen die Uhr. Ihre Antwort wird nach einer vom Programm vorgegebenen Zeit automatisch abgegeben.

  • Statistiken anzeigen:
    Mit dieser Option können Sie einstellen, dass MemoryLifter nach jeder Sitzung die jeweiligen Statistiken anzeigt.

  • Bilder anzeigen:
    Falls vorhanden, werden Bild(er) angezeigt.

  • Mediendateien autom. wiedergeben:
    Falls vorhanden, wird beim Anzeigen der Frage oder Antwort die zugehörige Audiodatei abgespielt.

  • Nur eine Instanz des MemoryLifter erlauben:
    Wenn diese Option markiert ist, stellt MemoryLifter sicher, dass immer nur eine Lernsitzung gleichzeitig laufen kann.
    Durch Doppelklicken auf die Datei eines anderen Lernmoduls wird das aktuelle Lernmodul geschlossen und das neue geöffnet.
    Wenn diese Option nicht markiert ist, wird durch das Doppelklicken eine neue Lernsitzung gestartet.

  • Kommentarton abspielen:
    Falls vorhanden,     werden automatisch die Kommentartöne abgespielt.

  • Anzeige der richtigen Antworten überspringen:
    Nach der Eingabe einer korrekten Antwort macht MemoryLifter gleich mit der nächsten Frage fort, ohne dass die Antwort noch einmal explizit angezeigt wird.

  • Pool zufällig:
    Die abzufragenden Lernkarten werden zufällig aus dem Pool ausgewählt. Wenn diese Option nicht markiert ist, werden die Karten in der Reihenfolge abgefragt, wie sie im Lernmodul hinzugefügt wurden.

  • Tieferstufen bestätigen:
    Das Tieferstufen von Lernkarten muss von Ihnen jedesmal gesondert bestätigt werden.
