Installieren / Starten von MemoryLifter

  1. Führen Sie den Download der MemoryLifter-Installationsdatei aus.

  2. Beginnen Sie die Installation, indem Sie die .exe-Datei öffnen.

 

 

  1. Markieren Sie nach Wunsch die Kontrollkästchen MemoryLifter registrieren und Nach dem Starten Hilfe anzeigen oder entfernen Sie die Markierung.

  2. Klicken Sie auf Weiter.

 

 

  1. Wählen Sie einen Ordner für Ihre Lernmodule.

  2. Markieren Sie falls gewünscht das Kontrollkästchen Demo-Lernmodul in ausgewählten Pfad kopieren.

  3. Klicken Sie auf Weiter.

 

 

  1. Füllen Sie die Registrierung aus:

    • Geben Sie im Feld Name Ihren Namen ein.

    • Geben Sie im Feld E-Mail Ihre eMail-Adresse ein.

    • Geben Sie im Feld Kommentar einen Kommentar ein.

    • Stellen Sie mit dem Kontrollkästchen ein, ob sie per E-Mail über Neuigkeiten zu MemoryLifter und Learnlift informiert werden möchten.

  1. Klicken Sie auf Beenden und beginnen Sie mit dem Lernen!
