Lernkarten exportieren

Exportieren eines Lernmoduls in eine Textdatei

Mit dieser Funktion können Sie die Lernkarten eines Lernmoduls für eine anderweitige Verarbeitung exportieren (z.B. für die Bearbeitung in einer Tabellenkalkulation oder einem Textverarbeitungsprogramm).

 

  1. Klicken Sie im Menü Bearbeiten auf den Eintrag Lernkarten exportieren oder drücken Sie Strg+E.

  2. Gehen Sie im Exportieren-Dialog wie folgt vor:

    • Legen Sie die zu exportierenden Kapitel im Auswahlbereich für die Kapitel fest.

    • Legen Sie im Auswahlbereich für die Felder fest, welche Felder exportiert werden sollen.

    • Legen Sie das Dateiformat fest: Sollen die Daten durch Strichpunkt, Komma oder Tabulator getrennt werden?

    • Klicken Sie auf Exportieren.

  3. Wählen Sie den Zielordner aus und geben Sie im Feld Dateiname den Namen für Ihre Exportdatei ein.

  4. Klicken Sie auf Speichern.
