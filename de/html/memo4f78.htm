Drucken

Ausdrucken von Lernkarten

Sie können die im Lernmodul vorhandenen Lernkarten ausdrucken.

Im Dialog Druckeinstellungen auswählen können Sie auswählen, welche Lernkarten Sie ausdrucken möchten.

 

Folgende Optionen stehen zur Auswahl:

  • Alle Karten

  • Nicht gewusste Karten

  • Karten auswählen

  • Kapitel auswählen

  • Boxen auswählen

 

Falls gewünscht, können Sie den Druckassistenten auch ausschalten und stattdessen den bisherigen Drucken-Dialog verwenden.

 

 

Drucken von Lernkarten

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen, welche Lernkarten Sie ausdrucken möchten.

  3. Klicken Sie auf Weiter.

 

 

  1. Wählen Sie im Feld Karten sortieren nach aus (oder geben Sie ein), ob und wie die Lernkarten sortiert sein sollen:

    • Frage

    • Antwort

    • Box

    • Kapitel

    • Nicht sortieren (die Karten werden in der Reihenfolge gedruckt, wie sie im Lernmodul hinzugefügt wurden)

  2. Wählen Sie im Feld Druckformat aus (oder geben Sie ein), wie die Karten gedruckt werden sollen:

    • Mit allen Informationen (Frage, Antwort, Beispielsätze und Bilder).

    • Oder nur mit Frage und Antwort.

  3. Um eine Vorschau des Ausdrucks anzuzeigen, klicken Sie auf Druckvorschau.

  4. Um die Seiteneinrichtung zu ändern, klicken Sie auf Seite einrichten.

  5. Klicken Sie auf Drucken, um den Druck der Lernkarten zu starten.

 

Drucken aller Lernkarten

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen die Option Alle Karten.

  3. Fahren Sie fort mit Schritt 3 der oben beschriebenen Vorgehensweise zum Drucken von Lernkarten.

 

 

Drucken der nicht richtig beantworteten Lernkarten

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen die Option Nicht gewusste Karten.

  3. Fahren Sie fort mit Schritt 3 der oben beschriebenen Vorgehensweise zum Drucken von Lernkarten.

 

 

Drucken ausgewählter Lernkarten

 

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen die Option Karten auswählen.

  3. Klicken Sie auf Weiter.

 

 

  1. Wählen Sie im Dialog Karten zum Drucken auswählen die Lernkarten aus, die Sie ausdrucken möchten.

  2. Fahren Sie fort mit Schritt 3 der oben beschriebenen Vorgehensweise zum Drucken von Lernkarten.

 

 

 

Drucken der Lernkarten ausgewählter Kapitel

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen die Option Kapitel auswählen.

  3. Klicken Sie auf Weiter.

 

 

  1. Wählen Sie im Dialog Kapitel zum Drucken auswählen im Auswahlbereich für die Kapitel die Kapitel aus, die Sie drucken möchten.

  2. Fahren Sie fort mit Schritt 3 der oben beschriebenen Vorgehensweise zum Drucken von Lernkarten.

 

 

Drucken der Lernkarten ausgewählter Boxen

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Wählen Sie im Dialog Druckeinstellungen auswählen die Option Boxen auswählen.

  3. Klicken Sie auf Weiter.

 

 

  1. Wählen Sie im Dialog Boxen zum Drucken auswählen die Boxen aus, die Sie ausdrucken möchten.

    • Um alle Karten des Lernmoduls auszudrucken, wählen Sie Alle Boxen.

    • Um nur die Karten auszudrucken, die Ihnen noch nicht als Frage angezeigt wurden, wählen Sie Nur der Pool.

    • Im Feld der Option Individuell können Sie die Nummern der Boxen eingeben, die Sie ausdrucken möchten.

  2. Fahren Sie fort mit Schritt 3 der oben beschriebenen Vorgehensweise zum Drucken von Lernkarten.

 

Drucken von Lernkarten ohne Verwendung des Druckassistenten

  1. Klicken Sie im Menü Datei auf den Eintrag Lernkarten drucken, oder in der Symbolleiste auf oder drücken Sie Strg+P.

  2. Klicken Sie im Dialog Druckeinstellungen auswählen auf Assistent ausschalten.

 

 

  1. Wählen Sie im Bereich Kapitel auswählen im Auswahlbereich für die Kapitel aus, welche Kapitel Sie ausdrucken möchten.
    Standardmäßig sind alle Kapitel ausgewählt.

  2. Wählen Sie im Bereich Boxen auswählen die Boxen aus, die Sie ausdrucken möchten.

  3. Wählen Sie im Bereich Druckoptionen im Feld Karten sortieren nach, ob und wie die Lernkarten sortiert sein sollen:

    • Frage

    • Antwort

    • Box

    • Kapitel

    • Nicht sortieren (die Karten werden in der Reihenfolge gedruckt, wie sie im Lernmodul hinzugefügt wurden)

  4. Wählen Sie im Bereich Druckoptionen im Feld Druckformat verwenden aus (oder geben Sie ein), wie die Karten gedruckt werden sollen:

    • Alles drucken (Drucken der Karten mit allen Informationen, also mit Frage, Antwort, Beispielsätzen und Bildern) oder

    • Drucken ohne Medien (Drucken der Karten nur mit Frage und Antwort)

  5. Klicken Sie auf Drucken. Es erscheint die Druckvorschau.

  6. Klicken Sie in der Druckvorschau auf Drucken, um den Druckvorgang zu starten.

Hinweis:

Falls gewünscht, können Sie durch Klicken auf Assistent einschalten jederzeit zum Druckassistenten wechseln.
