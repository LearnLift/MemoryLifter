Öffnen eines bestehenden Lernmoduls

Ein Lernmodul ist eine Datei mit einer Anzahl von Lernkarten, die in Kapiteln organisiert sind.

 

Lernmodule haben folgende Dateinamenerweiterungen:

  • .mlm für MemoryLifter Version 2.2 und höher

  • .odx für ältere Versionen von MemoryLifter.

 

 

Öffnen eines bestehenden Lernmoduls (.mlm oder .odx)

  1. Klicken Sie im Menü Datei auf den Eintrag Öffnen… oder in der Symbolleiste auf . Alternativ dazu können Sie auch Strg+O drücken.

  2. Öffnen Sie auf der Startseite das gewünschte Lernmodul im Bereich Meine Lernmodule, indem Sie

    1. das Modul markieren und die Eingabetaste drücken,

    2. das Modul markieren und auf OK klicken,

    3. auf dem Modulnamen doppelklicken,

    4. auf dem Modulnamen rechtsklicken und im Kontextmenü den Eintrag Öffnen wählen.

 

Hinweise:

  • Lernmodule können von anderen Orten per Drag&Drop in den Bereich Meine Lernmodule oder direkt auf einem Ordner im Bereich Lernmodule auf der Startseite eingefügt werden.

  • Lernmodule aus älteren MemoryLifter-Versionen (.odx) werden von MemoryLifter automatisch in das neue .mlm-Format konvertiert.  
    MemoryLifter-Dateien im .mlm-Format (Version 2.2 und höher) können nicht von älteren MemoryLifter-Versionen gelesen werden.

  • Auf unserer Website www.memorylifter.com stehen vorgefertigte, von Experten entwickelte, Wörterbücher zum Download bereit.

  • Die zuletzt verwendeten Lernmodule sind im Untermenü Vorige Lernmodule im Menü Datei aufgelistet.
    Zum Öffnen eines solchen Lernmoduls klicken Sie einfach auf den entsprechenden Eintrag.

  • Lernmodule werden automatisch in den Ordner installiert, den Sie während dem ersten Starten von MemoryLifter eingestellt haben. Standardmäßig ist dies der Unterordner "Learning Modules" im Ordner "Eigene Dateien".
    Diese Voreinstellung kann über den Menüpunkt Lernmodulordner ändern im Menü Werkzeuge geändert werden.
