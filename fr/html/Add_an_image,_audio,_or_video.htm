Ajout de fichiers images, audio ou vidéo

MemoryLifter offre la possibilité d'ajouter à vos cartes un fichier image, audio et/ou vidéo pour améliorer la réussite de votre apprentissage.

 

 

Pour ajouter un fichier audio

  1. Dans la boîte de dialogue Maintenir les cartes, cliquez sur le bouton Audio.

  2. L'enregistreur de sons s'ouvre.

  3. Cliquez sur Parcourir... pour sélectionner un fichier audio.
    ou
    Cliquez sur Enregistrement pour enregistrer votre propre fichier audio.

 

Remarque :

Formats de fichiers audio pris en charge : mp3, wav, mid

 

 

Pour ajouter un fichier vidéo

  1. Dans la boîte de dialogue Maintenir les cartes, cliquez sur le bouton Vidéo .

  2. Cliquez sur Parcourir.

  3. Sélectionnez le fichier souhaité et cliquez sur OK.

 

Remarque :

Formats de fichiers vidéo pris en charge : avi, wmv

 

 

Pour ajouter un fichier image

Il existe deux façons d'ajouter des images.

  1. Dans le menu Edition, cliquez sur Maintenir les cartes, appuyez sur F10 ou cliquez sur dans la barre d'outils.

  2. Procédez de l'une des manières suivantes :

  • Dans la boîte de dialogue Maintenir les cartes, cliquez sur le bouton Image, puis sur Parcourir, sélectionnez le fichier souhaité et cliquez sur OK.
    ou

  • Faites glisser l'image dans le champ Image.
     

L'image apparaît dans le champ Image.

Lorsque vous faites glisser la première image vers le champ Image, la boîte de dialogue Redimensionner l'image ? apparaît.

 

 

Vous pouvez, dans votre module d'apprentissage, sélectionner le format de sauvegarde de l'image : petit, moyen, grand ou sans redimensionnement, au format d'origine.

Vous pouvez effectuer votre choix en cliquant sur l'une des icônes de la boîte de dialogue.

Pour sélectionner la même option de redimensionnement pour la question et la réponse, sélectionnez Utiliser pour les deux faces.

 

Remarques :

Lorsque vous modifiez des cartes, vous pouvez changer l'option de redimensionnement en cliquant sur .

Pour ajouter la même image à la question et à la réponse, sélectionnez utiliser l'image de question.

Formats de fichiers vidéo pris en charge : bmp, gif, jpg, png, ico, emf, wmf
