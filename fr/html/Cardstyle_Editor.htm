Editeur de style de carte

L'éditeur de style de carte permet de modifier le style, c'est-à-dire, la façon dont les éléments contenus dans les champs Question et Réponse de la fenêtre d'apprentissage apparaissent.

 

Vous pouvez modifier le style des éléments et modes suivants :

  • Question

  • Exemple de question

  • Réponse

  • Exemple de réponse

  • Réponse correcte

  • Réponse erronée

 

Vous pouvez accéder à l'éditeur de style de carte depuis les boîtes de dialogue Maintenir les cartes et Editer les chapitres.

Par conséquent, les styles définis dans l'éditeur de style de carte sont valides pour la seule carte active dans la boîte de dialogue Maintenir les cartes ou pour le chapitre actif dans la boîte de dialogue Editer les chapitres.

 

Pour modifier le style

  1. Ouvrez l'éditeur de style de carte

    • en cliquant sur Ajouter/Editer un style de carte dans la boîte de dialogue Maintenir les cartes ou

    • en cliquant sur Ajouter/Editer un style dans la boîte de dialogue Editer les chapitres.

  2. Dans la boîte de dialogue Editeur de style de carte, cliquez sur l'onglet de l'élément ou mode dont vous souhaitez modifier le style.

  3. Cochez la case Couleur du premier plan pour sélectionner la couleur.

  4. Cochez la case Couleur d'arrière-plan pour sélectionner la couleur de l'arrière-plan.

  5. Cochez la case Famille de police pour spécifier une police.

  6. Dans le champ Style de police, désélectionnez la case Aucune pour sélectionner un style de police (gras, rayé, normal, italique ou souligné).

  7. Cochez la case Taille des caractères pour sélectionner ou indiquer une taille de caractères.

  8. Cochez la case Alignement horizontal pour sélectionner ou indiquer une valeur d'alignement.

  9. Cochez la case Alignement vertical pour sélectionner ou indiquer une valeur d'alignement.

  10. Dans le champ Autres éléments, ajoutez d'autres éléments css pour personnaliser davantage le style.

  11. Cliquez sur OK pour appliquer le style.

Remarque :
Un aperçu s'affiche dans le volet droit.
Vous pouvez cliquer sur Effacer le style à tout moment pour réinitialiser le style par défaut.
