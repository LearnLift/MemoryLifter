Installation/démarrage de Memory Lifter

  1. Téléchargez MemoryLifter.

  2. Ouvrez le fichier exécutable (.exe) pour installer MemoryLifter.

 

 

  1. Sélectionnez ou désélectionnez les options Enregistrer MemoryLifter et Afficher l'aide après le démarrage.

  2. Cliquez sur Suivant.

 

 

  1. Sélectionnez un dossier pour vos modules d'apprentissage.

  2. Sélectionnez ou désélectionnez l'option Copier la démo du module d'apprentissage sur le chemin d'accès sélectionné.

  3. Cliquez sur Suivant.

 

 

  1. Enregistrez le logiciel en :

    • indiquant votre nom dans le champ Nom ;

    • indiquant votre adresse e-mail dans le champ E-mail ;

    • indiquant un commentaire dans le champ Commentaire ;

    • sélectionnant ou désélectionnant l'option d'informations par e-mail des nouveautés concernant MemoryLifter et Learnlift.
       

  1. Cliquez sur Terminer et commencez l'apprentissage !
