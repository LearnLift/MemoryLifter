Maintenir les cartes

Edition de cartes

Vous pouvez modifier, organiser et rechercher des cartes dans un module d'apprentissage à l'aide de la boîte de dialogue Maintenir les cartes.  

Par défaut, la carte visible à l'écran correspond à la carte affichée pour édition.

 

Pour ouvrir la boîte de dialogue Maintenir les cartes

Dans le menu Edition, cliquez sur Maintenir les cartes, appuyez sur F10 ou cliquez sur dans la barre d'outils.

 

La boîte de dialogue comprend deux sections :
 

  • la partie supérieure contient les données de la carte actuellement sélectionnée ;
     

  • la partie inférieure présente toutes les cartes disponibles dans le module d'apprentissage.

 

dans la partie supérieure, vous pouvez modifier une carte à la fois en y ajoutant ou supprimant des informations.

Dans la partie inférieure, ou aperçu du module d'apprentissage, vous pouvez organiser les cartes de votre module d'apprentissage (rechercher, supprimer ou déplacer une carte ou un nombre sélectionné de cartes vers un autre chapitre ou une autre boîte).

 

Pour bloquer une carte

  1. Sélectionnez la carte dans la partie inférieure de la fenêtre.

  2. Désélectionnez l'option La carte est active.
    Remarque :
    Si vous désélectionnez l'option La carte est active, la carte est transférée dans la boîte "Aucun".

 

Pour modifier une carte

  1. Sélectionnez la carte dans la partie inférieure de la fenêtre.

  2. Modifiez-la dans la partie supérieure de la fenêtre.

  3. Cliquez sur Modifier pour appliquer les modifications.

 

Pour ajouter une nouvelle carte

  1. Cliquez sur Nouvelle carte.

  2. Renseignez les champs dans la partie supérieure de la fenêtre.

  3. Cliquez sur Ajouter la carte.

 

Pour ajouter un fichier image, audio ou vidéo

Vous pouvez ajouter des fichiers multimédia à la carte.

 

Pour entrer des caractères spéciaux

Si vous souhaitez entrer des caractères spéciaux, cliquez sur le bouton mappe de caractères .

 

Pour entrer des synonymes

Vous pouvez ajouter des synonymes sur votre carte.

 

Pour entrer des distracteurs

Vous pouvez ajouter des distracteurs sur votre carte.

 

Pour prévisualiser une carte

Si vous souhaitez afficher un aperçu de la carte sur laquelle vous travaillez, cliquez sur .

 

Pour ajouter ou modifier le style de carte

  1. Cliquez sur la carte dont vous souhaitez modifier le style.

  2. Cliquez sur Ajouter/Editer un style de carte.

  3. Modifiez le style des éléments visibles dans les champs Question et Réponse de la fenêtre d'apprentissage à l'aide de l'éditeur de style de carte.

 

Utilisation de plusieurs cartes

Vous pouvez rechercher et sélectionner plusieurs cartes et désactiver, trier, supprimer et affecter plusieurs cartes dans un nouveau chapitre, dans la boîte de dialogue Maintenir les cartes.

 

Remarque :
Pour enregistrer les modifications, vous devez cliquer sur Fermer dans la boîte de dialogue Maintenir les cartes.
