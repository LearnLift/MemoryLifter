Création de votre module d'apprentissage

Pour créer un module d'apprentissage

  1. Dans le menu Fichier, cliquez sur Nouveau…, appuyez sur Ctrl+N ou cliquez sur dans la barre d'outils.
     

  2. Dans l'assistant Créer un nouveau module d'apprentissage, choisissez en premier l'emplacement où vous souhaitez enregistrer votre nouveau module d'apprentissage.

  3. Cliquez sur Suivant.

  4. Saisissez ou sélectionnez les détails de votre module d'apprentissage, notamment

    • le nom de votre module d'apprentissage dans le champ Nom ;

    • la catégorie de votre module d'apprentissage dans le champ Catégorie ;

    • le nom de l'auteur du module d'apprentissage dans le champ Auteur ;

    • une brève description de votre module d'apprentissage dans le champ Description.

  5. Cliquez sur Suivant.

  6. Dans la page de l'assistant Paramètres des faces,

    •  renseignez les champs Titre correspondant à la question et à la réponse.

    •  saisissez ou sélectionnez dans les champs Langue la langue des questions et des réponses.

Remarque :

Vous pouvez modifier les options ultérieurement dans la boîte de dialogue Propriétés du menu Fichier.
 

  1. Cliquez sur Terminer.
     

  2. Dans la boîte de dialogue Gestion des chapitres procédez comme suit :

 

    • pour utiliser le titre de chapitre par défaut (chapitre 1), cliquez sur OK. 

ou

    • pour modifier le titre et la description du chapitre par défaut, modifiez les données dans les champs Titre et Description, cliquez sur Appliquer, puis sur OK.
      ou
    • pour créer d'autres chapitres et les organiser suivant un ordre spécifique, reportez-vous à la section Gestion des chapitres.
