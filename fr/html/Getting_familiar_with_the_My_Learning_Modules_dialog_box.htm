Familiarisation avec la page de démarrage

 

 

La page de démarrage (Mes modules d'apprentissage) s'affiche chaque fois que vous lancez MemoryLifter.

 

Remarque :

Si vous ne voulez pas que la page de démarrage s'affiche à chaque lancement de MemoryLifter, cochez la case Ne pas afficher la page de démarrage au bas de la fenêtre.

 

Pour accéder à tout moment à la page de démarrage, sélectionnez Ouvrir…dans le menu Fichier, appuyez sur Ctrl+O ou cliquez sur dans la barre d'outils.

 

 

La page de démarrage est constituée des parties principales suivantes :

 

  1. Mes modules d'apprentissage, champ

  2. Aperçu, champ

  3. Statistiques, champ

  4. Modules d'apprentissage, champ

  5. Modules d'apprentissage récents, champ

  6. Nouveautés, champ

 

La page comporte de plus des boutons, permettant de passer d'une vue à l'autre et de réorganiser les modules d'apprentissage.

Le champ de recherche permet d'effectuer une recherche directe. Le résultat de la recherche s'affiche directement dès la saisie de chaque caractère. Pour redéfinir le champ de recherche, cliquez sur .

 

 

Mes modules d'apprentissage, champ

Le champ Mes modules d'apprentissage comporte une liste des modules d'apprentissage enregistrés dans votre dossier par défaut Modules d'apprentissage.

Une icône figure en regard du nom de chaque module d'apprentissage.

 

  • représente un module d'apprentissage normal

  • représente un module d'apprentissage bloqué

 

La commande glisser-déposer vous permet d'ajouter des modules d'apprentissage dans le champ Mes modules d'apprentissage à partir d'emplacements divers.

Un menu contextuel s'affiche lorsque vous cliquez sur un module d'apprentissage à l'aide du bouton droit de la souris. L'aide de ce menu vous permet d'ouvrir le module d'apprentissage, de le supprimer, de le renommer et de modifier ses propriétés.

 

 

Aperçu, champ

Le champ Aperçu affiche une carte du module d'apprentissage actuellement sélectionné.

 

Statistiques, champ

Le champ Statistiques affiche les informations relatives au module d'apprentissage actuellement sélectionné : longueur de la dernière session d'apprentissage, nombre de cartes demandées, nombre de bonnes réponses, nombre de mauvaises réponses, proportion de cartes connues dans le module d'apprentissage, durée moyenne requise pour répondre à une carte et nombre de cartes étudiées par minute.

 

Modules d'apprentissage, champ

Le champ Modules d'apprentissage indique l'emplacement du module d'apprentissage actuellement sélectionné.

 

Modules d'apprentissage récents, champ

Le champ Modules d'apprentissage récents affiche les modules d'apprentissage récemment ouverts.

 

Nouveautés, champ

Le champ Nouveautés affiche les nouveautés relatives à MemoryLifter et aux modules d'apprentissage proposés par LearnLift.

 

Boutons, boutons Lien

Le bouton permet de masquer les champs Aperçu, Statistiques, et Nouveautés et de n'afficher que le champ Modules d'apprentissage.

 

Grâce au bouton , vous pouvez organiser vos modules d'apprentissage par emplacement, catégorie ou auteur.

 

Le bouton vous permet de passer d'une vue à l'autre, par exemple "Grandes icônes", "Liste", "Détails" ou "Mosaïques".

 

 

Vous pouvez créer un nouveau module d'apprentissage en cliquant sur Nouveau....

 

En cliquant sur Ouvrir à partir de..., vous pouvez rechercher un module d'apprentissage enregistré à un emplacement autre que le dossier par défaut.
