Ignorer les cartes déjà connues

Vous pouvez configurer MemoryLifter pour qu'il ignore les cartes que vous connaissez déjà.

 

Pour ignorer la carte actuelle

  1. Dans le menu Edition, cliquez sur Maintenir les cartes... ou sur dans la barre d'outils.

  2. Désélectionnez l'option La carte est active.

 

Pour ignorer une carte spécifique

  1. Dans le menu Edition, cliquez sur Maintenir les cartes... ou sur dans la barre d'outils.

  2. Sélectionnez la carte souhaitée dans l'aperçu du module d'apprentissage.

  3. Désélectionnez l'option La carte est active.
