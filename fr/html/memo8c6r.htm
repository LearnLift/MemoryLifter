Créer une sauvegarde

Pour créer une copie de sauvegarde d'un module d'apprentissage

 

  1. Dans le menu Fichier, cliquez sur Créer une sauvegarde....

  2. Choisissez l'emplacement où enregistrer le fichier du module d'apprentissage et saisissez un nom.

 

Remarque :

Vous devez disposer des autorisations requises pour créer une copie de sauvegarde en copiant le module d'apprentissage en cours.
