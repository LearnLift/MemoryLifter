Apprentissage des cartes de la boîte spécifiée uniquement

Par défaut, MemoryLifter appelle les cartes de toutes les boîtes.

Vous pouvez toutefois choisir de n'apprendre les cartes que d'une boîte spécifique.

 

Pour apprendre les cartes de la boîte spécifiée uniquement

 

  • Dans le menu Apprendre, cliquez sur Sélectionner la boîte et sélectionnez la boîte que vous souhaitez que MemoryLifter utilise pour appeler les cartes.
    Remarque :     la boîte sélectionnée est indiquée par une coche.
