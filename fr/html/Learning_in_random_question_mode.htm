Apprentissage en mode question aléatoire

Pour augmenter le niveau de difficulté, MemoryLifter peut afficher les questions extraites du pool de manière aléatoire.

 

Pour apprendre en mode question aléatoire

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

  2. Cliquez sur l'onglet Général.

  3. Sélectionnez Pool aléatoire.
