Réduction

L'option Réduction renvoie le programme dans la barre d'état système. Il y est réduit à l'état de simple icône en attendant que vous double-cliquez dessus où qu'un laps de temps donné s'écoule.

 

Pour activer le mode Réduction

Dans le menu Outils, cliquez sur Mode réduction, appuyez sur Ctrl+S, ou cliquez sur dans la barre d'outils.

 

 

Pour modifier les options de réduction

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

  2. Cliquez sur l'onglet Réduire/Quitter le planificateur.

  3. Sélectionnez l'option appropriée.  
    Les options de réduction sont décrites dans les Options d'apprentissage..
