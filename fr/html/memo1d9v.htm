Options d'apprentissage

Personnalisation de votre session d'apprentissage

MemoryLifter intègre de nombreuses options permettant de personnaliser votre session d'apprentissage.

 

Pour personnaliser les options d'apprentissage dans la boîte de dialogue Options d'apprentissage

Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

 

 

Enseignants, onglet

Vous pouvez définir comment sont évaluées vos réponses.

Les paramètres figurant dans les autres onglets (ci-dessous) sont automatiquement modifiés suivant le type de pédagogie choisie.

Pour connaître les paramètres qui sont modifiés en détail, se reporter à la section Enseignants.

Vous n'avez donc pas à modifier les paramètres des autres onglets.  

Testez chaque type de pédagogie pour découvrir celui que vous préférez.

Vous pouvez toujours restaurer les paramètres par défaut.

 

 

 

Mode d'apprentissage, onglet

 

Direction

Vous pouvez définir ici le sens des questions posées.  Il existe trois possibilités :

  • Question => Réponse, dans le cas d'un module Espagnol => Anglais, les questions seraient en espagnol et les réponses en anglais.
  • Réponse => Question, dans le cas d'un module Espagnol => Anglais, les questions seraient en anglais et les réponses en espagnol.
  • Mixte, dans ce cas, le sens est sélectionné de manière aléatoire pour chaque carte, dans un module Espagnol => Anglais, il pourra y avoir parfois des questions en espagnol et des réponses en anglais et parfois des questions en anglais et des réponses en espagnol.

 

 

Modes d'apprentissage

MemoryLifter propose cinq modes d'apprentissage :

 

Icône

Mode d'apprentissage

Description

Mode standard

MemoryLifter affiche la question dans le volet gauche tandis que vous saisissez la réponse dans le volet droit.
Toutes les informations disponibles (image, son, exemples, etc...) sur la question s'affichent.

Choix multiple

MemoryLifter vous indique plusieurs réponses possibles parmi lesquelles vous pouvez choisir.

Phrases

MemoryLifter vous demande de formuler une phrase et non de répondre un mot.

Compréhension auditive

MemoryLifter lit le mot pour vous. Il ne vous reste plus qu'à entrer la réponse.

Reconnaissance des images

MemoryLifter affiche l'image. Il ne vous reste plus qu'à entrer la réponse.

 

Si vous sélectionnez plusieurs modes, MemoryLifter choisit un mode aléatoire pour chaque carte.

 

Pour changer les modes d'apprentissage, utilisez l'onglet Mode d'apprentissage ou le menu Choisir le(s) mode(s) d'apprentissage de la barre d'outils.

 

En fonction du/des mode(s) d'apprentissage sélectionné(s), l'icône change de couleur.

 

Remarque :

Si vous ne pouvez pas afficher la carte dans le mode d'apprentissage sélectionné (absence d'image, de son, de phrase ou manque de cartes), le mode d'apprentissage standard est utilisé.

 

Vous pouvez spécifier les options de choix multiple.

 

 

 

Synonymes, onglet

 

Si vous ajoutez des synonymes à vos cartes, vous pouvez ajuster le comportement du programme en fonction du nombre de réponses que vous connaissez.

 

  • Promouvoir si 100% connues - Ne place la carte dans la boîte suivante que si vous connaissez tous les synonymes. Dans le cas contraire, la carte est renvoyée dans la boîte 1.

  • Promouvoir si >= 50% connues - Promeut la carte lorsque vous connaissez au moins la moitié des synonymes.

  • Promouvoir si premier mot connu - Ne promeut la carte que si vous connaissez le premier synonyme. Pratique lorsque le sens le plus courant est placé en haut de la liste.

  • Promouvoir si un mot connu - Promeut la carte si vous connaissez l'un des synonymes.

  • Inviter si < 100% - L'utilisateur est interrogé lorsque toutes les réponses ne sont pas connues.

 

 

Saisie, onglet

 

Corriger à la volée

Sélectionnez Corriger à la volée, si vous souhaitez que votre réponse soit corrigée à la volée (vous ne pouvez saisir que les bons caractères, dans le cas contraire le curseur ne bouge pas).

 

Sensibilité à la casse

Sélectionnez Sensibilité à la casse si vous souhaitez vérifier la casse pendant la saisie de la réponse.

 

Auto-évaluation

Sélectionnez Auto-évaluation si vous souhaitez décider vous-même de promouvoir ou rétrograder la carte.

 

Ignorer ces caractères pendant la correction :

Entrez dans cette zone de texte les caractères de ponctuation à ne pas corriger.

Vous êtes ainsi plus libre puisque vous n'avez pas à intégrer la ponctuation à votre réponse pour qu'elle soit correcte.

 

Erreurs typographiques
Si vous avez sélectionné l'option Corriger à la volée dans l'onglet Saisie, vous pouvez configurer MemoryLifter pour promouvoir/rétrograder les cartes en fonction de vos erreurs typographiques dans la zone de texte Erreurs typographiques. Quatre options sont disponibles :

 

  • Promouvoir uniquement si aucune - La carte est transférée dans la boîte suivante si la réponse ne comporte aucune erreur typographique.

  • Promouvoir si > 50% - Promouvoir si plus de la moitié du mot est correctement orthographiée.

  • Toujours promouvoir - Quel que soit leur nombre, les erreurs typographiques sont admises.

  • Inviter - Invite l'utilisateur à promouvoir la carte suite à des erreurs typographiques.

 

 

 

 

Réduire/Quitter le planificateur, onglet

Vous pouvez choisir de réduire ou de quitter l'application après un laps de temps, un nombre de cartes ou un nombre de réponses correctes donné.

En sélectionnant une ou plusieurs de ces options, le programme est réduit dans la barre d'état du système à la fin d'une session.

Il surgit de nouveau automatiquement une fois la durée spécifiée écoulée.

Vous pouvez, en option, paramétrer la fermeture complète du programme après la session.

 

 

 

Général, onglet

Vous pouvez sélectionner les options générales suivantes :
 

  • Horloge de temporisation :
    Répondez en temps limité.  Un temps limite est fixé avant l'envoi automatique de votre réponse.

  • Afficher les statistiques :
    Programmez MemoryLifter pour afficher les statistiques de la session après chaque session d'apprentissage.

  • Afficher les images :
    Le cas échéant, la(les) image(s) s'affiche(nt).

  • Sons à lecture automatique :
    Si une bande audio est disponible, elle est diffusée pendant l'affichage de la question/réponse.

  • N'autoriser qu'une seule instance de MemoryLifter :
    Si cette option est sélectionnée, MemoryLifter veille à ce que vous ne puissiez exécuter qu'une seule session d'apprentissage à la fois.
    Double-cliquer sur un autre fichier de module d'apprentissage ferme le module d'apprentissage ouvert et ouvre le nouveau module.
    Si cette option n'est pas sélectionnée, une autre session d'apprentissage démarre en parallèle.

  • Ecouter le commentaire audio :
    Le cas échéant,     des commentaires audio sont joués automatiquement.

  • Ignorer l'affichage des réponses correctes :
    MemoryLifter passe directement à la question suivante une fois la réponse saisie. Les réponses correctes ne sont pas affichées.

  • Pool aléatoire :
    De nouvelles cartes sont sélectionnées aléatoirement dans le pool. Dans le cas contraire, les cartes sont appelées dans l'ordre dans lequel elles ont été ajoutées au module d'apprentissage.

  • Valider la rétrogradation :
    Vous devez confirmer la rétrogradation des cartes avant que celle-ci ne soit effective.
