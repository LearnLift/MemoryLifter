Modifier la langue

Memory Lifter est disponible dans les langues suivantes :

 

  • Anglais

  • Français

  • Allemand

  • Portugais

  • Espagnol

 

 

Pour modifier la langue

  • Dans le menu Outils, cliquez sur Changer la langue, puis cliquez sur la langue souhaitée.

Remarque :

La langue sélectionnée est indiquée par une coche.
