Utilisation du planificateur d'apprentissage automatique

MemoryLifter optimise la durée de votre apprentissage grâce à un planificateur d'apprentissage automatique personnalisable.

 

Ce planificateur se comporte comme une horloge qui vous invite régulièrement à entamer une session d'apprentissage pour vous permettre de mémoriser les informations plus rapidement et plus longtemps.

 

Si vous êtes occupé et souhaitez faire une pause pendant votre session d'apprentissage, allez dans le menu Outils et cliquez sur Mode Réduction ou appuyez sur Ctrl+S ; cette action vous permet de réduire momentanément MemoryLifter dans la barre des tâches. Il s'affichera à nouveau une fois le laps de temps écoulé .  

 

Vous pouvez également personnaliser le bouton de réduction pour qu'il se déclenche automatiquement en fonction d'autres événements (après avoir répondu à x questions correctement ou après avoir passé en revue x cartes pendant une session d'apprentissage).

 

 

Pour personnaliser le planificateur d'apprentissage

  1. Dans le menu Apprendre, cliquez sur Options d'apprentissage ou cliquez sur dans la barre d'outils.

  2. Cliquez sur l'onglet Réduire/Quitter le planificateur.

  3. Sélectionnez l'option appropriée.  
    Les options de réduction sont décrites dans les Options d'apprentissage..
