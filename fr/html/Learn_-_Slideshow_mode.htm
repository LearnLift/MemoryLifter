Apprentissage en mode diaporama

Le mode diaporama vous permet de parcourir toutes les cartes que vous avez sélectionnées.

Les questions et les réponses sont visibles.

Ce mode est idéal pour un apprentissage sans évaluation.

 

Pour apprendre en mode diaporama

 

  1. Dans le menu Apprendre, cliquez sur Mode diaporama ou cliquez sur dans la barre d'outils.

  2. Cliquez sur la Flèche droite pour afficher la carte suivante (question et réponse) ou sur la Flèche gauche pour afficher la carte précédente.

  3. Pour revenir en mode normal, utilisez le menu à bascule Mode diaporama ou la barre d'outils comme à l'étape 1.
