Aprendizagem no modo padrão

Você pode responder uma pergunta por entrada padrão ou selecionando resposta(s) no modo de múltiplas escolhas.

Isso depende do modo de aprendizagem selecionado.

 

Para responder uma pergunta no modo padrão
 

  1. Certifique-se de que o cartão de memorização da resposta esteja no modo de entrada (cartão de memorização com uma caixa de texto no centro).
     

  2. Insira a resposta na caixa de texto.
    Use ENTER para continuar.
    Se uma pergunta tiver mais de uma resposta correta (sinônimos), pressionar ENTER o direciona para uma linha para inserir a outra resposta.
    Caso contrário, (ou depois de inserir o último sinônimo), pressionar ENTERfaz com que sua resposta seja enviada.

  3. Depois de enviar a resposta, você pode revisá-la e o botão da seta direita será ativado, permitindo que você vá para o próximo cartão.

 

Notas:

  • Se não souber a resposta, pressione ENTER e o MemoryLifter mostrará as respostas.
  • Se precisar inserir caracteres especiais, use o mapa de caracteres.

 

 

Para editar um cartão

Se quiser modificar um cartão durante o aprendizado, no menu Editar, clique em Manter cartões..., pressione F10 ou clique em na barra de ferramentas.
A caixa de diálogo Manter cartões é mostrada e você pode alterar todos os elementos de cada cartão.
Clique em Salvar Alterações para aplicar as alterações e depois em Fechar para continuar aprendendo.

 

Para finalizar uma sessão de aprendizagem

  1. Clique no botão X no canto superior direito da janela.

  2. Se quiser finalizar a sessão de aprendizagem, responda a nova caixa de diálogo com Sim, Cancelar retorna para o aprendizado.

  3. As estatísticas desta rodada são mostradas, clique em OK
