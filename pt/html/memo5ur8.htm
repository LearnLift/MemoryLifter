Exportar Cartões

Exportar um Módulo de Aprendizagem para um arquivo de texto

Use esta opção para exportar os cartões de um Módulo de Aprendizagem para processar de alguma outra forma (por exemplo, editá-los em uma planilha ou processador de texto).

 

  1. No menu Editar, clique em Exportar cartões... ou pressione Ctrl+E

  2. Na caixa de diálogo Exportar

    • especifique os capítulos para exportar usando o seletor de capítulos.

    • especifique os campos a serem exportados usando o seletor de campos.

    • especifique o formato de arquivo - quer você queira que seus dados sejam separados por ponto-e-vírgula, vírgula ou tabulação.

    • Clique em Exportar.

  3. Selecione a pasta de destino e digite o nome do arquivo exportado na caixa Nome de arquivo.

  4. Clique em Salvar.
