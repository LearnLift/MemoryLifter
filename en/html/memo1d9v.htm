Learning Options

Customizing your learning session

MemoryLifter provides many options for you to customize your learning session.

 

To customize your learning options in the Learning Options dialog box

Within the Learn menu, click Learning Options... or click on the toolbar.

 

 

Teachers tab

You can define how your answers should be assessed.

Depending on which teacher type you choose, settings in the other tabs (below) are changed automatically.

To see which setting are changed in detail, see Teachers.

You do not have to change any settings in the other tabs.  

Try each teacher to see which you prefer.

You can always reset to the Default.

 

 

 

Learning Mode tab

 

Direction

Here you can determine in which direction questions are asked.  There are three possibilities:

  • Question => Answer, in the case of a Spanish => English module this would be Spanish questions and English answers.
  • Answer => Question, in the case of a Spanish => English module this would be English questions and Spanish answers.
  • Mixed, in this case the direction will be randomly chosen for each card, in a Spanish => English module sometimes it will be Spanish questions and English answers and sometimes English questions and Spanish answers.

 

 

Learning modes

MemoryLifter offers five learning modes:

 

Icon

Learning mode

Description

Standard mode

MemoryLifter shows the question on the left side and you type the answer on the right side.
All available information (i.e., image, sound, example sentences, etc...) for the question is displayed.

Multiple choice

MemoryLifter shows you a specified number of possible answers you can choose from.

Sentences

MemoryLifter asks you to enter the example sentences instead of the word.

Listening Comprehension

MemoryLifter reads the word to you, and you have to enter the answer.

Image Recognition

MemoryLifter shows you the image and you have to enter the answer.

 

If you check more than one mode, MemoryLifter will choose a random mode for each card.

 

You can change the learning modes either by using the Learning mode tab or by using the Select Learning Mode(s) menu on the toolbar.

 

Depending on the selected learning mode(s) the icon changes its color.

 

Note:

If it is not possible to display the card in the selected learning mode (no image, audio, sentence, or not enough other cards), the standard learning mode will be used.

 

Multiple choice options can be specified.

 

 

 

Synonyms tab

 

If you add synonyms to your cards, you can adjust how the program reacts depending on how many you enter correctly.

 

  • Promote when 100% known - Only puts the card in the next box when you know all synonyms. Otherwise, the card is demoted to box 1.

  • Promote when >= 50% known - Promotes the card when you know at least half of the synonyms.

  • Promote when first word known - Promotes the card only when you know the very first synonym. Especially useful when you put the most common meaning first.

  • Promote when one word known - Promotes the card when you know any one of the synonyms.

  • Prompt when not 100% - When not all answers are correct, prompt user.

 

 

Typing tab

 

Correct on the fly

Check Correct on the fly, if you want your input to be corrected on the fly, i.e., you can only type correct characters, otherwise, the cursor will not move forward.

 

Case sensitivity

Check Case sensitivity, if you want your input to be checked against the case of the answer.

 

Self assessment

Check Self assessment, if you want to decide whether to promote or demote the card.

 

Ignore these characters during correction

In this box, type common punctuation characters that you don't want to be corrected.

This gives you more freedom as you do not have to type punctuation characters in your answer to get it correct.

 

Typing mistakes
If you have selected the Correct on the fly option on the Typing tab, you can select how MemoryLifter should promote/demote cards depending on your typing mistakes in the Typing mistakes box. There are four options:

 

  • Only promote when none - Card will be put in the next box if there are no typing mistake at all.

  • Promote when > 50% - Promotes when more than half of the word is written correctly.

  • Always promote - Any amount of typing mistakes allowed.

  • Prompt - Prompt user on whether to promote when there have been typing mistakes.

 

 

 

 

Snooze/Quit Planner tab

You can choose to snooze or quit the application after a certain amount of time / cards / correct answers.

By selecting one or more of these options, the program will minimize itself to the sysem tray after a session.

It automatically pops up again after the specified time.

Alternatively, the program can quit completely after the session.

 

 

 

General tab

You can select the following general options:
 

  • Countdown timer:
    Answer against the clock.  This sets a time limit before your answer is automatically submitted.

  • Display statistics:
    Have MemoryLifter show the session statistics after each learning session.

  • Display images:
    If available, image(s) will be shown.

  • Auto-play sounds:
    If available, sounds will be played upon displaying the question/answer.

  • Allow only one instance of MemoryLifter:
    If selected, MemoryLifter ensures that only one learning session is possible at a time.
    Double-clicking another Learning Module file will close the current Learning Module and open the new one.
    If this option is cleared, another parallel learning session will start.

  • Play commentary sounds:
    If available,     commentary sounds will be played automatically.

  • Skip showing correct answers:
    MemoryLifter will proceed right away to the next question after typing the answer – correct answers will not be shown.

  • Random pool:
    New cards are taken randomly from the pool. Otherwise, cards are asked in the same order as they were added to the Learning Module.

  • Confirm Demote:
    You have to confirm before cards are demoted.
