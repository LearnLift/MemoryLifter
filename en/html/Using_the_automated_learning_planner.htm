Using the automated learning planner

MemoryLifter optimizes your learning time with a customizable automated learning planner.

 

This automated learning planner acts like a learning alarm clock to regularly quiz you and help remember information faster and longer.

 

If you are busy and want to pause while learning, within the Tools menu click Snooze Mode or press Ctrl+S and MemoryLifter will be minimized to the taskbar for a specific amount of time and then popup to start again.  

 

Snooze mode can also be customized to automatically trigger from other events like answering x number of questions right, or after x number of cards during a learning session.

 

 

To customize the learning planner

  1. Within the Learn menu, click Learning Options... or click on the toolbar.

  2. Click the Snooze/Quit planner tab.

  3. Select the appropriate option.  
    See Learning Options for a description of the snooze options.
