Show / Edit Box sizes

The Leitner box system is an integral part of MemoryLifter's learning algorithm.

As you learn, MemoryLifter promotes and demotes cards through a series of 10 boxes.

At any time, you can see the distribution of the cards over the boxes currently in your learning session.

You can also change the maximum number of cards each box can hold.

 

To view the box sizes

Within the Learn menu, click Show/Edit Box Sizes…, press Ctrl+B, or click on the toolbar to show the Box Sizes dialog box.

 

To change box sizes

In the text boxes within the Box Sizes dialog box, type or select the amount of cards you want to allow in each box.

 

Note:

To reset the box sizes, click Default.
