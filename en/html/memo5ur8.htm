Export Cards

Exporting a Learning Module to a text file

Use this option to export the cards of a Learning Module for processing in some other way (e.g. editing them in a spreadsheet or word processor).

 

  1. Within the Edit menu, click Export Cards... or press Ctrl+E

  2. In the Export dialog box

    • specify the chapters to export using the chapter selector.

    • specify the fields to export using the field selector.

    • specify the file format – whether you want your data to be separated by semicolon, comma, or tab.

    • Click Export.

  3. Select the destination folder and type the name for the exported file in the File name box.

  4. Click Save.
