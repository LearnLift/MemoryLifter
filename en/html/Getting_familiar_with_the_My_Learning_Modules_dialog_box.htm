Getting familiar with the start page

 

 

The start page (My Learning Modules) is shown every time you start MemoryLifter.

 

Note:

If you do not want to see the start page each time you start MemoryLifter, check Don't show start page at the bottom of the window.

 

You can access the start page any time by either selecting Open… within the File menu, by pressing Ctrl+O, or by clicking on the toolbar.

 

 

The start page consists of these main parts:

 

  1. My Learning Modules box

  2. Preview box

  3. Statistics box

  4. Learning Modules box

  5. Recent Learning Modules box

  6. News box

 

Additionally, there are buttons and to switch to different views and to reorganize the Learning Modules.

The search box performs a live search, so that the result of the search is shown directly immediately after each character is entered. By clicking you can reset the search box.

 

 

My Learning Modules box

In the My Learning Modules box you find a list of the Learning Modules saved in your default Learning Modules folder.

There are icons next to the name of each Learning Module.

 

  • stands for a normal Learning Module

  • stands for a locked Learning Module. This Learning Modules can only be opened with MemoryLifter 2.3.

 

You can add Learning Modules from different locations to the My Learning Modules box by drag & drop.

If you right-click on a Learning Module, a context menu appears. You can open the Learning Module, delete it, rename it and change the its properties with the help of this menu.

 

 

Preview box

The Preview box shows one card from the currently selected Learning Module.

 

Statistics box

The Statistics box shows information about the currently selected Learning Module: the length of the last learning session, the number of cards asked, answered right, answered wrong, the ratio of known cards in the Learning Module, the average time needed to answer one card, and the amount of cards learned per minute.

 

Learning Modules box

The Learning Modules box shows the location of the currently selected Learning Module.

 

Recent Learning Modules box

The Recent Learning Modules box shows recently opened Learning Modules.

 

News box

The News box shows the news about MemoryLifter and the Learning Modules offered by LearnLift.

 

Buttons, Link buttons

With the help of the button you can hide the Preview, Statistics, and News boxes, and show only the Learning Modules box.

 

With the help of the button you can organize your Learning Modules by location, category, or author.

 

With the help of the button you can switch to different views, like "Large Icons", "List", "Details" or "Tiles".

 

 

By clicking on New..., you can create a new Learning Module.

 

By clicking on Open from..., you can browse for a Learning Module saved in a different location than the default folder.
