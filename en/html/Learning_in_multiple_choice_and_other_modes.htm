Learning in multiple choice mode

In addition to standard text input, MemoryLifter offers many other learning modes including multiple choice, image recognition, listening comprehension, self assessment, and random mode.

 

To learn in multiple choice mode

  1. On the Learn menu, click Learning Options or click on the toolbar.
     

  2. Click the Learning Mode tab.
     

  3. In the Learning Modes box, check the Multiple choice Learning mode.

 

Note:

If you select more than one mode, MemoryLifter will choose a random one from the checked ones every time a new card is displayed.

 

 

To answer a question in multiple choice mode

  1. Click the answer(s) or type in the number(s) of the answers you think is/are the right.

  2. Click on or press ENTER

 

or
If you don't know the answer, press ENTER.
The correct answer is shown, the card is added to the stack.
It will be asked again after a short time.

 

 

When learning in multiple choice mode, you can choose from different Multiple choice options.

MemoryLifter usually shows you a specified number of possible answers taken from the pool of answers, i.e. random distractors.

You can choose how many possible answers you want to see in multiple choice mode.

 

If you have created cards with distractors, you do not have to allow random distractors.  Defining specific incorrect answers increases the difficulty of the learning module.

 

Additionally you can allow multiple correct answers and define how many correct answers at most you want to have in multiple choice mode.

 

Note:

In multiple choice mode MemoryLifter will only assess an answer as correct if all of the selected answers are correct.

 

 

To set multiple choice options

 

 

  1. Within the Learn menu, click Learning Options... or click on the toolbar.
     

  2. Click the Learning Mode tab.
     

  3. In the Learning Modes box, check the Multiple choice learning mode.
     

  4. In the Multiple choice options box, select Allow random distractors to allow MemoryLifter to include additional distractors (in addition to those defined for the card) from the pool of answers.
     

  5. In the Multiple choice options box, select Allow multiple correct answers to allow MemoryLifter to have more than one correct answer for a question.
     

  6. In the Number of choices box, type or select, how many possible answers you want a question to have in multiple choice mode.
     

  7. In the Max. number of correct answers box, type or select, the maximum number of correct answers a question could have in multiple choice mode.
    Note    : The actual number of correct items is set in the card, this is just a max number for cards displayed during this session.
     

  8. Click OK to start learning in multiple choice mode.

 

 

 

To learn more about the other learning modes and other exciting options, check out the learning options.
