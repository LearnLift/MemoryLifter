Learning in standard mode

You can answer a question by standard input or by selecting answer(s) in multiple choice mode.

It depends on the learning mode you selected..

 

To answer a question in standard mode
 

  1. Make sure that the answer flashcard is in input mode (flashcard with a textbox in the center).
     

  2. Enter the answer in the text box.
    Use ENTER to proceed.
    If a question has multiple correct answers (synonyms) ENTER takes you down a line to enter the next one.
    Otherwise (or after the last synonym) ENTER submits your answer.

  3. After the answer has been submitted you can review it, and the right arrow button will be activated, allowing you to go to the next card.

 

Notes:

  • If you do not know the answer, press ENTER and MemoryLifter will show the answer(s).
  • If you need to enter special characters, use the character map.

 

 

To edit a card

If you want to modify a card while learning, within the Edit menu, click Maintain Cards..., press F10, or click on the toolbar.
The Maintain Cards dialog box is shown, and you can change all the elements of each card.
Click Save Changes to apply changes, then Close to continue learning.

 

To end a learning session
 

  1. Click the X button in the upper right corner of the window.

  2. If you want to end your learning session, answer the new dialog box with Yes, Cancel brings you back to learning.

  3. Statistics for this round are shown, click OK
