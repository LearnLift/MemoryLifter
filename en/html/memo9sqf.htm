Creating your own Learning Module

To create a Learning Module

  1. Within the File menu, click New…, press Ctrl+N, or click on the toolbar.
     

  2. In the Create new Learning Module wizard, first select where you want to store the new learning module.

  3. Click Next.

  4. Type in or select the details of your Learning Module including,

    • the name of your Learning Module in the Name box,

    • the category for your Learning Module in the Category box,

    • the name of the Learning Module's author in the Author box, and

    • a short description of your Learning Module in the Description box.

  5. Click Next.

  6. On the Side Settings wizard page,

    •  type the titles for question and answer in the Headline boxes.

    •  type or select the language of the questions and answers in the Language boxes.

Note:

You can change the options later in the Properties dialog box in the File menu.
