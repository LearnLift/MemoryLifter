Installing / Starting Memory Lifter

  1. Download MemoryLifter.

  2. Open the exe file to install MemoryLifter.

 

 

  1. Clear or select the check boxes Register MemoryLifter and Show help after startup.

  2. Click Next.

 

 

  1. Select a folder for your Learning Modules.

  2. Clear or select the check boxes Copy Learning Module Demo to the selected path.

  3. Click Next.

 

 

  1. Register by

    • typing your name in the Name field

    • typing your e-mail in the E-Mail field

    • typing any comments in the Comment field

    • clearing or selecting that you want to be informed by e-mail about what's new with MemoryLifter and Learnlift.
       

  1. Click Finish and start learning!
