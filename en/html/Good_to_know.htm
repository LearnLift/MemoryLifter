Good to know

If you have already learned with MemoryLifter before, there is still some information on new features and insider tips we would like to share with you.

 

Learning Module Styles

Furthermore, we extended the ability to modify styles.

You can now modify the style
 

  • of all predefined styles
    (for all Learning Modules opened in future)
    To modify all predefined styles, select global settings in the Style Editor.

  • of the active style only
    (for all Learning Modules opened in future)
    To modify the active predefined style only, select style settings in the Style Editor.

  • of chapters
    (for the active Learning Module only, when Use Learning Module Styles is selected on the Tools menu)
    To modify the style of a chapter, open the Cardstyle Editor in the Manage Chapters dialog box.

  • of cards
    (for the active Learning Module only, when Use Learning Module Styles is selected on the Tools menu)
    To modify the style of a card, open the Cardstyle Editor in the Maintain Cards dialog box.

Note that the properties are inherited.

 

Export

To save your Learning Module in other formats you can export it.

 

If you want to modify your cards with another software tool or if you want to change the order of the question, answer and example fields or if you want to add chapter information to your cards in a quick and easy way, export your Learning Module or specified fields of it to text files first.

 

Import

We recommend to import text files, if you are familiar with saving tables in delimited format.

Whereas importing from clipboard is recommended, if you want to import data while surfing on the internet or if you want to import data directly from other documents.
