Recent Learning Modules

To open a recently opened Learning Module

 

  1. Within the File menu, click Recent Learning Modules
  2. A sub-menu containing the most recently opened learning modules appears.
  3. Click a module in this sub-menu to open it.
 

Note: If a module is currently open, it will be closed.  Also note about the items in the sub-menu, the location of the Learning Module is in parentheses "( )" next to the name.
