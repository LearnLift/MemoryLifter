Cardstyle Editor

With the Cardstyle Editor you can modify the style, i.e. the way all the elements shown in the Question and Answer boxes of the learning window look.

 

You can modify the style of the following elements and modes:

  • Question

  • Question Example

  • Answer

  • Answer Example

  • Answer Correct

  • Answer Wrong

 

You can access the Cardstyle Editor from the Maintain Cards and the Edit Chapters dialog boxes.

The styles defined in the Cardstyle Editor are can be set for one single card which is active in the Maintain Cards dialog box or for one chapter which is active in the Edit Chapters dialog box.

 

To modify the style

  1. Open the Cardstyle Editor

    • by clicking Add/Edit Cardstyle in the Maintain Cards dialog box or

    • by clicking Add/Edit Style in the Edit Chapters dialog box.

  2. In the Cardstyle Editor dialog box click the tab of the element or mode which style you want to modify.

  3. Check the Foreground Color check box to choose a color.

  4. Check the Background Color check box to choose a color for the background.

  5. Check the Font-Family check box to specify a font.

  6. In the Font Style box clear the None check box to select a font style (bold, strikeout, regular, italic or underline).

  7. Check the Font-Size check box to select or type a font size.

  8. Check the Horizontal-Alignment check box to select or type .

  9. Check the Vertical-Alignment check box to select or type .

  10. In the Other Elements box add other CSS elements to personalize the style further.

  11. Click OK to apply the style.

Note:
A preview is displayed on the right hand side.
You can click Clear Style at any time to restore the style to the default.
