Properties

Setting Learning Module options

Within the File menu, click Properties... to display the Learning Module Options dialog box with its tabs:

 

  • Properties

  • Information

  • Headlines / Languages

  • Allowed Learning Options

  • Advanced

 

 

Properties tab

In the Name text box the name of your Learning Module is shown.

  1. In the Category combo box type or select the category for your Learning Module.

  2. In the Author text box you can type the name of the Learning Module's author.

  3. In the Description text box you can type a short description of your Learning Module.

 

Information tab

In the File location box the location of the Learning Module is shown.

Additionally, you can get information on the Number of cards your Learning Module consists of, the Disk space used, the Number of media files, your Module Knowledge percentage and your Highest Module Knowledge percentage.

 

Headlines / Languages tab

  • In the Headline text boxes type the headlines for the question and answer panes.

    Example:

If you are working on a "Deutsch - English" Learning Module, type "Deutsch" in the Headline text box on the Question side and "English" on the Answer side.

If you are working on a "Medical Terms" Learning Module, type "Definition" in the Headline text box on the Question side and "Term" on the Answer side.

 

  • In the Language text boxes type or select the language for the questions and answers of your Learning Module.

 

Allowed Learning Options tab

To specify in which learning modes the cards of your Learning Module may be learned, check different options in the Learning modes and Direction boxes.

It depends on your content which learning modes make sense.

For example, you can learn vocabulary and definitions in both directions.

 

  • To choose which Directions you allow or recommend for learning, select the appropriate check boxes.

 

Note:

The names of the check boxes are adjusted to the titles you typed in the Headline text boxes on the Headlines / Languages tab.

 

  • To choose which Learning modes you allow or recommend for learning, check the appropriate check boxes.

 

Note:

These selections have an effect on the Learning Options dialog box, in which you specify the Learning Mode for your present learning session.

The Directions and Learning modes you disable are grayed and cannot be chosen for learning sessions anymore.

 
 

Advanced tab

The advanced options allow you to

 

  • specify sound files to be played as commentary while learning and

  • add or edit the default style for the Learning Module

 

There are two groups of commentary sounds, with six sounds each.

The first group, displayed on the left hand side, is played when the learning mode direction is "Question => Answer" (default).

The second group, displayed on the right hand side, is played when the learning mode direction is "Answer => Question".

 

Each of the group is divided into two subgroups:

 

The first three sound files prefixed with "Stand-alone:" are used, when there is no audio associated with the card's answer. The commentary could for instance just say "That was correct".

 

The second subgroup is used when the card has audio. This second commentary could be something like "Sorry, the right answer is" followed by the audio of the correct answer (which can be set on each individual card).

 

 

To change the commentary sounds

  1. In the Question or Answer box click the appropriate commentary.

  2. To load an existing sound file or to record a sound file for the selected commentary, click Change Sound .

  3. The standard sound recorder pops up.

 

To add or edit the default style for the Learning Module

  • Click Add/Edit Styles.

  • Use the Card Style Editor.
