Printing cards

You can print out a Learning Module's cards on paper.

In the Select Print Settings dialog box you can select which cards you want to print.

 

You can choose between:

  • All cards

  • Unknown cards

  • Select cards

  • Select chapters

  • Select boxes

 

Alternatively you can switch off the wizard and use the former Print dialog box.

 

 

To print cards

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select which cards you want to print.

  3. Click Next.

 

 

  1. In the Sort cards by box, type or select whether you want the cards to be sorted by:

    • Question

    • Answer

    • Box

    • Chapter

    • None (i.e. the order in which you added the cards to your Learning Module file)

  2. In the Define Print-Style box, type or select whether you want to print the cards

    • with all information (question, answer, example sentences and images) or

    • with question and answer only

  3. If you want to see a print preview, click Print Preview.

  4. If you want to change the page setup, click Page Setup.

  5. Click Print to print the cards.

 

To print all cards

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select All cards.

  3. Continue with step 3 of To print cards.

 

 

To print only the unknown cards

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select Unknown cards.

  3. Continue with step 3 of To print cards.

 

 

To print a specified selection of cards only

 

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select Select cards.

  3. Click Next.

 

 

  1. On the Select the cards to print dialog box, select the cards you want to print.

  2. Continue with step 3 of To print cards.

 

 

 

To print the cards of specified chapters only

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select Select chapters.

  3. Click Next.

 

 

  1. On the Select Chapters to print dialog box, select the chapters you want to print using the Chapter selector.

  2. Continue with step 3 of To print cards.

 

 

To print the cards of specified boxes only

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, select Select boxes.

  3. Click Next.

 

 

  1. On the Select boxes to print dialog box, select the boxes you want to print.

    • Select All boxes to print all cards from your Learning Module.

    • Select Only the pool to print the cards you have not been asked yet only.

    • In the Individual field type the numbers of the boxes you want to print.

  2. Continue with step 3 of To print cards.

 

To print cards without using the print wizard

  1. On the File menu, click Print Flashcards…, press Ctrl+P, or click on the toolbar.

  2. On the Select Print Settings dialog box, click Switch Off the Wizard.

 

 

  1. In the Select chapters box, select the chapters you want to print using the chapter selector.
    By default, all the chapters are selected.

  2. In the Select boxes box, select the boxes you want to print.

  3. In the Print options box, type or select in the Order cards by box, whether you want the cards to be sorted by:

    • Question

    • Answer

    • Box

    • Chapter

    • None (i.e. the order in which you added the cards to your Learning Module file)

  4. In the Print options box, type or select in the Use print style box:

    • Print All (to print the cards with all information - question, answer, example sentences and images) or

    • Print without Media (to print the cards with question and answer only)

  5. Click Print to see the print preview.

  6. In the Print preview box, click Print to start printing.

Note:

You can change back to the print wizard any time by clicking Switch to Wizard.
