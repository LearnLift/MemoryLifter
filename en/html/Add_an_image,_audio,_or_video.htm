Adding an image, audio, or video files

MemoryLifter allows you to add an image, sound, and/or video to your cards to increase their effectiveness.

 

 

To add an audio file

  1. Within the Maintain Cards dialog, click the Audio button .

  2. The sound recorder opens.

  3. Click Browse... to select an audio file.
    or
    Click Record to record your own audio file.

 

Note:

Supported audio file formats are: mp3, wav, mid

 

 

To add a video file

  1. Within the Maintain Cards dialog, click the Video button .

  2. Click Browse.

  3. Select the desired file and click OK.

 

Note:

Supported video file formats are: avi, wmv

 

 

To add an image file

There are two ways to add images.

  1. Within the Edit menu, click Maintain Cards, press F10, or click on the toolbar.

  2. Do one of the following:

  • In the Maintain Cards dialog box, click the Image button , click Browse, select the desired file and click OK.
    or

  • Drag an image into the Image box.
     

The image is shown in the Image box.

When you drag the first image in the Image box, the Resize Image? dialog box pops up.

 

 

You can choose, in what size the image should be saved in your Learning Module: small, medium, large, or without resizing, in its original size.

You choose by clicking one of the icons in the dialog window.

To choose the same resize option for the question and the answer check Use for both sides.

 

Notes:

When editing cards you can always change the resizing option by clicking .

To add the same image for the question and the answer check use question-image.

Supported image file formats are: bmp, gif, jpg, png, ico, emf, wmf
