Import Cards From Clipboard

Importing cards from the clipboard

MemoryLifter allows you to import cards from web pages and Excel spread-sheets copied to the clipboard.

 

To import from the clipboard

  1. Within the Edit menu, click Import Cards then click From Clipboard....

  2. In the Import from Clipboard dialog box, select the fields you want to import.

  3. Arrange the order,

    • by selecting the field to move up or down in the list and then

    • by clicking the arrows or dragging the item up or down.

      Note:

It does not matter, whether there are unchecked fields between checked items, as MemoryLifter will only import the checked items in the order they appear in the list.
 

  1. Click Finish to open the MemoryLifter CardCollector.

  1. Open your source file, select the data to be collected in the source file and copy it to your clipboard.

Note:

If you selected "Beep on complete row" in the CardCollector, you will hear a beep.

 

  1. Return to MemoryLifter to see your collected data in the table below.

  2. Click Stop.

Caution:
At this point of time, your data has not been saved yet!
Do not just close the CardCollector by clicking the Close button!
 

  1. Select one card in the table below, so that your collected data appears in the related text fields.

  2. Add the card to your Learning Module by clicking Add Card.

  3. Repeat for all the cards you want to add.

  4. Close the window to start learning.
