Learning Module activation

If you purchase pre-made, expert-developed Learning Modules from our website www.memorylifter.com, you will need to activate your Learning Module in order to use it.

Learning Modules which are marked with a silver key icon on the start page need to be activated before learning with them.

 

There are two ways to activate a Learning Module depending on whether your computer has access to the internet or not:

online and offline activation

 

 

To activate a Learning Module online

If you open a Learning Module with a silver key icon, the activation wizard starts.

 

 

You are asked to enter a license key, which you have received by email after purchasing the Learning Module.

 

 

After entering the 16-digit license key the activation is done.

 

 

To activate a Learning Module offline

 

If the computer you are using to learn with MemoryLifter does not have access to the internet, you need to do "offline activation".

The Activation Wizard will explain each step to you and guide you through this procedure.

 

 

 

 

You need a computer with internet access now.

Open the following webpage: http://www.memorylifter.com/web-activation.html

Enter the license key and machine id in the Learning Module Web-Activation form.

 

 

Return to the computer you are using to learn with MemoryLifter.

You will have to run the activation wizard again. Then you enter the Unlock Key and the Activation Key.

 

 

 

 

Note:

Please always check whether you use the correct keys corresponding to the Learning Module you would like to activate. If you mix up the keys having purchased several modules at a time, the activation will fail.
